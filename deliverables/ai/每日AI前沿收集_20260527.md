# 每日AI前沿收集

> 目标：每天收集最前沿AI发展，包括思想、工具和对社会工作方式的改变
> 频率：每天1次 | 耗时：5分钟浏览 | 输出：一句话+一个来源链接

---

## 一、 收集范围（4个维度）

| 维度 | 关注什么 | 例子 |
|------|---------|------|
| **思想** | AI领域的新理念、新框架、新范式 | Agent架构、Multi-Agent协作、AI安全观、提示工程新方法 |
| **工具** | 新工具发布、重大更新 | Claude新能力、开源模型突破、AI编程工具、AI+金融工具 |
| **工作方式改变** | AI如何改变具体行业/岗位的工作流 | 银行用AI做尽调、律师用AI审合同、教师用AI备课 |
| **行业信号** | 融资、政策、大公司动向 | AI公司融资、监管新规、科技巨头AI战略调整 |

---

## 二、 收集方式（3条管道）

### 管道1：Claude Code 自动收集（每日定时）

```
/loop 1d 搜索今天关于AI的最前沿发展，包括新工具发布、重要论文、思想观点和对工作方式的改变。用5句话以内总结，记录到deliverables/ai/每日AI简报_202605.md中，格式：日期+标题+一句话+来源链接。
```

### 管道2：手动快读（5分钟）

每天固定时间打开以下任意2个来源，扫标题，只读真正重要的1-2篇：

| 来源 | 为什么选它 |
|------|-----------|
| Hacker News (news.ycombinator.com) | 全球程序员筛选的每日最重要的技术新闻 |
| The Verge / TechCrunch | 科技行业动态 |
| 机器之心 (jiqizhixin.com) | 中文AI前沿，有深度 |
| 量子位 (qbitai.com) | AI产业动态，中文 |
| 36氪 AI频道 | AI创业公司+融资动态 |

### 管道3：深度追踪（按需）

当你对某个方向特别感兴趣时（如"AI+金融风控"），让Claude专门做深度追踪：

```
"帮我追踪AI+金融风控领域最近两周的所有重要动态，包括新工具、新论文、新政策。整理成表格，标注重要性（高/中/低）。"
```


## 四、 判断标准：什么值得记录

不是所有AI新闻都值得关注。用以下标准过滤：

| 值得记录 | 不值得记录 |
|---------|-----------|
| 改变"做事方式"的工具或思想 | 融资多少亿（纯资本新闻） |
| 和你的工作有交集的领域 | 纯技术细节（新模型参数多少） |
| 让人"没想到还能这样"的应用 | 重复报道 |
| 政策/监管的重大变化 | 产品小版本更新 |

**一句话判断**：如果这条信息会让你对同事说"你知道吗...？"——记录。如果只是"哦知道了"——跳过。

## 五、 本周启动

| # | 行动 | 耗时 |
|---|------|------|
| 1 | 在Claude Code中启动每日AI收集循环 | 1分钟 |
| 2 | 收藏2个快读来源（建议Hacker News+机器之心） | 1分钟 |
| 3 | 今天完成第一次收集并记录 | 5分钟 |
| 4 | 手机设一个每日提醒："看AI简报" | 30秒 |

---

> **核心原则**：不是收集越多越好，是筛选越准越好。每天5分钟，一条真正重要的信息，胜过50条过目即忘的标题。
