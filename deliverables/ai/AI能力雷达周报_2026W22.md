# AI能力雷达周报 #1 — 2026年5月第4周

> 首次雷达扫描 | 耗时：~15分钟

## Part 1：四源扫描

### 源1：基准与榜单
**有变化**——Google I/O发布了Gemini 3.5 Flash，coding/agent/multimodal benchmark均达frontier水平，4x输出速度。Gemini Omni支持视频生成与编辑。
- 对我而言：暂时不切换。Claude Opus 4.7在投资分析+写作场景的表现已足够好。但Gemini 3.5 Flash的速度优势值得关注——如果Agent并行分析20家公司，速度提升4x意味着财报季半天→1小时。

### 源2：产品变更日志
**有变化，3条与我的工具栈直接相关**：
1. **Claude Code 2.0系列**（已在用）：checkpoints、subagents、plan mode——这三个功能直接支撑Agent编排维度的提升
2. **DeepSeek V4发布**：开源模型继续追赶，作为备用/降级方案值得关注
3. **Grok Build发布**（xAI）：CLI-first coding agent，$300/月——定价高于Claude Code，暂不试用

### 源3：实战案例
**有变化，2条重要信号**：
1. **Andrej Karpathy加入Anthropic预训练团队**——今年最高级别AI人才流动。说明Anthropic的模型能力还将提升，我选择Claude生态是正确的
2. **Anthropic收购Stainless**（SDK/CLI/MCP server生成器）——开发者体验将成为产品重点，MCP生态会更完善
3. **Claude Code 50天质量坍塌事件**（3/4-4/20）：思考深度下降73%，由外部用户发现而非内部监控。教训——AI工具也需要"监控"，管理模式中的"异常检测"同样适用于AI工具本身

### 源4：反向信号
**有变化，3条**：
1. **Roo Code关闭**（5/15）→转向云端Roomote。信号：IDE-based AI coding tool竞争白热化，独立小工具难以存活
2. **Google关闭Project Mariner**（浏览器AI Agent）→浏览器Agent路线被放弃，行业转向code-capable工具
3. **Heretic工具**可在10分钟内剥离开源模型安全护栏→3500+个"去安全"模型，1300万+下载。对投资者：AI安全是真实风险，不是PR话题
