### 0.1 为什么需要这份总纲

已有文件各自覆盖了某个目标或某个时间段的方案，但缺少一个"俯瞰图"——一眼看清全部6个目标如何通过AI产业链这个统一杠杆同时加速。

### 0.3 AI产业链的定义（本文件语境）

本文件中的"AI产业链"包含五个层次，陈颖芳在每一层都有参与：

| 层级 | 定义 | 陈颖芳的参与方式 |
|------|------|----------------|
| **算力层** | AI芯片、服务器、数据中心 | 投资标的（昆仑万维等）、信贷客户（AI芯片企业） |
| **模型层** | 大语言模型、多模态模型 | Claude Code的底层能力、行业认知升级 |
| **应用层** | 各行业AI应用落地 | 科技金融方法论中"AI+金融"场景 |
| **工具层** | AI开发工具、自动化工具 | Claude Code/Agent/插件/脚本，个人效率杠杆 |
| **知识层** | AI对个人知识体系的影响 | Obsidian知识库AI化、学习路径加速 |

### 2.3 协同效应的核心洞察

**最重要的三条协同链（按杠杆率排序）**：

1. **AI能力 → 全部5个目标**（一对五杠杆，最高优先级）
   - 每提升一个AI技能，投资研究、事业产出、家庭支持、健康管理、知识库全部受益
   - 这是整个总方案中的"牛鼻子"——抓住AI能力提升，其他目标就被撬动

2. **投资研究 ↔ 事业客户**（双向供给，最独特的协同）
   - 这是陈颖芳区别于"纯投资者"和"纯银行人"的独特协同
   - AI产业链是这两个领域的重叠区：AI芯片企业既是投资标的池，也是信贷服务对象
   - 一次深度研究（如AI芯片行业），同时产生投资信号和信贷机会

3. **知识库 → 投资 + 事业**（复利效应，最长期的协同）
   - 知识库的每一次充实，都让AI在未来的投资研究和事业产出中更精准
   - 这是"越用越好用"的正反馈循环——知识复利

## 四、每周AI产业链 × 6目标联动检查机制

> 目标：每周花15分钟，确认6个目标在AI产业链这个统一杠杆下是否同步前进。不是详细复盘，是"扫一眼有没有脱轨"。

### 4.1 检查时间与形式

- **时间**：每周日晚 21:00-21:15（在固定检视日历中已有此窗口）
- **形式**：打开本文件，逐项过一遍4个问题
- **工具**：Claude Code 自动汇总本周各目标的关键数据（投资信号/步数/血压/错题数/AI学习时长）

### 4.2 四个检查问题

#### 问题一：AI产业链有什么新动向？（3分钟）

> 本周AI产业链（算力/模型/应用/工具/知识）有什么值得注意的事件？

| 检查项 | 具体动作 | 时间 |
|--------|---------|:---:|
| AI算力层 | 有没有AI芯片/半导体重大新闻？影响持仓昆仑万维的估值逻辑吗？ | 30秒 |
| AI模型层 | 有没有新的大模型发布/更新？影响Claude Code等工具的能力边界吗？ | 30秒 |
| AI应用层 | 有没有AI+金融的重要案例或政策？可以作为本月行业评论素材吗？ | 1分钟 |
| AI工具层 | 这周学到了什么新的AI用法？对哪个目标有帮助？ | 1分钟 |

**不需要查资料**。如果本周没有关注AI产业链新闻（这是正常的），直接过。**关键是"本周有没有发生需要我注意的事"，而不是"我有没有关注到"**。

#### 问题二：6个目标各自在轨道上吗？（6分钟）

> 逐目标扫一眼：1分钟以内每个目标。用下面的单目标检查清单。

**投资成功**（1分钟）：
- [ ] 这周看了持仓吗？哪怕只是一眼？
- [ ] 昆仑万维有没有重大异动？（/morning-note每天跑的话有底）
- [ ] 减仓/清仓按计划推进了吗？（如果不是减仓周，跳过）
- [ ] 量化策略有什么新进展？（如果不是7月以后，跳过）
- **脱轨信号**：连续2周没看持仓、连续2周没跑/morning-note

**事业进阶**（1分钟）：
- [ ] 这周有没有产生任何"可用于发表"的内容？（哪怕只是100字的想法记录）
- [ ] 中级经济师学习推进了吗？（如果不是备考期，跳过）
- [ ] 月度行业评论有在写吗？（月底检查即可）
- **脱轨信号**：连续2周没有任何与科技金融相关的新输入或新产出

**家庭支持**（1分钟）：
- [ ] 陈于东本周学习状态知道吗？（如果不知道，下周问一句）
- [ ] 两位母亲本周通话了吗？
- [ ] AI辅助（错题分析/周报生成）本周运行正常吗？
- **脱轨信号**：连续2周不知道孩子的学习状态、连续2周没和母亲通话

**百岁健康**（1分钟）：
- [ ] 这周运动≥3天了吗？
- [ ] 血压/体重数据在采集吗？（AI预警系统会主动提醒异常）
- [ ] 有没有身体不适需要关注？
- **脱轨信号**：连续2周运动<2天、血压数据连续缺失>3天

**AI能力**（1分钟）：
- [ ] 这周用Claude Code了吗？哪怕只是改了一行字？
- [ ] 有没有学会任何新的AI用法？（哪怕只是一个新的插件命令）
- [ ] Python/Agent编排按计划推进了吗？
- **脱轨信号**：连续2周没有使用Claude Code做任何新的事情（只用它干已经会的）

**知识库**（1分钟）：
- [ ] 这周有没有产生值得沉淀的知识？（如果有，花1分钟语音说"把这个记下来"）
- [ ] 有没有发现两个领域之间新的连接？
- **脱轨信号**：连续4周没有新增任何知识卡片

#### 问题三：本周有协同发生吗？（3分钟）

> 检查6大目标之间有没有产生协同效应。不需要每个都有，但需要至少意识到。

| 检查项 | 如果发生了... | 时间 |
|--------|-------------|:---:|
| 投资↔事业 | 今天研究的一只AI股票，有没有可能也是潜在信贷客户？或者反之？ | 45秒 |
| AI→投资/事业 | 这周学的新AI技能，有没有让投资研究或事业产出更快了？ | 45秒 |
| 健康→全部 | 身体状态好（或不好）有没有影响其他目标的推进？ | 30秒 |
| 知识库→任意 | 这周有没有从知识库里调取过以前沉淀的内容来辅助当前决策？ | 30秒 |
| 家庭↔事业 | 家庭琐事有没有挤占事业和投资的时间？AI有没有帮忙减轻？ | 30秒 |

**协同的"最低标准"**：每周至少有一个跨目标的积极互动（哪怕只是"今天研究AI芯片时想到了可以作为信贷客户"），就说明系统在运转。

#### 问题四：下周重点是什么？（3分钟）

> 基于以上检查，决定下周6个目标的优先级排序。不需要面面俱到，只需要知道"如果只能做3件事"是哪3件。

**优先级快速判定表**：

| 如果本周出现以下信号... | 那么下周的P0是... |
|------------------------|------------------|
| 投资：昆仑万维异动/接近止损线/减仓窗口到来 | 投资行动优先 |
| 事业：投稿截止日临近/中级经济师模块测试 | 事业产出或备考优先 |
| 健康：黄色/红色预警触发 | 健康（就医/调整运动）优先 |
| 家庭：孩子考研关键节点（报名/确认院校） | 家庭支持优先 |
| 多目标同时告急 | AI能力投入加倍——用AI效率补偿其他目标的时间缺口 |

**输出**：在手机备忘录或Obsidian日记中写一行：
```

N/A: [本周可以跳过的目标]
```
