## 你将确认什么，为什么重要

这次通话直接决定9月还本方案的选择：

| 确认项目 | 影响哪个决策 | 如果答案是"不行" |
|----------|-------------|-----------------|
| 1. 确切到期日 | 所有方案的时间轴 | 精确到天，标注日历 |
| 2. 还款账户号 | 方案A/B/D的资金归集路径 | 向错误账户转账=资金损失 |
| 3. 提前还款罚金 | 方案A（6-8月提前还） | 有罚金则等到期日再还 |
| 4. 单日转账限额 | 所有方案的还款操作 | 限额不足需柜台或分日转 |
| 5. 部分还款+展期 | **方案B是否成立** | 方案B直接失效，回归方案A/D |
| 6. 展期后再提前还 | 方案B执行后的退出路径 | 有罚金则展期后不要提前还 |

## 已有信息核对（先确认再往下问）

> 根据贷款合同和月供记录，已知以下信息。打电话时请客服逐项确认是否正确。

### 问题0a：当前贷款利率

> "请帮我确认一下，这笔贷款的年化利率是多少？我这边记录的月供利息大约是1,000元，对应的年化利率是2%左右，对吗？"

**记录栏：**
- 当前年化利率：______%
- 利率类型：固定利率 / 浮动利率
- 如为浮动利率：参考基准是______，当前加点______bp
- 月供利息：______元（是否与我记录的~1,000元一致？）
- 利率在合同期内是否会调整？是 / 否
- 如会调整：调整频率______，下次调整日期______

**追问提示：** "这个利率是签订合同时就定下来的，还是中间调整过？"

### 问题0b：当前还款方式

> "请确认一下我当前的还款方式——我这边记录的是'先息后本'，就是每个月只还利息，到期一次性还本金60万，对吗？"

**记录栏：**
- 当前还款方式：先息后本 / 等额本息 / 等额本金 / 其他______
- 月供金额：______元（本金______+利息______）
- 如果记录有误（非先息后本）：月供总额______元，其中本金______元，利息______元
- **这直接影响方案C/D的可行性评估**

**追问提示：** "如果续贷或展期，还款方式可以继续保持先息后本吗？还是会变成等额本息？"

## 6个必须确认的问题

### 问题1：确切到期日

> "请问我这笔贷款的精确到期日是哪一天？是9月几号？"

**记录栏：**
- 到期日：______年____月____日
- 是到期日还是最后还款日？________________
- 有无宽限期（到期日后几天内还款不算逾期）？______天
- 如果到期日是周末/节假日，顺延到下一工作日吗？是 / 否

**追问提示：** 如果客服只说"9月"，追问"具体是哪一天？我需要在那天之前安排好资金。"

### 问题2：还款账户号

> "还款是还到哪个账户？这个账户和贷款发放时的收款账户是同一个吗？"

**记录栏：**
- 还款账号：________________
- 开户行：________________
- 户名：________________
- 是否为自动扣款账户？是 / 否
- 如果是自动扣款：扣款日是到期日当天还是提前几天？______
- 如果账户余额不足，银行会部分扣款还是不扣？________________

**追问提示：** "如果我有多个账户，是否可以从任意账户转账还款，还是必须还到指定的还款账户？"

### 问题3：提前还款罚金

> "如果我在到期日之前提前还款，有没有违约金或手续费？比如我8月份就想还清。"

**记录栏：**
- 是否收提前还款违约金？是 / 否
- 如有违约金：收取比例______% 或固定金额______元
- 违约金计算方式：________________
- 是否分时间段？（如：到期前1个月内不收，1-3个月收1%）________________
- 提前还款是否需要预约/提前申请？是 / 否
- 提前还款流程：________________

**追问提示：** "提前多久还款不收违约金？比如说到期前一周还行不行？"

### 问题4：手机银行单日转账限额

> "我的手机银行单日转账限额是多少？我需要一次性转账60万用于还款，这个额度够吗？"

**记录栏：**
- 当前单日转账限额：______万元/日
- 单笔转账限额：______万元/笔
- 60万需要分______天或______笔才能转完
- 提额流程：________________
- 提额需要什么材料？________________
- 提额审批需要多长时间？______工作日
- 是否支持柜台大额转账？是 / 否
- 柜台转账手续费：______元或______%

**追问提示：** "如果手机银行限额不够，我到柜台办理大额转账，有没有金额上限？需要带什么证件？"

### 问题5：部分还款+展期可行性

> "如果到期时我无法一次性还清60万，是否可以还一部分（比如30万），剩余部分申请展期继续使用？"

**记录栏：**
- 是否支持"部分还款+剩余展期"？是 / 否 ← **方案B的核心前提**
- 如支持：部分还款的最低金额是多少？______元
- 展期最长期限：______个月（或______年）
- 展期后利率：______%（年化）
- 展期利率是否与当前合同一致？是 / 否（新利率可能更高）
- 展期是否需要重新审批？是 / 否
- 如需审批：需要什么材料？________________
- 审批时间：______工作日
- 展期是否有手续费？是 / 否，______元
- 展期后还款方式：先息后本 / 等额本息 / 其他______
- 年龄限制：49岁是否可以申请______年展期？

**追问提示：** "不展期的话，全额展期（60万全部续贷）的条件是怎样的？利率和期限呢？"

### 问题6：展期后再提前还款

> "如果我申请了展期，之后有资金了想提前把展期部分也还清，有没有违约金？"

**记录栏：**
- 展期后提前还款是否收违约金？是 / 否
- 如有：收取比例______% 或规则________________
- 展期后多久可以提前还款？（是否有锁定期？）______个月
- 提前还款是否需要预约？是 / 否

**追问提示：** "展期合同和原合同的提前还款政策是一样的吗？还是会重新约定？"
