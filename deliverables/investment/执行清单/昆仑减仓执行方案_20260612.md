## 三条操作线

### ① 反弹卖出线（如到达→执行减仓）
```
目标价：45元 或 50元
触发：股价反弹至此价位
执行：卖出第1批 3,000股（释放~13.5-15万）
理由：反弹到45-50仍亏10-20%，但已大幅缩小亏损，降低集中度风险
```

### ② 止损底线（如跌破→必须执行）
```
底线价：30元（较当前再跌约19%）
触发：收盘价跌破30元
执行：强制卖出第1批 3,000股（释放~9万）
理由：跌破30意味着亏损超43%，进一步下跌空间打开，必须止血
```

### ③ 中间区（30-45元之间）
```
策略：持有不动，不做任何操作
心理纪律：不因为"跌多了"加仓摊平，也不因为"涨了一点"就急着卖
```

## 心理提示

> **"等回本"是所有散户亏损的核心原因之一。**
> 当前方案不是"无限期等回本"，而是"设定反弹卖出目标+设定止损底线"。
> 到45-50就卖一部分，不是为了赚，是为了**降低集中度风险**。
> 到30必须卖，是为了**防止亏损继续扩大**。
> 这两个价格无论到哪个，方案都要求你行动。
