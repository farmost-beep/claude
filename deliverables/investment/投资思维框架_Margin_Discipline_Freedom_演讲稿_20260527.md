## 演讲正文（全文约2,500字，20分钟）

---

### 开场（2分钟）：我不是来教你赚钱的

各位好，我是陈颖芳。

在开始之前，我需要做一个声明：我不是投资大师，不是一个有20年复合回报率可以吹嘘的人，不是来教你怎么选股的。

我是一个49岁才开始认真对待投资的人。我的起始资金是100万人民币，还背着60万的信用贷款。我的投资组合里曾经有一只股票占了69.5%的仓位——我知道这个数字你们听了都觉得很夸张，但它就是事实。

所以今天我不是以成功者的身份来的。我是以"一个正在路上的初学者"的身份，和你分享我在搭建自己投资体系的过程中，找到的三个最重要的单词。

这三个单词是——**Margin. Discipline. Freedom.**

安全边际。纪律。自由。

这三个词，就是我今天想讲的唯一内容。

---

### 第一部分：Margin（安全边际）—— 7分钟

**"安全边际不只是便宜买"**

在座的各位应该都听过"安全边际"这个词。它来自格雷厄姆——用四毛钱买一块钱的东西。

这是我学会的第一课。但今天我想说的是：安全边际不止这一层。

在我自己的框架里，安全边际有三层。

**第一层：价格的安全边际。**

这个大家都知道——买得便宜，你就有了犯错的空间。格雷厄姆说用四毛买一块，你就算看错了，值八毛，你还是赚的。这是最基础的安全边际，也是最容易计算的那一层。

但问题是——你怎么知道它值一块钱？这引出了第二层。

**第二层：认知的安全边际。**

芒格说过一句话，我把它翻译得直白一点：只投你真的懂的。不是"大概懂""看过一篇研报""听朋友说过"——是"真的懂"。

什么叫真的懂？我有一个很简单的测试：你能不能不用任何专业术语，用一个比喻，给一个完全不懂这个行业的人讲清楚这家公司是怎么赚钱的？如果你做不到，你就不懂。

如果你不懂，你就算买了极其便宜的价格，你也拿不住。因为当它再跌20%的时候，你不会想"我更便宜了"，你会想"我是不是哪里判断错了"——然后你在最低点卖了。

**认知的安全边际 = 你懂的程度 × 你持仓的比例。** 你不懂的，别重仓。不管别人怎么看好。

**第三层：时间的安全边际。**

这是我自己的补充。前两层来自格雷厄姆和芒格，这一层来自我自己的血泪教训。

时间的安全边际就是：你有一笔钱，你要想清楚——这笔钱什么时候必须用？如果你的钱三个月后要还债，你就没有时间安全边际。三个月内股价可能涨，也可能跌——但你不能承受跌，所以你就没有资格承受涨。

所以我给自己定了一条铁律：永远保留12个月生活费的现金，放在货币基金里。这笔钱不是"踏空了可惜"，是"活得下去的底气"。还本的钱、教育的钱、救命的钱——不进股市。

**三层叠加。**

价格安全边际让你买得便宜，认知安全边际让你看得清楚，时间安全边际让你等得起。三者都具备的时候，你才能睡得着觉。

缺任何一层，你都会在凌晨三点醒来，打开手机看美股行情。

---

### 第二部分：Discipline（纪律）—— 7分钟

**"世界上最难管的人是自己"**

我有一份文件叫"投资政策声明"——Investment Policy Statement，简称IPS。它就是我给自己立的投资宪法，有六条不可违反的条款。

第一条：永不上杠杆。第二条：单一品种不超过总资产的25%。第三条：永远持有12个月的现金。等等。

这份IPS写得挺漂亮的。但问题是——写一份宪法容易，遵守它难。

**纪律的第一关：承认错误。**

我持仓的一只股票叫昆仑万维，它在最高的时候占了69.5%的仓位。69.5%——这个数字意味着什么？意味着这只股票涨5%，我的账户涨3.5%；它跌5%，我的账户跌3.5%。

这不是投资，这是赌博。

但把一只占了69.5%的股票减到25%以下——这个动作，在操作上只需要5分钟，在心理上却需要几个月。因为减仓意味着你要承认："我之前的决定是错的，我的集中持仓是错的，我在这只股票上已经发生的亏损是我必须接受的。"

很多人宁可账面亏50%，也不愿意卖出一股——因为卖出 = 确认亏损 = 承认自己错了。

**但亏损已经发生了。** 卖出只是确认，不是造成。这是纪律的第一关。

**纪律的第二关：在涨的时候卖。**

这不是反人性吗？涨的时候不是应该拿着等它再涨吗？

如果你的目标是"卖在最高点"，那确实应该在涨的时候不卖。但没有人能持续卖在最高点。如果你把投资当成——在相对合理的位置，把仓位降到相对安全的水平——那涨的时候卖，就是纪律的一部分。

**纪律的第三关：什么都不做。**

这可能是最难的一关。因为什么都不做，看起来像在偷懒。但交易的真相是：大部分时间应该什么都不做。一年可能只有3-5个真正值得行动的决策点。其他时间都是在等。

"不操作"本身就是一种操作。它是一种主动的选择——选择不被市场情绪牵着走，选择不让FOMO（Fear of Missing Out，害怕错过）驱动你的手指点下"买入"。

---

### 第三部分：Freedom（自由）—— 3分钟

**"投资的终极目的是自由，不是财富"**

我问过自己一个问题：我到底为什么投资？

为了赚到900万？为了退休的时候账户好看？为了证明自己比隔壁老王聪明？

都不是。

我投资的终极目的是——**自由**。

自由意味着：有说"不"的权利。对不喜欢的工作说不，对消耗你的关系说不，对被别人安排的生活说不。

所以900万不是目的。900万带来的"我不需要再为钱做我不想做的事"——这个才是。

如果我想清楚了这个目的，投资中的很多纠结就自动消失了。

该不该追那个已经涨了3倍的热门股？——不需要，因为它不能给我带来更多的自由，只会带来更多的焦虑。

要不要上杠杆搏一把？——不需要，因为杠杆不会让我更自由，只会让我更不自由。

该不该因为短期回撤感到痛苦？——不需要，因为短期回撤不影响30年的自由目标。

**自由，才是投资的终极回报。** 而Margin和Discipline，只是通往那里的路。

---

### 结语（1分钟）

回到开头的那个声明。

我49岁，金融资产100万，还背着债。我不是来教你赚钱的。

但如果这三个单词——Margin, Discipline, Freedom——在你做下一个投资决策的时候，能在脑海里出现一秒钟，那今天这个分享就有价值了。

身体不会一夜之间变好，组合不会一个季度变健康，自由不会一笔交易变到手。

但它会回应每一份持续的善意。

谢谢你。

不是投资大师，不是来教你怎么选股的

### Slide 3: 三个单词
```
Margin  ·  Discipline  ·  Freedom
安全边际     纪律          自由

不是三个技巧，是一个体系的三层：
Margin  →  怎么控制风险
Discipline → 怎么管理自己
Freedom   → 为了什么而投资
```

### Slide 4: Margin — 三层安全边际
```
安全边际 ≠ 便宜

① 价格安全边际（格雷厄姆）
   买得便宜 → 有犯错空间

② 认知安全边际（芒格）
   真的懂 → 跌了敢拿住

③ 时间安全边际（我的补充）
   等得起 → 不需要被迫卖出

三层叠加 = 睡得着觉
```

### Slide 5: Margin — 认知安全边际测试
```
你怎么知道自己"真的懂"？

测试：不用任何术语，用一个比喻
给外行讲清楚这家公司怎么赚钱

做不到？→ 你不懂
不懂？→ 别重仓

认知安全边际 = 懂的程度 × 持仓比例
```

### Slide 6: Margin — 时间安全边际
```
一笔钱的"时间约束"决定它能去哪

┌──────────────┬─────────────┐
│ 钱的用途     │ 能承受波动？ │
├──────────────┼─────────────┤
│ 3个月后还债  │   不能      │ → 现金
│ 11年后退休   │   能        │ → 权益
│ 30年后传承   │   更能      │ → 权益+实物
└──────────────┴─────────────┘

12个月生活费 = 永远在货基里
不是"可惜"，是"底气"
```

### Slide 7: Discipline — IPS投资宪法
```
世界上最难管的人是自己

我的IPS六条铁律：
① 永不上杠杆
② 单票≤25%
③ 永远持有12个月现金
④ 不在情绪极端时做决策
⑤ 每季度检视一次，不是每天
⑥ 违反任何一条 → 立即停止交易，冷静72小时

写宪法容易，遵守它难
```

### Slide 8: Discipline — 三重考验
```
纪律 ≠ 频繁操作

考验1: 承认错误
→ 亏损已经发生，卖出只是确认

考验2: 在涨的时候卖
→ 目标不是"卖在最高点"，是"降到安全线"

考验3: 什么都不做
→ 一年3-5个决策就够了。等待是主动选择
```

### Slide 9: Freedom — 终极目的
```
投资不是为了赚900万

900万只是数字

我要的是900万能买来的东西：
→ 说"不"的权利
→ 不需要为钱做不想做的事
→ 时间归自己支配

自由 = 终极回报
Margin + Discipline = 通往自由的路径
```

### Slide 10: 三条线
```
这三条线，构成了我的投资体系：

      Margin (安全边际)
        ↓
      保护自己不受市场的伤害

    Discipline (纪律)
        ↓
      保护自己不受自己的伤害

     Freedom (自由)
        ↓
      告诉你为什么出发
```

### Slide 11: 最后
```
身体不会一夜之间变好
组合不会一个季度变健康
自由不会一笔交易变到手

但它会回应每一份持续的善意。

Margin. Discipline. Freedom.

谢谢你。
```
