# 考研AI辅助系统 — 启动包

> 使用者：陈于东（20岁，大三→考研备考） | 2026-05-27
> 监督者：陈颖芳（妈妈，看每周一句话摘要即可）
> 这套系统帮你用Claude免费做三件事：**周计划生成 / 错题分析 / 进度追踪**

---

## 一、每周日晚：周计划生成（5分钟）

### 使用方法

打开Claude App或Claude Code，输入以下内容（把 `[...]` 替换成你的实际情况）：

```
[考研周计划]

本周完成情况：
- 英语：单词打卡[X]天，精读[Y]篇阅读（正确率[Z]%）
- 数学：[学到哪一章，例如：高数上第4章不定积分]
- 专业课：[如有，学到哪了]
- 本周总学习时间：约[X]小时

最难的地方：
- [具体的知识点，例如：不定积分的换元法总是选错u]
- [如果有第二个，继续写]

下周特殊安排：
- [例如：周三全天实习 / 周六同学聚会 / 没有特殊安排]
```

Claude会自动输出：
- 下周每天的学习计划表（精确到上午/下午/晚上时段）
- 针对你卡住的知识点，推荐3-5道练习题
- 一句话鼓励

### 示例

输入：
```
[考研周计划]

本周完成情况：
- 英语：单词打卡7天，精读2篇阅读（正确率80%）
- 数学：高数上第三章微分中值定理
- 本周总学习时间：约18小时

最难的地方：
- 拉格朗日中值定理的构造函数技巧
- 证明题不知道从哪里入手

下周特殊安排：
- 周三下午实习
```

Claude输出（2分钟后）：
```
【第__周学习计划 | 2026年X月X日-X月X日】

数学 — 重点攻克：中值定理（拉格朗日+柯西）
- 周一(上)：李永乐P85-92 拉格朗日中值定理例题精讲
- 周一(下)：做课后习题 第3章选择题1-15题
- 周二(上)：看完柯西中值定理 P93-98
- 周二(下)：整理中值定理三种构造方法对比表
- 周三(上)：做证明题专项5道（重点：构造函数技巧）
  下午：实习
- 周四(上)：中值定理错题重做 + 整理本周错题
- 周四(下)：精读2009年Text 3
- 周五(上)：开始第4章不定积分 P102-112
- 周五(下)：不定积分基础练习题10道
- 周六(上)：（自由安排/补本周未完成的）
- 周六(下)：精读2009年Text 4
- 周日：休息 / 晚上15分钟更新本周进度

英语 — 保持节奏
- 每日：30新词+复习旧词（墨墨自动安排）
- 周四+周六：精读2009年Text 3/4
- 碎片时间：听考研英语听力磨耳朵

鼓励：
"中值定理卡住很正常 — 1950年代的数学系学生也为它熬夜。
 你已经过了最难的连续性和可导性证明，中值定理只是换个角度。
 这周重点是掌握构造函数的三板斧：加减、乘除、复合。"
```

## 二、随时可用：错题分析系统

### 使用方法

做错的题，拍照发给Claude，附上文字：

```
[错题分析]

这道题我卡在____（选填：选择题/填空题/证明题）

我的思路是____（写一下你当时怎么做错的，一两句话就行）
```

Claude会自动输出：
1. 这道题的知识点标签（如"拉格朗日中值定理·构造函数技巧"）
2. 你的错误类型（概念不清 / 计算粗心 / 思路错误 / 题型不熟）
3. 推荐3道同类题
4. 一句解题技巧提示

### 错题积累到20道以上时

对Claude说：
```
[生成薄弱点分析]

以下是我的错题库（已累积20+道），请分析我的薄弱章节热力图
```

Claude会输出：
- 哪个章节错题最多（一目了然）
- 哪个错误类型最频繁
- 考前15分钟"个人易错点清单"

## 三、每月末：月度报告

### 使用方法

每月最后一个周日，对Claude说：

```
[月度考研报告]

本月：
- 英语单词打卡[X]天 / 30天
- 精读完成[Y]篇
- 数学学到[第X章]
- 专业课进度：[X]%
- 本月总学习时间：约[X]小时
```

Claude会输出：
- 英语打卡趋势（与上月对比）
- 数学各章节掌握度评估
- 与暑假验收标准的差距（还差什么、还要多久）
