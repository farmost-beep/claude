## 如何使用

### 完整流程

```
陈于东拍照错题 → 发给Claude（附上"[错题分析]"） → Claude分析 → 追加到本文
```

### 具体步骤

1. **拍照**：用手机拍下错题，确保题目和你的作答都清晰可见
2. **发送给Claude**：打开Claude App，发送图片并附上文字：
   ```
   [错题分析]
   我卡在____（选填：选择题/填空题/证明题）
   我的思路是____（写一下你当时怎么做错的）
   ```
3. **Claude自动分析**：Claude会识别题目、分析错误类型、给出正确思路，并将结果追加到下方错题列表
4. **定期回顾**：每周日晚回顾本周新增错题；累积20道后生成薄弱点分析

### 关键触发词

| 你想做的事 | 对Claude说什么 |
|-----------|--------------|
| 录入一道错题 | `[错题分析]` + 图片 + 你的思路 |
| 生成薄弱点热力图 | `[生成薄弱点分析]`（需累积20+道错题） |
| 生成考前易错清单 | `[生成易错点清单]`（考前1个月使用） |
