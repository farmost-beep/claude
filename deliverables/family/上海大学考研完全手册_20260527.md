## 一、 为什么选上海大学

### 1.1 学校定位

| 维度 | 详情 |
|------|------|
| 全称 | 上海大学（Shanghai University，简称SHU） |
| 层次 | 国家"211工程"重点建设高校、"双一流"建设高校 |
| 主管部门 | 上海市人民政府 |
| 校区 | 宝山校区（主校区）、延长校区、嘉定校区 |
| 在校生 | 约38,000人（研究生约20,000人） |
| 学科特色 | 工科强校、艺术设计突出、社科经管稳步上升 |

### 1.2 为什么是适合的选择

上海大学在211高校中处于**"努力够得到"**的位置——比复旦交大同济容易考，比双非院校含金量高。

| 优势 | 说明 |
|------|------|
| **地理位置** | 上海，实习/就业资源丰富，和母亲同城 |
| **考试难度** | 多数专业复试线在320-360，比985低一档，但比双非高 |
| **招生规模** | 2025年招收5,331人（统考3,710人），名额充裕 |
| **就业认可度** | 上海本地企业认可度高，金融/IT/制造业均有校友网络 |
| **成本** | 学费与上海其他211持平，生活成本可控（住校） |

```
2026年
  7-8月   确定目标专业 + 开始公共课打基础

2027年

  3月     国家线公布 → 上海大学复试线公布
  3-4月   复试（面试+笔试）
  4-5月   录取结果公布

```

## 四、 考试内容与备考策略

### 4.1 考试科目结构

| 科目 | 分值 | 考试时长 | 说明 |
|------|------|---------|------|
| **政治** | 100分 | 3小时 | 全国统一命题 |
| **英语** | 100分 | 3小时 | 英语一（学硕）/ 英语二（专硕） |
| **数学** | 150分 | 3小时 | 数学一/二/三，根据专业而定 |
| **专业课** | 150分 | 3小时 | 上海大学自主命题 |

### 4.2 分科目备考策略

#### 政治（目标70+）

| 阶段 | 时间 | 任务 | 资料 |
|------|------|------|------|
| 基础 | 2026年7-9月 | 过一遍核心知识点，建立框架 | 肖秀荣《精讲精练》+徐涛网课 |
| 强化 | 2026年10-11月 | 刷选择题，整理错题 | 肖秀荣《1000题》 |
| 冲刺 | 2026年12月 | 背大题，做模拟卷 | 肖四、肖八 |

#### 英语（目标70+）

| 阶段 | 时间 | 任务 | 资料 |
|------|------|------|------|
| 持续 | 全程 | 每天背50个单词，雷打不动 | 墨墨背单词APP / 红宝书 |
| 基础 | 7-8月 | 每天精读1篇真题阅读 | 张剑《黄皮书》英语一/二 |
| 强化 | 9-10月 | 刷完近年真题（阅读+完型+新题型） | 近10年真题 |
| 冲刺 | 11-12月 | 准备作文模板+翻译+模拟 | 王江涛作文 + 真题模拟 |

#### 数学（目标120+）

**这是拉开分差的关键科目。英语政治拉不开20分，数学可以拉开50分。**

| 阶段 | 时间 | 任务 | 资料 |
|------|------|------|------|
| 基础 | 7-8月 | 系统学完高数+线代+概率全部考点 | 汤家凤/张宇基础班 + 课本 |
| 强化 | 9-10月 | 刷题+总结题型，建立解题框架 | 李永乐《复习全书》+ 张宇《1000题》 |
| 真题 | 11月 | 近15年真题，限时3小时模拟 | 真题套卷 |
| 冲刺 | 12月 | 查漏补缺，做模拟卷保持手感 | 李林6+4套卷、张宇8+4套卷 |

#### 专业课（目标120+）

| 阶段 | 时间 | 任务 |
|------|------|------|
| 定位 | 7月 | 确定目标专业 → 到上海大学研究生招生网查专业课考试科目和参考书 |
| 基础 | 8-9月 | 系统研读指定教材，做课后习题 |
| 强化 | 10-11月 | 找近5年真题（校内打印店/学长学姐/考研论坛），逐题吃透 |
| 冲刺 | 12月 | 重点章节再过一遍，模拟考试 |

## 六、 AI辅助备考方案

> 详细方案见《考研AI辅助系统_启动包》。以下是快速使用指南。

### 6.1 每周日（5分钟）

把以下信息发给Claude：
1. 本周完成了哪些学习任务？
2. 哪个科目/章节最难？
3. 身体/心态状态怎么样？

→ Claude输出：**下周学习计划**

### 6.2 错题管理（每次15分钟）

1. 把做错的题拍照发给Claude
2. Claude自动分类：知识漏洞 / 粗心 / 解题方法错误
3. 累计20道错题后 → Claude生成"你的薄弱环节热力图"

### 6.3 月末报告（每月1次）

把本月学习数据发给Claude：
- 各科学习时长
- 完成真题套数
- 模拟分数趋势

→ Claude输出：**月度考研备考报告**（含进度评估+下月调整建议）

### 6.4 模拟面试（复试前）

> "假设你是上海大学XX学院的复试考官。问我5个专业面试问题，一次问一个。"

## 九、 母亲的支持角色

> 你的角色：不是监督者，是支持者。详见《考研AI辅助系统_启动包》。

| 你该做的 | 你不该做的 |
|---------|-----------|
| 每周日帮他把学习数据录入AI，生成下周计划 | 每天问"今天学了多少" |
| 每月末和他一起看AI生成的月度报告 | 拿别人家孩子比较 |
| 他主动求助时，帮他找资源/资料 | 替他做计划、定目标 |
| 关注他的状态（身体/心态），而不是分数 | 因为一次模拟分低就焦虑 |

**最重要的一句话**（建议在7月启动时告诉他）：
> "这个计划是参考框架，你可以按自己的节奏调。妈妈不检查进度。"
