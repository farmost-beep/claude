# 戴维斯双击/双杀

**一句话**：盈利和估值同向变动时，股价产生乘数级涨跌。

**核心要点**：
- 戴维斯双击：企业盈利增长 + 市场给更高估值倍数 → 股价 = EPS x PE 双升，产生超额收益；双杀则相反
- 双击的经典场景：周期性行业低谷买入（低盈利+低PE），等待盈利恢复+估值修复，赚两份钱
- 双杀的危险场景：高成长股一旦增速放缓，市场不仅杀盈利预期还杀估值倍数，跌幅远超基本面恶化程度

**与我相关**：昆仑万维当前处于什么阶段？如果AI业务兑现盈利增长且市场认可其估值，就是双击；如果AI故事证伪且估值回归传统游戏公司水平，就是双杀。69.5%的仓位面对的是倍数级双杀风险——应该为这种场景做压力测试。

**来源**：戴维斯家族投资理念，见约翰·罗斯柴尔德《戴维斯王朝》(The Davis Dynasty, 2001)
