# 昼夜节律

**一句话**：人体内置的24小时生物钟，调控睡眠-清醒、激素分泌、体温等生理过程。

**核心要点**：
- 核心机制：下丘脑的视交叉上核(SCN)通过光线信号同步体内时钟——清晨蓝光抑制褪黑素让人清醒，夜晚黑暗促进褪黑素分泌诱导睡眠
- 节律紊乱的代价：长期熬夜或作息不规律会打乱皮质醇、褪黑素的正常节律，增加肥胖、糖尿病、心血管疾病风险，降低免疫力和认知表现
- 最重要的节律锚点是"固定起床时间"——比固定入睡时间更重要，因为起床光照是重置生物钟的最强信号

**与我相关**：作为银行从业者工作日作息相对规律，但周末容易报复性晚睡打乱节律，产生"社交时差"。在百岁健康目标下，需要建立三个节律锚点：固定起床时间(如6:30)、固定早餐时间、固定晚间运动窗口(不在睡前3小时内剧烈运动)。每天早晨晒15分钟自然光，帮助重置生物钟。

**来源**：2017年诺贝尔生理学或医学奖（霍尔、罗斯巴什、杨）；萨钦·潘达《昼夜节律密码》(The Circadian Code, 2018)
