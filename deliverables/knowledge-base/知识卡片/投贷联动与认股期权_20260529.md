# 投贷联动与认股期权

**一句话**：投贷联动的本质不是"银行也做VC"，而是"用股权回报对冲债权无法定价的技术风险"——将传统银行无法承受的尾部风险转化为认股期权的上行收益。

**核心要点**：
- 为什么需要认股期权：银行对"技术会不会成功"的判断力远不如VC——银行不应该试图比VC更懂判断技术。但银行可以"如果我判断对了（放款），我需要分享成功的好处"，用上行收益对冲下行风险。认股期权就是这个逻辑的金融工具
- 期权设计的关键参数：(1)行权价=放款日的企业估值（银行不应等到企业估值涨了再行权——银行的风险承担在最早阶段，期权回报应对应这个风险）；(2)期权比例=贷款金额/估值×风险调整系数（通常2-5%，足够小以不挤压创始人，足够大以有意义地覆盖坏账损失）；(3)退出机制=IPO时转股/并购时转股/下一轮融资时转股——银行不谋求长期持股，退出时点应清晰
- 与VC的关键差异：银行不是LP，资金来源是存款而非LP出资——这意味着更低的风险容忍度；银行不参与公司治理（不进入董事会），只分享回报；银行贷款+期权的组合提供的是"可预期回报"，VC的纯股权投资提供的是"可爆发回报"

**与我相关**：金规〔2025〕27号首次允许参股型并购贷款（≥20%股权）——这为"贷款+认股期权"模式提供了制度基础。邮储可以先从"并购贷款+认股期权"起步（已在27号文框架内），再逐步推进到一般科技贷款+期权。
