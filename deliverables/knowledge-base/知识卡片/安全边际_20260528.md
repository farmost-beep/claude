# 安全边际

**一句话**：用4毛钱买价值1块钱的东西，那6毛的差价就是安全边际。

**核心要点**：
- 安全边际是价值投资的核心原则，由格雷厄姆提出：买入价格必须显著低于内在价值，为判断错误预留缓冲空间
- 安全边际不是"买便宜货"，而是用价格折扣对冲认知不确定性——你对企业的估值越不确定，需要的安全边际越大
- 安全边际不仅适用于股价，也适用于仓位管理、杠杆使用、现金流预留等所有涉及风险的决策场景

**与我相关**：作为邮储银行科技金融事业部高级副经理，对科技企业的授信决策本质也是安全边际——企业估值打几折放款才安全。投资组合中昆仑万维69.5%的集中度问题，缺乏仓位层面的安全边际。需要将安全边际思维从"买价要便宜"扩展到"仓位要分散"。

**来源**：本杰明·格雷厄姆《聪明的投资者》(The Intelligent Investor, 1949)
