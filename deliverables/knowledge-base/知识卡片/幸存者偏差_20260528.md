# 幸存者偏差

**一句话**：只看到活下来的成功案例，却看不到大量失败案例，导致对成功概率的系统性高估。

**核心要点**：
- 经典案例：二战时军方想加固返航飞机弹孔最多的部位，瓦尔德指出应该加固弹孔最少的部位——因为那些部位中弹的飞机根本没飞回来
- 在投资中，我们看到的"十倍股"故事全是幸存者，大量退市、腰斩的股票从视野中消失了——这导致高估选股能力、低估风险
- 避免幸存者偏差的方法：做决策前刻意寻找"失败案例集"，问"如果这件事失败了，最可能的原因是什么"

**与我相关**：做科技金融授信时，看到的都是活下来的企业，容易高估创业成功率。看昆仑万维等AI概念股时，市场只报道成功转型的案例，需要刻意寻找"AI转型失败"的样本做参照。投资决策日志应记录"未买入但后来涨/跌的股票"，防止选择性记忆。

**来源**：亚伯拉罕·瓦尔德二战研究；纳西姆·塔勒布《随机漫步的傻瓜》(Fooled by Randomness, 2001)
