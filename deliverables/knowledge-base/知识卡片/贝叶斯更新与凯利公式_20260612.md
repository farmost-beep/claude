# 知识卡片：贝叶斯更新

| 字段 | 内容 |
|:----|:------|
| **标题** | 贝叶斯更新 — 投资决策的概率思维框架 |
| **核心观点** | 投资不是"对或错"的二元判断，而是持续用新证据更新概率的过程。每次看到新信息，先问"这如何改变我的赔率？"而不是"这证明我对了吗？" |
| **公式** | P(假设丨新证据) = P(假设) × P(新证据丨假设) / P(新证据) |
| **投资中的应用** | ① **初始概率**：买入时对"这只股票会上涨"的信心（如70%）② **新证据**：财报/行业政策/技术突破 ③ **后验概率**：每次新证据后更新信心值 ④ **决策阈值**：概率降到<50%时考虑止损，升到>85%时考虑加仓 |
| **与我相关** | 昆仑万维持有的正确思维：不是"我错了还是对了"，而是"在目前37.22元这个价格上，基于所有已知信息（亏损/商誉/AI概念），上涨到45以上的概率还有多大？" 如果概率低于阈值，就应该执行卖出计划。 |
| **来源** | 托马斯·贝叶斯(1763) / 纳特·西尔弗《信号与噪声》/ 安妮·杜克《对赌》 |
| **→规范映射** | 投资行为规范§4.2：所有决策应基于更新后的信息集，而非初始买入逻辑 |

# 知识卡片：凯利公式

| 字段 | 内容 |
|:----|:------|
| **标题** | 凯利公式 — 仓位大小的数学边界 |
| **核心观点** | 凯利公式回答了"在知道胜率和赔率的前提下，应该押多大仓位"的问题。公式核心：**永远不要下注超过凯利比例，否则长期必然爆仓**。 |
| **公式** | f* = (bp - q) / b = (赔率×胜率 - 败率) / 赔率 |
| | 其中：f*=最优仓位比例，b=赔率（盈利/亏损），p=胜率，q=1-p=败率 |
| **投资中的应用** | ① **单只股票上限**：凯利建议的比例通常不超过25-30%（即使非常确定）。你的昆仑万维69.5%远超任何理性模型的建议。② **分散化含义**：按凯利半仓模型(f*/2)，合理单票仓位应≤15%。③ **对当前决策**：69.5%的仓位不是一个可以"持有等待"的仓位——即使你判断它有60%概率涨回45，凯利建议仓位也仅约20%。 |
| **与我相关** | 当前昆仑万维69.5%的集中度是凯利公式的典型反面案例。使用半凯利模型(f*/2)，合理仓位约15-20%。即使你认为这只股票有巨大潜力，仓位也应该比现在小得多。这就是为什么要设定"反弹到45-50卖出一部分"——不是为了止损，而是为了把仓位降到安全范围内。 |
| **来源** | 约翰·凯利(1956) / 爱德华·索普《战胜庄家》/ 马科维茨现代投资组合理论 |
| **→规范映射** | 投资行为规范§5.1：单只股票仓位≤40%（这是凯利公式的简化安全版本） |
