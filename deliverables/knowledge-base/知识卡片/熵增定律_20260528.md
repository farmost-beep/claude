# 熵增定律

**一句话**：封闭系统中，事物天然从有序走向混乱——维持秩序需要持续注入能量。

**核心要点**：
- 热力学第二定律：孤立系统的熵（混乱度）永不减少，只会增加或不变——房间不收拾就会乱、技能不练就会退步、组织不管理就会官僚化
- 对抗熵增的唯一方法：把系统打开，持续注入外部能量——学习是给认知系统注入能量、运动是给身体系统注入能量、反思是给决策系统注入能量
- 熵增的最后阶段是"热寂"——系统达到最大混乱度、失去做功能力——个人层面的表现是彻底躺平、丧失目标感

**与我相关**：六大目标体系（投资、事业、家庭、健康、AI能力、知识库）的日常维护本质上是对抗熵增。知识卡片如果不持续新增和复习，就会从有序知识退化为零散信息。健康基线如果不持续测量，也会从量化管理退化为"凭感觉"。每周的复盘、每月的目标检视，都是在给人生系统注入对抗熵增的能量。

**来源**：热力学第二定律，克劳修斯(1865)；管理应用见彼得·德鲁克对组织熵增的论述
