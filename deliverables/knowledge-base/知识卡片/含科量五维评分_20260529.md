# 含科量五维评分

**一句话**：让银行看得懂科技企业——从研发强度、专利密度、技术壁垒、人才结构、产业位势五个维度量化"含科量"，评分结果直接挂钩贷款比例、利率和期限。

**核心要点**：
- 五维框架：(1)研发强度——研发投入/营收，关注连续3年趋势而非单年；(2)专利密度——发明专利占总专利比×PCT国际专利申请数，区分"数量型"和"质量型"；(3)技术壁垒——技术被替代的成本有多高？竞争对手绕过需要多长时间？(4)人才结构——核心技术人员占比+平均从业年限+过去2年离职率；(5)产业位势——在产业链中的地位（是定价者还是接受者？替代品可及性？）
- 与传统评分的关键差异：不含营收门槛、不含利润要求、不含抵押率——这三点是传统框架"卡死"科技企业的三个阀门
- 风控逻辑：评分不替代风险判断，而是让风险判断从"说不清为什么行/不行"变成"这五个维度各得几分，加权总分反映了什么"。可讨论、可审计、可迭代

**与我相关**：这是邮储科技金融差异化竞争的核心工具。当前银行间科技贷款竞争主要比价格（利率），有了评分体系后就可以比判断力——"我们不是因为利率低才选你，是因为我们真的理解你的技术价值"。每做一笔贷款更新一次评分表，6个月后就可以积累邮储自有的科技企业信用数据库。
