# 科技金融五层生态位

**一句话**：一个银行家如果只看到"这笔贷款"，看不到背后五层生态系统（制度/语言/免疫/循环/适应），就是在盲人摸象。

**核心要点**：
- 骨骼层（制度框架）：并购贷款新规（金规〔2025〕27号）解决了"能不能做"。三大突破——首次允许参股型并购贷款（≥20%持股）、提比例（60%→70%）、延期限（7→10年）
- 神经层（通用语言）：技术价值评估标准，当前全行业空白。银行/VC/监管各说各话——VC说不清"为什么A轮估值10亿"，银行说不清"AI算法为什么值5亿"。建立共同标准是整个生态运转的前提
- 免疫层（风险消化）：增信基金+科技保险双层分担。担保基金覆盖第一损失层0-20%，科技保险覆盖20-50%，银行自留50%。三重分担机制将银行实际风险敞口压缩至可商业化运营水平
- 循环层（资金流转）：银团贷款+ABS试点。科创并购贷款ABS在中国尚属空白——首单的定义权和定价权都在先行者手中
- 适应层（政策进化）：从18城试点到全国推广，通过民建/政协/行业协会渠道反馈市场信号给政策制定方

**与我相关**：五层生态位框架已提交为2026年民建社情民意提案。对邮储的价值——在哪个生态层发力，就占有哪个生态层的定义权。邮储应该优先攻克神经层（建立技术价值评估方法论），然后在循环层推动首单ABS试点。
