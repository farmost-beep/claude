# 事业决策规范

## 核心评估维度

| 维度 | 说明 | 权重 |
|:---|:---|:---:|
| 专业权威积累 | 科技金融方法论产出、文章发表、行业影响力 | 35% |
| 硬条件升级 | 中级经济师→高级经济师、职称、证书 | 25% |
| 社会资源网络 | 民建/中科大/浙江/邮储四重网络拓展 | 25% |
| 当前岗位表现 | 科技金融事业部工作质量、上级评价 | 15% |

## 判断标准

**好**：
- 每季度有至少1篇专业产出（文章/报告/建言）
- 中级经济师备考按计划推进（每周完成预定章节）
- 四重网络中至少2个有实质性互动（非点赞，是讨论/推荐/合作）
- 岗位工作零重大失误，至少1项工作超出预期

**需要关注**：
- 连续2个月无外部可见产出
- 备考进度落后计划>2周
- 社会网络互动出现"断联"——某重网络超过1个月无实质性互动

**有问题**：
- 连续1个季度无产出，或产出质量明显下降
- 备考严重落后（>1个月）
- 岗位出现重大失误或上级负面评价
- 任何一重网络出现"人脉流失"（关键联系人离开或关系恶化）

## 风险信号

1. 岗位工作量突然大幅增加 → 检查是否挤占了备考和产出时间，必要时向上沟通
2. 连续3次民建/行业活动未参加 → 社会可见度下降，需要立即调整优先级
3. 上级/同事对你的定位从"科技金融专家"变成"普通干部" → 专业权威在内部被边缘化
4. 中级经济师某科目连续2次模拟考不及格 → 需要调整备考策略或寻求辅导

## 产出标准

- **公众号文章**：3500-5000字，有原创框架/数据/观点，非知识搬运
- **民建建言**：1500-3000字，有具体政策建议+实施路径，非空泛呼吁
- **学术文章**：按投稿期刊格式，有文献综述+方法论+实证/案例分析
- **行业评论**：800-1500字，对一个热点事件的独立判断，非观点摘编

## 我必须回答的问题（每季度）

1. 这个季度我最重要的一项专业产出是什么？对谁产生了影响？
2. 四重网络中，哪一重有实质性进展？哪一重停滞了？为什么？
3. 中级经济师备考进度是否在计划内？最需要加强的是哪个科目？
4. 我的"科技金融专家"定位在内部和外部是否还在强化？有没有偏离？
5. 下一个季度我应该做的一件事——能产生最大杠杆效应的事是什么？
