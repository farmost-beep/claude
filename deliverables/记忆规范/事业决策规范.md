# 事业决策规范

> v3.0 三层结构：引擎（你投入什么）→ 加速器（什么放大了引擎）→ 结果（你得到什么）。层次不混淆，因果关系可追溯。

## 定位：事业是投资的发动机

事业不是独立目标。事业产出三样东西，每一样都直接服务于投资成功：

| 产出 | 说明 | 对投资的作用 |
|:---|:---|:---|
| **现金流** | 工资+奖金+其他收入 | 可投资本金的持续来源 |
| **信息优势** | 科技金融行业一线的政策/产业/企业信息 | 投资决策的信息Alpha |
| **社会杠杆** | 四重网络（民建/中科大/浙江/邮储） | 资金渠道+交易机会+知识网络 |

**核心公式**：事业产出 = 现金流 × 信息优势 × 社会杠杆。乘数关系——任何一个为零，总产出归零。

---

## 三层结构

```
第一层：引擎          第二层：加速器        第三层：结果
（你投入什么）        （什么放大了引擎）    （你得到什么）

专业产出力 ──→ AI杠杆率 ──→ 位置安全（底线）
     │                           │
网络经营力 ──→              ├─→ 资源获取（增长）
                                 │
                                 └─→ 资本转化（终极）
```

每一层有且只有明确的因果关系：引擎不给力→检查AI杠杆是否够→结果没跟上→检查引擎是否对准了需求。

## 第一层：引擎（你投入什么）

### 维度1：专业产出力 — 权重40%

**定义**：你能产出多少被他人认可的专业价值。含文章/建言/报告/项目/授信分析。

| 等级 | 标准 |
|:---|:---|
| **强** | 每季度≥2篇外部可见产出，且至少1篇被引用/转发/讨论 |
| **正常** | 每季度1篇外部可见产出，或每1-2月有内部可见产出 |
| **弱** | 连续2月无任何可见产出，或产出无人关注 |

**领先指标**（不是滞后指标）：
- 这周写了什么别人能看到的东西？
- 最近一次有人因为你的产出来找你讨论/合作是什么时候？
- 你的产出是否对准了"科技金融"这个标签？

**AI加速点**：文章→Claude写初稿人注入观点；建言→Claude收集政策背景人定论点；授信报告→AI预审人复核关键点。目标产出速度3×以上。

### 维度2：网络经营力 — 权重30%

**定义**：四重网络中产生实质性机会的能力。实质性=讨论/推荐/合作/引荐，非点赞/评论。

| 网络 | 关键联系人 | 最近互动 | 产生机会 |
|:---|:---|:---|:---|
| 民建 | （填入） | （日期） | （描述） |
| 中科大 | （填入） | （日期） | （描述） |
| 浙江 | （填入） | （日期） | （描述） |
| 邮储 | （填入） | （日期） | （描述） |

**判断标准**：
- **强**：≥2个网络有实质性互动，过去3个月至少带来1个新机会
- **正常**：≥1个网络有实质性互动，无机会但有关注
- **弱**：某网络断联>1月，或互动只停留在点赞层面

**领先指标**：
- 上周主动联系了几个关键联系人？（不是等他们找你）
- 最近一次有人因为你而认识了另一个人？（你在网络中的节点价值）
- 如果你明天需要找一个业务机会/投资信息/政策解读，有没有人能一个电话搞定？

**注意**：网络经营不靠"维护关系"——靠"产出值得被传播的东西"。产出力是经营力的燃料。没有产出的社交是消耗。

## 第二层：加速器（什么放大了引擎）

### 维度3：AI杠杆率 — 权重30%

**定义**：AI在多大程度上放大了产出力和经营力。这是乘数，不是加数。

| 场景 | 无AI耗时 | AI加速后 | 当前倍数 | 目标倍数 |
|:---|:---:|:---:|:---:|:---:|
| 公众号文章 | 8-12h | 2-3h | 3-4× | 5× |
| 民建建言 | 5-8h | 1-2h | 3-5× | 5× |
| 中级经济师备考 | 320h总 | AI梳理+出题 | 省30-40% | 省50% |
| 行业研究 | 2-3天 | 半天 | 4-6× | 6× |
| 授信报告 | 2-4h | AI预审+人复核 | 5-10× | 10× |
| 网络经营 | 手动整理 | AI生成互动提醒 | 0×（未用） | 2× |

**判断标准**：
- **强**：AI辅助事业产出频率≥3次/周，至少3个场景达到目标倍数
- **正常**：AI辅助频率1-2次/周，场景倍数在提升
- **弱**：连续2周未用AI辅助事业产出，或使用频率在下降

**关键认知**：AI杠杆率不是一个独立的工作项——它是"用AI做你本来就要做的事"。不是多一件事，是把事做得快3-5倍。

## 第三层：结果（你得到什么）

三个结果指标是验证前两层是否有效的硬数据。如果引擎在转、加速器在工作，但结果没跟上——说明方向错了，不是努力不够。

### 结果1：位置安全（底线指标，通过/不通过）

**不是目标，是底线。** 如果这个不通过，其他全是空谈。

| 信号 | 状态 |
|:---|:---|
| 上级最近一次正面反馈是什么？ | >3个月无→⚠️ |
| 你的工作有没有人能做？有多少人能做？ | 很多人能做→⚠️ |
| "科技金融"标签在上级心中是否与你绑定？ | 模糊或未提及→⚠️ |
| 最近是否有重要工作被分配给其他人？ | 有→🔴 |

**通过**：无红色信号，黄色信号≤1个。**不通过**：任何红色信号，或黄色信号≥2个。

### 结果2：资源获取（增长指标）

**定义**：收入、职位、资源是否在增长。含工资/奖金/职级/权限/团队。

- **增长**：年收入增速≥通胀+5%，或职级/权限有明显提升
- **持平**：年收入增速与通胀持平，职级权限不变
- **下降**：收入实际下降，或权限被削减

### 结果3：资本转化（终极指标）

**定义**：年收入中有多少进入了投资账户（含还本专户）。

- **好**：储蓄率≥60%
- **正常**：储蓄率40-60%
- **需要关注**：储蓄率<40%，或连续3月下降

## 判断引擎

三层联动判断，不是独立打分：

```
引擎在转 + 加速器够 + 结果在涨 = ✅ 飞轮正常
引擎在转 + 加速器不够 + 结果在涨 = ⚠️ 你还有AI杠杆未用，用了会更快
引擎在转 + 加速器够 + 结果不涨 = 🔴 方向错了，产出没对准需求
引擎不转 + 加速器够 + 结果靠惯性 = 🔴 你在吃老本，引擎停转迟早暴露
引擎不转 + 加速器不够 + 结果在跌 = 🔴🆘 四项全红，需要根本性改变
```

---

## 风险信号（领先指标）

1. **上级开始把你的工作分配给其他人** → 位置安全红灯。立即检查：最近3个月的工作中有什么是只有你才能做的？
2. **你最近一次在外部被提及/引用/邀请 >3个月** → 产出力在下降，定位在被遗忘
3. **"科技金融"在日常工作中实际占比 <30%** → 你在做普通银行干部的工作，定位在稀释
4. **最近一次主动联系关键联系人 >2周** → 网络经营在断联
5. **AI辅助事业产出频率 <每周1次** → 杠杆率在下降，在用手动模式工作
6. **储蓄率连续3月下降** → 资本转化率恶化，事业产出没变成投资弹药

## 飞轮与卡点检测

```
专业产出 ──→ 外部可见度 ──→ 网络机会 ──→ 资源获取
    ↑                                          ↓
    │                                          ↓
AI加速 ←────────── 更多时间 ←────── 更高收入
    │                                          ↓
    └──────────────────────────→ 更多本金 → 投资成功
```

**卡在哪里**：
- 产出没增加 → 瓶颈在"写不出来"（用AI加速）
- 产出有但没人看 → 瓶颈在"分发渠道"（检查四重网络中谁在帮你传播）
- 有人看但没机会 → 瓶颈在"转化"（产出是否对准了需求方的痛点？）
- 有机会但收入没增 → 瓶颈在"变现时机"（是否到了可以谈的条件窗口？）
- 收入增但资本没增 → 瓶颈在"储蓄纪律"（钱花到哪里去了？）

## 与AI能力发展的协同

三层结构与AI能力五维度的对应：

| 事业层 | 对应AI能力维度 | 协同方式 |
|:---|:---|:---|
| 专业产出力 | 规范力 | 产出需要规范驱动——写作模板、研究框架、审核清单 |
| 网络经营力 | 编排力 | 网络经营需要多Agent并行——联系人追踪、互动提醒、机会评估 |
| AI杠杆率 | 扩展力+编排力 | AI辅助产出的能力边界扩展，多场景Agent组编排 |
| 结果验证 | 验证力 | AI检查储蓄率、产出频率、网络互动频率，交叉验证数据一致性 |
| 方向判断 | 感知力 | AI扫描行业/政策/机会信号，辅助"什么事值得做"的决策 |

## 每季度必须回答的问题

1. **产出力**：这个季度写了什么被人看到的东西？有没有引发讨论或机会？
2. **经营力**：四重网络中哪个产生了实质机会？哪个在萎缩？为什么？
3. **杠杆率**：AI在这个季度帮我省了多少时间？哪个场景的加速倍数在提升？
4. **位置**：上级和同事怎么看你？"科技金融专家"的标签是在清晰还是模糊？
5. **资源**：收入/职位/权限有没有实质性增长？如果没有，卡在飞轮的哪个环节？
6. **资本**：这个季度收入中有多少进入了投资账户？转化率是升了还是降了？
