# 家庭管理规范

## 核心关注维度

| 维度 | 说明 | 回顾频次 |
|:---|:---|:---:|
| 子女成长 | 学业+兴趣+品格+心理健康 | 每周 |
| 配偶关系 | 沟通质量+共同目标+情感支持 | 每周 |
| 家庭财务 | 收支平衡+储蓄率+大额支出规划 | 每月 |
| 父母关怀 | 健康状况+情感联系+实际支持 | 每月 |
| 家庭时间质量 | 共同时光的质量（非时长） | 每周 |

## 判断标准

**好**：
- 每周至少3次高质量家庭共处（吃饭+聊天+活动，手机放下）
- 子女学业稳定或进步，情绪状态良好
- 配偶之间每周有至少1次"只聊我们俩，不聊孩子不聊钱"的对话
- 家庭月度储蓄率达到目标

**需要关注**：
- 连续2周高质量家庭共处<2次
- 子女出现持续的情绪低落或成绩明显下滑
- 配偶沟通中有"你最近都不在家/不在状态"的信号
- 家庭意外支出>月度预算20%

**有问题**：
- 子女出现行为问题（学校反馈、社交退缩、沉迷电子产品）
- 配偶直接表达不满或出现长期冷战
- 家庭储蓄率持续下降>3个月
- 父母出现需要你立即介入的健康问题

## 风险信号

1. 工作占用家庭时间连续2周>60小时/周 → 立即调整，优先确保周末家庭时间
2. 子女说"爸爸/妈妈每次都说'等忙完这阵'"超过3次 → 信任危机前兆
3. 配偶不再主动分享ta的日常 → 情感连接断裂的早期信号
4. 家庭大额支出（超过月收入50%）没有提前讨论 → 财务沟通机制失效
5. 你连续1个月没有主动和父母联系 → 立即打电话

## 每周家庭回顾模板

```
本周家庭回顾：
1. 高质量共处：[X]次，最开心的一次是：[描述]
2. 子女：学业[正常/进步/退步]，情绪[开心/一般/低落]，需要我：[具体]
3. 配偶：这周和ta聊了[话题]，ta的心情状态是[描述]，我有件事需要和ta商量：[描述]
4. 家庭支出：本周[正常/超支]，原因是：[描述]
5. 下周我打算：[一件改善家庭关系的事]
```

## 我必须回答的问题（每月）

1. 这个月我在家庭上花的时间够吗？质量好吗？
2. 子女最需要我的是什么——不是辅导作业，是倾听、陪伴、还是引导？
3. 配偶这个月有没有说过什么我没认真听的话？
4. 我有没有因为投资/工作的压力把情绪带回家？
