# 健康追踪规范

> v2.0 三层因果链：输入（你直接控制的）→ 过程（每周可见的）→ 结果（体检验证的）。不是盯10个数字，是盯5个因果相连的指标。

## 定位：健康是百岁人生的发动机

所有目标的底线。身体不行，投资、事业、家庭全是空中楼阁。

**核心认知**：49岁是代谢问题可逆转的黄金窗口期。2025年9月体检发现18项异常——不是18个坏消息，是18个"趁现在还来得及"的提醒。

> 完整医学分析、18项异常逐一方案、三阶段百岁路线图、上海三甲医院清单 → 见 **《百岁健康计划》**。本文件是追踪框架，不是医学指南。

## 三层指标体系

```
你直接控制的（输入）  →  每周可见的（过程）  →  体检验证的（结果）

运动完成率  ──→  腰围  ──→  LDL-C
睡眠时长    ──→  血压  ──→  HbA1c
饮水量                      →  体重/BMI
```

因果关系：输入达标 → 过程必然改善 → 结果在下次体检时自然好转。不需要同时盯10个数字。

---

## 5个核心指标

### 输入层（你直接控制的，做了就是做了）

| 指标 | 目标 | 频率 | 为什么选它 |
|:---|:---|:---:|:---|
| **运动完成率** | 每周有氧≥3次 + 力量≥1次，达标率≥80% | 每周 | 唯一被证明能同时降血压、降LDL、减腰围、逆转脂肪肝的干预手段 |
| **睡眠时长** | 日均≥7小时，无连续3天<6小时 | 每天 | 睡眠不足→意志力崩溃→运动饮食计划一起垮。所有健康行为的底座 |
| **饮水量** | ≥2.5L/天（肾结石预防） | 每天 | 针对双肾多发结石。多饮水是唯一被证实的非药物预防手段 |

### 过程层（每周可见变化，验证输入是否有效）

| 指标 | 当前值 | 目标值 | 频率 | 为什么选它 |
|:---|:---:|:---|:---:|:---|
| **腰围** | 87cm | <85cm（6个月） | 每周 | 直接测量内脏脂肪。比BMI精准，比体脂秤可靠（软尺不会骗人） |
| **晨起血压** | 138/90 | <130/85（3个月） | 每天 | 已经在高血压警戒线。4周趋势就能判断生活方式干预有没有用 |

### 结果层（体检时验证，不做日常追踪）

LDL-C、HbA1c、体重/BMI、尿酸 → 纳入**半年一次**的体检追踪，不作为每周指标。这些是输入和过程改善后的自然结果，不是日常盯的数字。

## 为什么删掉v1.0的5个指标

| 删除的指标 | 理由 |
|:---|:---|
| 空腹血糖 | 半年测一次，纳入体检追踪。不放在每周核心指标里 |
| 血脂三项 | 同上。LDL-C 3.68增高已在P0计划中，3个月后复查 |
| 尿酸 | 287正常。每年体检跟踪即可 |
| 静息心率 | 受咖啡因/压力/测温时间干扰太大，信号噪声比低 |
| 体脂率 | 家用体脂秤误差±3-5%，腰围用软尺更可靠 |
| 深度睡眠占比 | 需要专业设备，不如"睡了多久"直接可测 |

## 判断标准

**好**：
- 运动完成率≥80%（即每个月4周中至少3周达标）
- 睡眠日均≥7小时，无连续3天<6小时
- 饮水量日均≥2L（不低于这个底线）
- 腰围趋势向下（即使每周波动，月均趋势下降）
- 晨起血压趋势向下或稳定<135/85

**需要关注**（任何一个触发）：
- 运动完成率连续2周<50%
- 连续3天睡眠<6小时
- 腰围连续4周无下降趋势（平台期>1个月）
- 晨起血压连续2周>140/90

**有问题**（任何一个触发）：
- 连续1个月运动基本停滞（完成率<30%）
- 腰围上升>2cm（从最低点反弹）
- 晨起血压>150/95
- 任何部位的持续疼痛>2周
- **立即行动**：血压>160/100 → 当天就医，不等

## 风险信号（领先指标）

1. **晨起血压连续2周上升趋势**（如从135→140→145） → 检查盐摄入/运动频率/睡眠/压力。4周趋势不降则就医。
2. **腰围停止下降>4周** → 检查饮食记录是否在"无意识摄入"（外卖酱料/应酬酒精/零食）
3. **连续3天睡眠<6小时** → 排查：睡前看行情？工作邮件？咖啡过了14:00？打鼾/呼吸暂停？
4. **连续2周运动完成率<50%** → 不是"懒"，是有什么在阻碍（加班？受伤？失去兴趣？）。找出障碍，调整计划。
5. **任何部位持续疼痛>2周** → 就医，不拖延。
6. **体重非意愿性下降>3kg/月** → 就医排查（49岁男性非意愿体重下降需要排除器质性疾病）
