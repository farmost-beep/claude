# 健康追踪规范

## 核心指标体系

| 指标 | 目标值（49岁男性） | 测量频率 | 数据来源 |
|:---|:---|:---:|:---|
| 体重（BMI 22-24） | 目标体重范围内 | 每周 | 体重秤 |
| 静息心率 | 60-70 bpm | 每周 | 手表/手环 |
| 睡眠时长 | 7-8小时/天 | 每天 | 手表/手环 |
| 睡眠质量（深度睡眠占比） | >20% | 每周 | 手表/手环 |
| 每周运动次数 | ≥3次 | 每周 | App追踪 |
| 每周运动时长 | ≥150分钟 | 每周 | App追踪 |
| 血压 | <130/85 mmHg | 每月 | 血压计 |
| 空腹血糖 | <6.1 mmol/L | 每季度（体检） | 体检报告 |
| 血脂（LDL/HDL/TG） | LDL<3.4, HDL>1.0, TG<1.7 | 每半年（体检） | 体检报告 |
| 尿酸 | <420 μmol/L | 每半年（体检） | 体检报告 |

## 判断标准

**好**：
- 所有核心指标在目标范围内
- 运动频率和时长达标
- 睡眠质量稳定（深度睡眠>20%）
- 精力充沛，白天不依赖咖啡/茶

**需要关注**：
- 1-2项指标连续2周偏离目标（如体重↑、睡眠↓）
- 运动频率或时长不达标连续2周
- 连续3天睡眠<6小时
- 出现反复的身体不适（头痛、腰痛、胃痛等）

**有问题**：
- 3项以上指标偏离目标，且趋势持续恶化
- 连续1个月运动不达标
- 体检指标出现红色预警（血糖、血脂、血压任一超出正常范围）
- 精力持续低迷影响正常工作和生活

## 风险信号

1. 体重连续4周上升且无平台期 → 检查饮食结构和运动强度
2. 静息心率连续上升>10 bpm（如从65→75） → 可能过劳或潜在健康问题
3. 连续3天入睡时间>30分钟 → 排查压力源、咖啡因、屏幕时间
4. 血压>140/90 → 立即预约医生，不等待下次体检
5. 任何部位的持续疼痛>2周 → 就医检查

## 我必须回答的问题（每季度）

1. 这个季度的健康趋势是向好还是向坏？（用数据，不用感觉）
2. 最大的健康风险是什么？（睡眠/运动/饮食/压力）
3. 上次体检的异常指标有没有改善？
4. 我的运动计划还需要调整吗？（强度/类型/时间）
5. 有没有我应该看医生但一直在拖的问题？
