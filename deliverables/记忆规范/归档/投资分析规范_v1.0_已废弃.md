# 投资分析规范

## 核心指标体系（满分100分加权评分）

| 指标 | 权重 | 数据来源 |
|:---|:---:|:---|
| 营收增速（YoY + QoQ） | 15% | 季度财报 |
| 毛利率 + 8季度趋势线 | 15% | 季度财报 |
| 经营现金流 / 净利润（盈利质量） | 20% | 季度财报+现金流量表 |
| ROE（杜邦拆解） | 15% | 季度财报 |
| 应收账款周转天数 | 10% | 季度财报 |
| 研发费用率 + 资本化比例 | 10% | 季度财报附注 |
| 前5大客户集中度 | 5% | 年报/招股书 |
| 人均产出（营收/员工数） | 5% | 年报 |
| 估值：PEG + 历史分位数 | 5% | 市场数据 |

## 对比维度

- **公司自身**：近8季度趋势
- **行业对标**：与行业75分位数对比
- **组合内排序**：持仓+自选池横向排名

## 判断标准

**好（综合评分≥70）**：
- 营收和现金流同向增长
- 毛利率稳定或上升
- 应收账款增速≤营收增速
- 研发资本化比例稳定

**需要关注（综合评分50-69）**：
- 营收增长但现金流下降
- 毛利率连续2季度下降1-3个百分点
- 应收账款增速略高于营收增速

**有问题（综合评分<50）**：
- 营收和现金流反向
- 毛利率连续2季度下降>3个百分点
- 应收账款增速远高于营收增速
- 前5大客户集中度>50%且其中一个客户占比>25%

## 风险信号（任一触发即深度排查）

1. 营收增长但经营现金流连续2季度下降 → 检查应收账款账龄、渠道压货
2. 毛利率连续2季度下降>3个百分点 → 检查竞争格局、成本结构
3. 研发资本化率突然上升>10个百分点 → 排除利润调节
4. 前5大客户集中度>50% → 逐一评估每个大客户的稳定性
5. 应收账款增速 > 营收增速 → 检查回款能力和坏账计提
6. 关联交易金额>营收10% → 检查定价公允性
7. 管理层大规模减持 → 检查减持原因和节奏

```
Q[N]，[公司名] 综合评分 [X]/100，[优于/持平/弱于]上季度。

操作建议：[加仓/持有/减仓/观察]，理由一句话。
```

## 我必须回答的问题

对每家有异常的公司：
1. 异常是周期性的还是结构性的？
2. 管理层的解释是否可信？（对照历史行为验证）
3. 市场是否已经反映了这个异常？（看估值变化）
4. 如果最差情况发生，我的最大亏损是多少？
5. 这笔投资的核心逻辑有没有被破坏？
