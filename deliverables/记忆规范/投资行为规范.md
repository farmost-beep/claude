# 投资行为规范

> 这份文件约束的不是"怎么分析"，是"怎么行动"。分析框架告诉你一家公司好不好。行为规范告诉你——知道了以后，该做什么、不该做什么、什么时候做、做多少。

---

## 第一章：投资哲学——我的底层假设

这些不是"我觉得"，是"我相信"。相信的意思是——即使短期结果不利，也坚持。

### 1.1 我是投资者，不是交易者

- 我赚的是企业价值增长的钱，不是市场波动的钱
- 持有期以年为单位，不以天/周为单位
- 不盯盘。价格波动不是信息，是噪音

### 1.2 安全边际是第一原则

- 任何投资在买入时就要有安全边际——买入价低于内在价值至少30%
- 安全边际不是"跌了再补"的借口。买贵了就是买贵了，跌了可能更贵
- 如果商业模式不可逆（网络效应极强的平台），安全边际可降至20%
- 如果管理层有道德瑕疵，安全边际必须>50%——或者直接不投

### 1.3 能力圈是铁律

- **圈内**：AI产业链（半导体、算力、应用）+ 银行科技金融
- **圈边缘**：新能源、Biotech。需要至少6个月系统学习才能开始建仓，学习期只买观察仓（≤2%仓位）
- **圈外**：不投。无论多少人赚钱、无论故事多好。错过不可惜，做错会亏钱
- 能力圈的边界每半年review一次。可以扩展，但不能临时扩展（"这个机会太好了我先买再说"=违规）

### 1.4 市场先生是仆人，不是向导

- 市场先生的出价是给你利用的，不是给你当参考的
- 他恐惧时你贪婪（买入），他贪婪时你恐惧（卖出/持有）
- 股价下跌不是"你做错了"的信号。买入逻辑没破坏，下跌是加仓机会；买入逻辑破坏了，跌再多也要卖

### 1.5 不杠杆

- 不用借来的钱投资
- 不融资、不配资、不买期权（除非是卖出put来接货，且有足额现金备付）
- 杠杆把"暂时亏损"变成"永久亏损"。投资已经够难了，不需要给自己加难度

## 第二章：买入规则——什么时候可以买

### 2.1 买入前必须完成的动作

**以下全部完成之前，不下单：**

1. [ ] 读完最近3份季度财报+2份年度财报
2. [ ] 按核心指标体系（第三章）完成评分，综合评分≥70
3. [ ] 读完至少3份不同券商研报，并用交叉审查识别分歧点
4. [ ] 写下买入逻辑（≤200字）：我为什么买、我的优势在哪（我看到了什么市场没看到的东西）
5. [ ] 写下最差情况：如果逻辑全错，最大亏损是多少、我能不能接受
6. [ ] 确定买入价格上限（DCF估值×0.7或等价的合理估值折让）
7. [ ] 确定仓位上限（按第四章的规则）
8. [ ] 等至少24小时再下单（冷却期，避免冲动）

### 2.2 禁止买入的情况

以下任一条满足，不买：

- 买入逻辑中有"大家都说好""最近涨了很多""XX大佬也买了"或类似的社会证明
- 在强烈情绪状态下做决策（兴奋、焦虑、FOMO、报复性交易）
- 为了"摊低成本"而加仓已经亏损的头寸（这不是买入理由——买入的唯一理由是"这家公司好且便宜"）
- 股价在1个月内涨了50%以上——不在狂热期追高
- 看不懂公司怎么赚钱的——商业模式不清晰的不投

### 2.3 分批建仓规则

- 首次建仓不超过目标仓位上限的50%
- 剩余50%在以下时机分批加：①下一份季报验证了买入逻辑 ②股价下跌10%以上且逻辑未被破坏
- 一只股票从零到满仓至少经历2个季度。不给任何公司"一次性信任"

## 第三章：分析框架——买了以后怎么追踪

### 3.1 核心指标体系（满分100分）

| 指标 | 权重 | 说明 |
|:---|:---:|:---|
| 营收增速（YoY+QoQ） | 15% | 增长是价值的源泉 |
| 毛利率+8季度趋势 | 15% | 趋势比绝对值重要 |
| 经营现金流/净利润 | 20% | 盈利质量，最重的一票 |
| ROE（杜邦拆解） | 15% | 股东回报的效率 |
| 应收账款周转天数 | 10% | 回款能力 |
| 研发费用率+资本化比例 | 10% | 创新投入的质量 |
| 前5大客户集中度 | 5% | 收入稳定性 |
| 人均产出 | 5% | 运营效率 |
| 估值（PEG+历史分位） | 5% | 不买贵的 |

### 3.2 季度追踪标准

每季度财报发布后，对持仓和自选池中每家公司：

**好（≥70）**：营收和现金流同向增长；毛利率稳定或上升；应收≤营收增速
- → 持有。如果估值进入历史低位30%分位，考虑加仓

**关注（50-69）**：营收增但现金流降；毛利率连续2季下降1-3pp；应收略高于营收增速
- → 暂不加仓。下季度再评估。连续2季"关注"→启动深度排查

**有问题（<50）**：营收现金流反向；毛利率连续2季降>3pp；客户集中度>50%
- → 立即启动深度排查。72小时内决定：持有/减仓/清仓

### 3.3 风险信号（任一触发，立即排查）

1. 经营现金流连续2季度下降 → 查应收账款账龄、渠道压货
2. 毛利率连续2季度下降>3pp → 查竞争格局、成本结构变化
3. 研发资本化率突然上升>10pp → 排除利润调节
4. 前5大客户集中度>50%，且单一客户>25% → 逐一评估客户稳定性
5. 应收账款增速>营收增速 → 查回款和坏账计提
6. 关联交易>营收10% → 查定价公允性
7. 管理层大规模减持 → 查原因和节奏
8. 核心管理层（CEO/CFO/CTO）离职 → 直接影响买入逻辑中对管理层的判断
9. 审计师辞职或更换 → 财务质量红旗

## 第四章：仓位与风险管理——投多少、怎么控

### 4.1 仓位上限

| 类型 | 单只股票上限 | 说明 |
|:---|:---:|:---|
| 核心持仓（圈内，评分≥80，持有>1年） | 15% | 高度确信的长期持有 |
| 一般持仓（圈内，评分≥70） | 10% | 正常持有 |
| 观察仓（圈边缘/圈内评分50-69/新建仓<2季） | 2% | 试水不重仓 |
| 圈外 | 0% | 不投 |

### 4.2 组合层面约束

- 单行业（如半导体）总仓位≤40%
- AI产业链总仓位≤60%（虽然这是能力圈，但集中度有上限）
- 现金仓位≥10%（永远保留一笔"格雷厄姆式"的现金，用来在市场恐慌时买）
- 海外资产总仓位≤30%（汇率风险+信息劣势）

### 4.3 最大回撤控制

- 单只股票自买入价下跌20% → 强制复盘：买入逻辑是否被破坏？
  - 破坏了 → 卖出，不犹豫
  - 没破坏 → 不动或加仓（只加一次，不超过原仓位的50%）
- 组合总资产自高点回撤15% → 暂停所有新买入，只做持有/卖出决策
- 组合总资产自高点回撤25% → 强制全面复盘。检查是否组合整体出了问题（杠杆、集中度、方向性错误）

## 第五章：卖出规则——什么时候必须卖

买入是选择题，卖出是必答题。卖出规则比买入规则更重要——因为人在亏损时最容易做出非理性决策。

### 5.1 强制卖出（触发即执行，不犹豫）

1. **买入逻辑被破坏**。这是唯一最重要的卖出理由。不是在下跌时恐慌卖出，而是"我当时买它的核心理由，现在不成立了"。
2. **发现财报造假或管理层不诚信**。立刻清仓，不计成本。诚信问题没有第二次机会。
3. **仓位被动超标**（一只股票因上涨导致仓位超过15%上限）。卖出超出部分，不是清仓。这是纪律。
4. **发现更好的机会，且当前持仓有可以替代的**。换仓前完成新标的的全部买入前检查（第二章），并在新旧之间做一次"A vs B，为什么换"的书面比较。

### 5.2 禁止卖出的情况

1. **"跌太多了受不了"**。这恰恰可能是该买的时候。回看第四章的风险信号和买入逻辑——如果都没问题，跌是机会不是威胁。
2. **"涨太多了先落袋"**。除非触及仓位上限需要被动减持。好公司不需要因为"涨多了"而卖——那是交易思维，不是投资思维。
3. **"市场都在说这个板块不行了"**。市场说什么是噪音。你要看的是你自己的分析框架在说什么。
4. **需要钱但不是因为紧急情况**。投资的钱是长期资本。如果需要钱用于消费，说明当初不该投进来。提前规划现金流，不要让投资被迫卖出。

## 第六章：行为禁区——无论如何不能做的事

这些不是"建议"，是"禁令"。每一条背后都有一个别人（或自己）亏过大钱的教训。

### 6.1 决策禁令

1. **不在非固定时间做投资决策**。投资决策只在每周固定的"投资时间"（如周六上午）进行。盘中不做买卖决策。
2. **不在情绪波动时做任何操作**。愤怒、恐惧、狂喜、焦虑→不买不卖。关掉App，明天再说。
3. **不因为"别人赚钱了"而买任何东西**。FOMO是投资的第一杀手。
4. **不因为"回本"而持有**。"等回本了就卖"是最贵的心理账户。这只股票现在的价值跟你的买入价毫无关系。

### 6.2 信息禁令

5. **开盘时间不盯盘**。股价波动不是信息输入，是噪音输入。看盘消耗注意力且容易诱发情绪交易。
6. **不看股吧/论坛/自媒体对持仓个股的讨论**。你不知道发帖的人是什么动机。信息质量为零甚至为负。
7. **不根据单一信息源做决策**。任何信息至少交叉验证两个独立来源。

### 6.3 操作禁令

8. **不日内交易**。买了当天不卖，卖了当天不买回来。T+0是投资者的敌人。
9. **不重仓单一事件**。"等财报出来就涨""等政策出来就涨"——这是在赌博，不是投资。
10. **不向任何人推荐股票**。推荐股票是双输——赚了他们不会分你，亏了他们怪你。你也不应该根据任何人的推荐买股票（包括朋友、大佬、专家）。

## 第七章：节奏与回顾——在什么时间做什么事

### 7.1 固定节奏

| 频率 | 行动 | 时长 |
|:---|:---|:---:|
| **每天** | 不盯盘。不看App。如果收到异常推送（触发第五章风险信号），记录，投资时间内处理 | 0分钟 |
| **每周** | 周度持仓体检Agent跑一次→看异常（不看正常的）→只对有异常的公司做判断 | 15分钟 |
| **每月** | 回顾本月所有交易决策→检查有无违反本规范的条目→记录"这个月我在投资上犯的一个错误" | 30分钟 |
| **每季度** | 财报季：20家公司并行分析→异常聚焦→深度研究异常公司。更新一次组合内排名 | 半天 |
| **每半年** | 能力圈review→投资哲学review→本规范review→更新 | 2小时 |
| **每年** | 年度投资复盘：收益率vs基准、最大错误、最大教训、明年改进方向 | 半天 |

### 7.2 月度自我审计（必答）

1. 这个月有没有违反第六章的任何一条禁令？如有，原因是什么？
2. 这个月做的最差的一个决策是什么？如果重来会怎么做？
3. 这个月有没有"知道该做但没做"的事？（如：该卖的没卖、该查的没查）
4. 目前的持仓中，哪一只我最不确定？（标记，下个月优先研究或考虑减仓）

## 第八章：这份规范本身的维护规则

这份规范不是一成不变的。它会进化，但进化必须有纪律：

1. **修改需要触发条件**：①投资上犯了一个规范没有覆盖到的错误 ②投资上获得了一个规范没有涵盖的正确经验 ③市场环境发生了结构性变化（如新工具、新资产类别出现）
2. **修改前需要冷却期**：不要在刚犯错或刚成功时立刻修改。等至少一周。
3. **修改需要记录原因**：每次修改附修改日志（日期+旧规则+新规则+为什么改）
4. **AI在每次投资相关协作中自动调用本规范**：不只是在分析时调用，在用户提出任何投资相关问题时，先用本规范检查该行为是否在框架内
