## 二、逐个短板深度分析

### 短板1：量化策略实操能力 🔴

**现状**：backtrader、akshare、tushare、autostrategy已安装。量化策略框架由Agent生成，回测由Agent运行。她自己不会独立操作。

**为什么是短板**：
- 策略是别人的代码，自己调不了参数
- 市场变了，策略需要迭代，没有人帮她维护代码
- 如果不懂回测逻辑，可能被过拟合的结果欺骗
- 价值因子轮动策略评分81.5/100，但如果不会自己跑回测，这个分数只是纸上的

**根因**：49岁开始学编程不现实，但"会用工具"和"会写代码"是两回事。

**改进路径**（不需要学编程）：
1. **6月**：学会在Claude Code中运行 `python3 scripts/autostrategy/scripts/run_backtest.py <策略目录>`，看懂回测输出的5个指标
2. **7月**：学会修改 `config.yaml` 中的参数（因子权重、风控阈值），让Agent帮她解释每次改参数后的回测变化
3. **8月**：能独立完成"改参数→跑回测→看结果→判断策略是否退化"的闭环
4. **判断标准**：9月底前，独立完成一次策略参数调整+回测+结果解读

### 短板2：AI产业链行业判断力 🔴

**现状**：AI产业链研究报告已由Agent生成。她知道芯片→算力→模型→应用的层次，但不能独立判断"这家AI公司技术到哪了、值不值这个估值、银行该不该给它贷款"。

**为什么是短板**：
- 科技金融方法论的核心是"技术信用替代抵押信用"——但如果判断不了技术，就替代不了
- 投资AI标的如果不懂技术路线，就是赌博不是投资
- 事业上要推"AI科创贷"，如果自己不懂AI，客户来了一问三不知

**根因**：AI产业链太广，不知从何入手。Agent生成的报告是"别人的知识"，没有内化。

**改进路径**（聚焦3个环节，不做全产业链）：
1. **6月**：只学光模块——中际旭创vs新易盛vs天孚通信，3家对比，看2份研报+1份年报，写出自己的判断
2. **7月**：只学金融AI——恒生电子vs同花顺vs东方财富，理解AI在银行业的落地方式
3. **8月**：只学AI芯片国产替代——海光信息vs寒武纪vs华为昇腾，能说出"为什么寒武纪PE>300还有人买"
4. **判断标准**：9月底前，能独立完成一家AI公司的TRL判断（不靠Agent），准确率由专家复核

### 短板3：投资执行纪律 🔴

**现状**：IPS六条宪法写得漂亮。但昆仑万维还在69.5%的仓位，19个僵尸品种还没清，60万还本资金还没开始调度。IPS一次都没有被市场考验过。

**为什么是短板**：
- 投资最大的敌人是"自己"，IPS就是用来管自己的——但IPS如果从未被违反过，就不知道它是否真的有效
- 减仓昆仑万维是一个"痛苦测试"——她敢不敢在涨的时候卖？敢不敢承认过去的集中持仓是错的？
- 6月第1周投资执行清单已生成，但执行清单≠执行

**根因**：从"规划"到"交易"之间有心理摩擦。写清单容易，点"卖出"按钮难。

**改进路径**：
1. **本周（5/27-6/1）**：完成清单中的"战前准备"（核对合同、致电银行、设闹钟）
2. **6月2日**：执行第一笔清仓交易（选1-2个最小的僵尸品种，金额<5000元），体验"卖出"的心理感觉
3. **6月每周**：按清单执行，不因为涨了就犹豫、不因为跌了就恐慌
4. **判断标准**：6月底前，昆仑万维≤24%（从69.5%降下来），19个僵尸品种全清，还本专户有钱进账

### 短板4：写作与发表能力 🟡

**现状**：科技金融方法论、市场全景、AI产业链研究等文档质量很高。但这些都是"内部文档"。对外发表=0。事业AI加速方案规划了6篇文章，1篇已由Agent生成初稿。

**为什么是短板**：
- 方法论不发表=不存在于行业对话中
- 入库和竞聘要靠"可见的成果"——发表文章是最硬的可量化成果
- 她的事业突破口不在银行内部（竞聘路径已断），在外部影响力

**根因**：完美主义——总觉得不够好、不够完整、还需要再改改。

**改进路径**：
1. **6月第1周**：从Agent生成的发表文章中选1篇，人工修改后投出（不要追求完美，投出去就是胜利）
2. **6月第2周**：写第2篇（AI产业链+科技金融结合点，1500字短评），投公众号或内刊
3. **Q3**：每月至少1篇，累计3篇投稿、争取1篇被接受
4. **判断标准**：2026年底前，至少1篇被正式刊物接受/发表

### 短板5：人际网络激活 🟡

**现状**：中科大校友、浙江籍贯、民建会员——三重身份，三个网络。但都是"被动持有"，不是"主动调用"。没有一份联系人清单，没有定期的联络节奏。

**为什么是短板**：
- 中科大AI校友能告诉她AI芯片公司哪家技术真行——比自己看研报快10倍
- 浙江籍金融同行能引荐科技企业客户——比自己陌拜效率高10倍
- 民建渠道能提交政策建议——但联系方式和稿件还没准备好

**根因**：没有"管理人际网络"的习惯。银行20年靠业务往来自然形成关系，没有主动经营过"校友/同乡/党友"网络。

**改进路径**：
1. **6月**：梳理三个网络各10个关键联系人（姓名+机构+职位+最近一次联系+下次联系计划）
2. **7月**：联系其中5个人（不是求人办事，是"最近在研究XX，想到你在这方面很专业，想请教"）
3. **Q3**：参加1次中科大上海校友会活动、1次民建上海金融委活动
4. **判断标准**：9月底前，三个网络各有3个以上"活跃联系人"（最近3个月内有实质性交流）

### 短板6：健康数据素养 🟡

**现状**：百岁健康计划、6月启动包、健康AI支持方案、6/1基线采集清单都已生成。但16项基线数据还是0。血压计还没买。

**为什么是短板**：
- 健康方案写得再好，血压数据不开始测，等于0
- 49岁是女性围绝经期的关键窗口——现在开始追踪，数据才有意义
- 她的健康风险是真实的（血压偏高、甲状腺结节、BMI偏高）——不是在"预防"，是在"管理已有问题"

**根因**：健康管理被排在6目标中优先级最低。投资/事业有deadline，健康没有紧迫感——直到出事。

**改进路径**：
1. **今天**：京东下单血压计（欧姆龙U10K，¥300）
2. **6月1日**：按基线采集清单完成首次16项数据采集
3. **6月每天**：晨起测血压（10秒语音输入），这是非 negotiable 的
4. **判断标准**：6月底前，血压记录≥20天，拿到第一份月度健康趋势报告

### 短板7：中级经济师备考 🟡

**现状**：7月考试，0小时投入。入库与职称攻坚计划写了，但没有后续执行。

**为什么是短板**：
- 中级经济师是体制内晋升的硬条件——不过就没有下一步
- 7月考试意味着现在只有不到2个月
- 再不过，又是一年

**根因**：备考被淹没在更"有趣"的事情中（写方法论、研究AI产业链、做量化策略，都比背书有意思）。

**改进路径**：
1. **6月**：每天30分钟（早起第一件事，不放在晚上），只看教材+真题
2. **每周日**：做一套真题，Claude帮她分析错题分布
3. **判断标准**：7月考试通过

### 短板8-10：英语/演讲/编程 🟢

这三个是"有了更好，没有也能活"的能力。但因为当前有更紧迫的短板（1-7），建议：

- **英语**：维持阅读能力，不专门投入时间学听说
- **演讲**：第1篇发表文章被接受后，自然会有演讲机会，届时再准备
- **编程**：不需要学。Claude Code+Agent是她不需要编程就能用代码的桥梁

## 五、诚实直面——最大的三个风险

### 风险1："规划快感"替代"执行痛苦"

写了30份方案文档。每写一份都感觉"已经做了"。但昆仑万维仓位没动、血压计没买、经济师没学、文章没投。

**自检问题**：这周是我写了第N份方案，还是做了1件从前没做过的事？

### 风险2：6目标平均用力=都不够

6目标同时推进听起来全面，实际上是"投资只减仓不清仓、事业只规划不投稿、健康只写方案不测血压"。

**自检问题**：如果只能完成3件事，我选哪3件？（答案是：还本资金、测血压、投1篇文章）

### 风险3：身份资源不用=没有

中科大、浙江籍、民建——三个网络都在手里，但联系人清单还没写。

**自检问题**：这周联系了一个校友/同乡/民建同志吗？
