# 幸福人生30天启动计划（2026年6月）

> 配套《幸福人生描述》L0纲领 | 从框架到执行的第一份月计划
>
> 核心原则：**6月只做启动系统的事，不追求结果。系统跑起来，结果自然会来。**

---

## 一、六月总览

```
        第1周(6/1-6/7)        第2周(6/8-6/14)       第3周(6/15-6/21)       第4周(6/22-6/28)
        ┌──────────┐         ┌──────────┐         ┌──────────┐         ┌──────────┐
        │ 基线测量周 │   →    │ 系统建立周 │    →   │ 执行启动周 │    →   │ 复盘校准周 │
        └──────────┘         └──────────┘         └──────────┘         └──────────┘
        测出所有起点             建好每周习惯            真正动起来              检视+调整
```

---

## 二、每周六大目标行动

### 第1周（6/1-6/7）—— 基线测量周

> 核心任务：为每个目标测出明确的起点，没有基线的计划只是猜测。

| 目标 | 本周行动 | 产出 |
|------|---------|------|
| **投资** | 从招商证券APP导出全部持仓到Excel；确认信用贷月供/利率/期限 | 持仓清单Excel + 信用贷准确数据 |
| **事业** | 与人力资源部确认：后备干部库状态、竞聘周期、年龄政策 | 竞聘信息确认记录 |
| **家庭** | 与陈于东散步聊天，了解考研意愿和暑期想法；确认两位母亲健康状况 | 孩子意愿笔记 + 母亲健康初评 |
| **健康** | **基线测量周**：测体重/BMI/体脂率（体脂秤）、血压（血压计）、记录一周睡眠时间 | 6项核心指标全部有数字 |
| **AI** | 百日计划第1周启动——每天30-45分钟，按7天课程表执行（详见AI操作手册） | 第1周学习笔记 |
| **知识库** | 在Obsidian创建8大领域文件夹（INV/TF/BIZ/TECH/FAM/HEALTH/PHIL/LIFE） | 文件夹结构就位 |

### 第2周（6/8-6/14）—— 系统建立周

> 核心任务：建立每周固定动作的节奏。

| 目标 | 本周行动 | 产出 |
|------|---------|------|
| **投资** | 设定每周六上午1小时"投资检视时间"；按操作手册开始清理第1批死仓位（持仓<5000元的品种） | 首批死仓位清理完成 |
| **事业** | 开始读《科技金融》（邮储银行课题组）第1章；写1张阅读卡片 | 首张阅读卡片 |
| **家庭** | 与陈于东一起下载背词APP，选定词库；启动每日30词 | 背词APP就位+首周打卡 |
| **健康** | 开始每周3次运动（快走40min/游泳/力量各一次）；按饮食模板试运行3天 | 运动×3 + 饮食记录×3 |
| **AI** | 百日计划第2周——Python基础（详见AI操作手册） | 第2周学习笔记 |
| **知识库** | 创建首张MOC：`MOC-科技金融.md`；为8套已有框架各写1张索引卡片 | MOC科技金融+8张索引卡 |

### 第3周（6/15-6/21）—— 执行启动周

> 核心任务：各目标的系统真正开始运转。

| 目标 | 本周行动 | 产出 |
|------|---------|------|
| **投资** | 继续清理第2批死仓位；启动昆仑万维减仓计划（第1笔卖出） | 死仓位继续减少 |
| **事业** | 确认中级经济师评审路径和时间节点；继续读《科技金融》第2-3章 | 中级经济师路径明确 + 阅读卡片×2 |
| **家庭** | 与陈于东一起调研3所目标院校，填写院校对比表初稿 | 院校对比表v0.5 |
| **健康** | 运动×3继续；饮食模板全天试运行；记录体重变化 | 运动×3 + 饮食记录 + 体重趋势 |
| **AI** | 百日计划第3周——Claude Code进阶（并行Agent、记忆系统）（详见AI操作手册） | 第3周学习笔记 |
| **知识库** | 写周记（300字）；回顾本周新增内容，更新MOC | 第3周周记 |

### 第4周（6/22-6/28）—— 复盘校准周

> 核心任务：第一个PDCA循环完成，为7月调整方向。

| 目标 | 本周行动 | 产出 |
|------|---------|------|
| **投资** | 月度资产快照：净资产、持仓集中度、月供合计；对比月初计划看偏差 | 6月投资月报 |
| **事业** | 6月阅读回顾；规划7月学习重点 | 6月学习小结 |
| **家庭** | 与陈于东回顾暑假第一个月进展，调整7月计划 | 暑假执行进度检视 |
| **健康** | 对比6/1和6/28的基线数据：体重、BMI、运动次数、饮食依从率 | 6月健康月报 |
| **AI** | 百日计划第4周——基础工具期收尾，产出第1个自动化脚本 | 自动化脚本×1 |
| **知识库** | 6月总结周记（300字）；盘点知识卡片数量；规划7月重点 | 6月知识库月报 + 7月计划 |

## 三、每日节奏（工作日模板）

```
06:30  起床，称体重（记录）
07:00  早餐（参考饮食模板）
08:00  通勤（听音频/播客，可选AI/投资主题）
09:00  工作（核心4小时，处理最重要的任务）
12:00  午餐（控糖控脂原则）
13:00  午休20分钟
13:30  工作（下午处理会议/沟通/行政）
17:30  通勤
18:30  晚餐（家庭时间）
19:30  运动（一三五快走/游泳/力量，二四休息）
20:30  学习/AI/知识库（30-45分钟）
21:30  自由时间
22:30  睡觉
```

## 四、六月成功标准（最低可接受结果）

| 目标 | 6月底必须完成 | 完成就算赢 |
|------|-------------|-----------|
| 投资 | 死仓位清理≥10个；信用贷数据确认；6月投资月报 | 仓位开始变干净 |
| 事业 | 竞聘3问有答案；科技金融阅读≥3章；阅读卡片≥3张 | 竞聘路径清晰 |
| 家庭 | 背词APP打卡≥20天；院校对比表初稿；母亲健康初评 | 暑假考研启动 |
| 健康 | 基线6项有数字；运动≥8次；体重下降0.5kg+ | 运动习惯起步 |
| AI | 百日计划前4周完成；自动化脚本1个；学习笔记4篇 | 百日计划不中断 |
| 知识库 | Vault 8大领域有文件夹；卡片≥12张；MOC≥1个 | Vault初具雏形 |
