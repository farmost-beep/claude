# 社交关系AI管家 · 产品说明书 v2.0

> 聚焦 OPC/创业者，极致体验 | 2026-06-13

---

## 产品定位

**一句话定义**：一个让你在投资人、合伙人、客户面前永远像做过功课的AI管家——而且你什么额外的事都不用做。

**目标用户**：创业者、初创公司创始人、OPC社群成员

**核心承诺**：AI自动记录你的每一次关键互动，在你需要的时候提醒你、帮你拟好消息，推到该去的地方。

---

## 一、创业者三大核心关系链

```
投资人关系                   合伙人关系                   客户关系
─────────────────────  ─────────────────────  ─────────────────────
天使轮跟进 / TS谈判    股权分配 / 分工协作     合同进展 / 项目交付
路演反馈 / DD配合      冲突调解 / 决策同步     复购机会 / 转介绍
```

相比银行客户经理的100-300个标准化客户，创业者只需要管理**30-80个深度关系**——但每个关系都是高度个性化的，每个互动都需要推进实质性进展。

## 二、极致体验三层设计

### L1 极致自动化：你不用做任何事

| 你的行为 | AI自动完成 |
|:--------|:----------|
| 跟投资人聊完微信 | 提取关键反馈 → 存入时间线 → 生成待办 |
| 收到合伙人消息 | 识别新事项 → 更新进度 → 提醒你回复 |
| 客户发了邮件 | 抓取关键信息 → 关联到对应项目 → 建议下一步 |
| 什么也不做 | 扫描所有关系 → 发现需要跟进的 → 生成提醒 |

目标：**用户只管聊天，剩下AI做。**

### L2 极致精准：每条提醒都恰到好处

| 场景 | 提醒内容 | 触发时机 |
|:----|:--------|:--------|
| 投资人跟进 | "张总上周说等TS，今天第5天，建议问问进展" | 约定时间+3天 |
| 合伙人同步 | "上次分工方案还没回复" | 2天未回复 |
| 客户回访 | "项目交付两周了，该问问满意度" | 交付后14天 |
| 引荐致谢 | "李总帮你引荐了投资人" | 会面后24小时内 |
| 关系预警 | "和王总快一个月没联系了" | 超过21天静默 |

### L3 极致顺畅：消息拟得就像你写的

```
你：给张总拟条消息，跟进一下TS进度
AI：张总好，上次聊完TS的事，法务那边进度如何？
    材料已备好，随时配合。

你：太正式了，改成平时聊天的感觉
AI：张总，TS那边咋样了？材料都备好了，随时配合哈。

你：行，发微信。
```

**支持语气调节**：正式/亲切/简洁，可反复调，满意再发。

## 五、交互设计

### 主入口：微信对话（唯一入口）

创业者不需要学任何新工具。在微信里聊天就是操作：

```
→ 记一下：和张总聊了TS，等法务回复
  ✓ 已记录，3天后提醒

→ 张总最近啥情况？
  📋 6/10聊了TS，等法务回复中，建议明天跟进

→ 给张总拟条消息，轻松点
  📝 "张总，TS那边咋样了？有啥需要配合的随时说哈"

→ 发
  ✓ 已发送
```

## 九、竞品对比

| 维度 | 手机通讯录 | Notion/CRM | 本方案 |
|:----|:---------|:----------|:------|
| 记录了什么 | 电话号码 | 手动填表 | **一切互动，AI自动记** |
| 主动性 | 无 | 手动提醒 | **AI主动提醒+拟稿** |
| 使用门槛 | 零门槛 | 需要搭建 | **微信聊天即操作** |
| 对创业者的价值 | 存号码 | 项目管理 | **不用记、不用想、直接发** |

---

> **定位**：不是为了替代CRM，而是让创业者再也不用"记"和"想"——AI替你记住所有关系，在你需要的时候帮你说出来。
