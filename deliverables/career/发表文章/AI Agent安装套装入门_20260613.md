# AI Agent 安装套装入门：从 Claude Code 到微信对话，10分钟打通

> 2026-06-13 | 作者：陈颖芳

---

## 为什么需要这套"安装套装"？

Claude Code 是当前最强的 AI 编程 Agent，但它默认只跑在终端里。装上 **cc-switch**（模型切换器）+ **cc-connect**（微信桥接），就能：

- ✅ 用 **DeepSeek** 等国产模型替代官方 Claude（省钱+国内直连）
- ✅ 在 **微信** 里远程操控 Claude Code
- ✅ 通过 **Web界面** 管理项目

---

## 第一步：安装 Claude Code

```bash
npm install -g @anthropic-ai/claude-code
claude --version  # 验证安装
```

国内用户加镜像：
```bash
npm install -g @anthropic-ai/claude-code --registry=https://registry.npmmirror.com
```

> 💡 如果遇到登录验证弹窗，在用户目录创建 `.claude.json`：
> `{"hasCompletedOnboarding": true}`

---

## 第二步：安装 cc-switch（模型切换器）

cc-switch 是一个桌面应用，一键切换 Claude Code 背后的模型供应商。

```bash
# macOS（推荐）
brew tap farion1231/ccswitch && brew install --cask cc-switch

# Windows/Linux → 从 GitHub Releases 下载
```

**在 cc-switch 中添加 DeepSeek：**

打开 cc-switch → Add Provider → 选择 DeepSeek 预设 → 填入：

```
Base URL: https://api.deepseek.com/anthropic
API Key:  你的 DeepSeek API Key
主模型:   deepseek-v4-pro[1M]
快速模型: deepseek-v4-flash
接口类型: Anthropic 兼容格式
```

> ⚠️ DeepSeek 需先充值（platform.deepseek.com，支持微信/支付宝），否则报 402。

---

## 第三步：安装 cc-connect（微信桥接）

cc-connect 是核心桥梁，把 Claude Code 接到微信、飞书、钉钉等平台。

```bash
npm install -g cc-connect@beta
cc-connect --version  # v1.3.3+
```

**配置微信连接：**

编辑 `~/.cc-connect/config.toml`：

```toml
[[projects]]
  name = "my-project"

  [projects.agent]
    type = "claudecode"
    [projects.agent.options]
      work_dir = "/你的工作目录"

  [[projects.platforms]]
    type = "weixin"
    [projects.platforms.options]
      token = "你的iLink Bot Token"
      base_url = "https://ilinkai.weixin.qq.com"
```

> 微信 Token 获取方式：用微信扫码绑定 iLink Bot，或直接用 wechat-claude-code 的 `npm run setup` 生成。

**启动：**

```bash
cc-connect                # 启动主服务
cc-connect web            # 启动 Web 管理界面 → http://localhost:9820
```

---

## 第四步：配置更多模型供应商（可选）

cc-connect 的 `[[providers]]` 配置支持同时接入多个模型。以下是我实际在用的配置：

| 供应商 | 模型 | 用途 |
|--------|------|------|
| **DeepSeek V4 Flash** | `deepseek-v4-flash` | 日常快速任务 |
| **DeepSeek V4 Pro** | `deepseek-v4-pro[1M]` | 深度推理任务 |
| **OpenRouter** | `claude-sonnet-4.6` | 官方 Claude 通道 |
| **小米 Mimo** | `mimo-v2-pro` | 备用 |
| **云知声** | `u2` | 备用 |

配置格式：
```toml
[[providers]]
  name = "deepseek-v4-pro"
  api_key = "sk-xxx"
  base_url = "https://api.deepseek.com/anthropic"
  model = "deepseek-v4-pro[1M]"
  agent_types = ["claudecode"]
```

## 最终效果

全部装好后，你的微信就是 Claude Code 的远程终端：

```
微信发消息 → cc-connect → Claude Code → 执行结果 → 微信回复

Web管理界面 → http://localhost:9820
```

支持命令：
- `/mode yolo` — 全自动执行模式
- `/new` — 新建会话
- `/list` — 查看会话列表
- 直接发文字/图片 → AI 实时处理
