## 一、创业者社交关系的独特场景

### 三大核心关系链

```
投资人关系         创始人/合伙关系        客户/合作伙伴
┌──────────┐     ┌──────────┐     ┌──────────┐
│ 天使轮跟进 │     │ 股权分配   │     │ 合同进展   │
│ TS条款谈判 │     │ 分工协作   │     │ 项目交付   │
│ 路演反馈   │     │ 冲突调解   │     │ 复购机会   │
│ DD配合    │     │ 决策同步   │     │ 转介绍     │
└──────────┘     └──────────┘     └──────────┘
```

### 创业者 vs 银行客户经理的核心差异

| 维度 | 银行客户经理 | OPC/创业者 |
|:----|:-----------|:----------|
| 关系数量 | 100-300个客户 | 30-80个关键关系 |
| 关系深度 | 偏标准化维护 | 高度个性化 |
| 关注点 | 不遗忘、不漏跟 | 每次联系都有价值、推进实质性进展 |
| 痛点 | 记不住上次说了什么 | 不知道上次聊的事推进到哪了 |
| 沟通工具 | 微信为主 | 微信+邮件+飞书+线下 |
| 技术接受度 | 低（需极简） | 高（可接受CLI） |
| 愿意付费 | 个人掏钱犹豫 | **公司报销，决策快** |

## 二、极致体验定义

### 核心价值主张

> **"让你在投资人、合伙人、客户面前，永远像做过功课。"**

### 三个"极致"体验

#### 1. 极致自动化：不需要手动记录

每次关键互动后，自动提取：
- 投资人说的关键反馈 → 存入时间线
- 约定的事项 → 自动生成待办
- 下次沟通最佳时机 → AI建议

**用户什么都不用做，只管聊天。**

#### 2. 极致精准：每次提醒都恰到好处

| 场景 | 提醒内容 | 时机 |
|:----|:--------|:----|
| 投资人跟进 | "张总上周说等TS，今天第5天，建议今天问问进展" | 约定时间+3天 |
| 合伙人同步 | "你们周三讨论的分工方案，他还没回复" | 2天未回复 |
| 客户回访 | "王总项目交付两周了，该问问满意度" | 交付后14天 |
| 引荐致谢 | "李总帮你引荐了投资人，该发个消息感谢" | 会面后24h内 |

#### 3. 极致顺畅：消息拟得"就是你写的"

```
你：给张总拟条消息，跟进一下TS进度

AI：张总好，上次聊完TS的事，不知道法务那边进度如何？
    我们这边的材料已经准备好了，随时可以配合。
    如果有任何问题随时沟通。

你：太正式了，改成我们平时聊天的感觉

AI：张总，TS那边进展咋样了？我们材料都准备好了，
    有啥需要配合的随时说哈。

你：好，就这个，发微信。
```

## 三、功能设计（创业者版）

### MVP（1周可出）

| 功能 | 说明 | 体验标准 |
|:----|:-----|:--------|
| 关键关系录入 | 投资人/合伙人/客户/导师 四类标签 | 30秒录入一个 |
| 互动快速记录 | 语音输入/文字/转发消息自动解析 | 10秒记一条 |
| AI待办提取 | 从记录中自动提炼待办事项 | 准度>80% |
| 消息拟稿 | 基于上下文生成，支持语气调节 | 修改不超过2次 |
| 微信推送 | 提醒+消息发送到微信 | 即时送达 |

### V1.0（+1周）

| 功能 | 说明 |
|:----|:-----|
| 关系日历 | 投资人联系频率/合伙人默契度/客户活跃度 |
| 融资跟进看板 | 当前在谈的TS/条款/进度一目了然 |
| 团队协作同步 | 合伙人间共享关键客户进度 |
| Email接入 | 自动抓取邮件往来 |

### V2.0（+2周）

| 功能 | 说明 |
|:----|:-----|
| 微信消息自动抓取 | 已发/已收消息自动进入时间线 |
| 关键时刻提醒 | 生日/节日/里程碑（融资成功/产品上线） |
| 关系健康度评分 | 每个关系打分，预警"快凉了"的关系 |

```

→ 张总最近啥情况？

→ 给张总拟条消息，轻松点
```

## 五、商业验证

### 为什么创业者更容易付费？

| 因素 | 银行客户经理 | OPC/创业者 |
|:----|:-----------|:----------|
| 决策链 | 自己决定 | 自己决定 |
| 付费来源 | 个人收入 | **公司运营费用** |
| 价格敏感度 | 高 | 低（每月几百块不算事） |
| 试用意愿 | 低（怕麻烦） | **高（愿意试新工具）** |
| 流失成本 | 低 | **高（关系数据价值大）** |
| 推荐意愿 | 低 | **高（创业者圈子互通）** |
