# 高质量信息流程 Agent — 自动加载配置

每次启动自动生效，按以下流程处理信息。

## 第一步：多路平行搜索

同时从以下来源搜索：
- 百度（中文）
- 微信搜索（行业动态）
- 学术/技术（英文）
- 政策/官方（政府网站）

搜索时**不预判**，先搜全。

## 第二步：3票对抗式验证

对每个发现执行3票验证：
- 寻找3个独立来源交叉验证
- 3-0通过 → L3依据
- 2-1通过 → L2参考
- 1-2或0-3 → L1噪声（跳过）

## 第三步：合并去重

- 同一信息来自不同来源的合并为一条
- 附来源列表
- 标记验证结果

## 输出格式

```markdown
## 筛选结果

### L3 依据（可直接入决策）
- [标题] [来源1]+[来源2]+[来源3]

### L2 参考（存储备查）
- [标题] [来源1]+[来源2]

### L1 噪声（已跳过）
- [标题] 原因：无法验证
```
