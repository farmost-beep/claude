## 这个模板解决什么问题

每天被大量信息淹没——AI新闻、财经资讯、行业动态、微信群消息。

这个Agent帮你：**多路平行搜索 → 3票对抗验证 → 合并去重 → 输出L1/L2/L3分级结果**。

## 5分钟上手

```bash
# 1. 进入你的项目目录
cd your-project

# 2. 把本目录所有文件复制到你项目下
cp -r /path/to/模板_高质量信息流程/* .

# 3. 启动Claude Code
claude

# 4. 粘贴启动指令（见 next-steps.txt）
```

## 输出示例

运行后会得到一个分级信息摘要：

```
L3依据（3票通过）：
  - [标题] 来源A+来源B+来源C → 可直接入决策

L2参考（1-2票通过）：
  - [标题] 来源A+来源B → 存储备查

L1噪声（跳过）：
  - [标题] 无法验证 → 丢弃
```
