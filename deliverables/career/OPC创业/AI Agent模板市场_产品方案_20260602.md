## 三、每个模板的交付物

以Agent A-E投资分析管道为例：

```
Agent A-E投资分析管道/
├── README.md                           ← 使用说明+5分钟上手
├── 投资分析Agent组配置.md              ← 5个Agent的完整配置
├── 启动指令.txt                        ← 一句话启动命令
├── 输出示例.md                         ← 跑完长什么样
└── Agent A/B/C/D/E各自模板.md          ← 独立可复用
```

买家只需要：
1. 复制到自己的项目
2. 修改股票代码
3. 运行 `claude`
4. 粘贴启动指令
