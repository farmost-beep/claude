## 一、问题背景

2025年是中国并购贷款制度的"大修之年"。国家金融监督管理总局于2025年3月推出科技企业并购贷款试点政策（18城市，贷款比例80%、期限10年），并于2025年12月31日发布《商业银行并购贷款管理办法》（金规〔2025〕27号），实现三项关键突破：首次允许参股型并购贷款（单次持股≥20%即可）、贷款比例上限提至70%（科技试点80%）、期限延至10年。

然而制度框架搭建不等于市场活力自动释放。笔者在邮储银行上海分行科技金融事业部的实践中发现，《办法》解决了"能不能做"的顶层设计，但"敢不敢做"和"怎么做好"的配套机制仍然缺失。具体存在四个短板：

1. **技术价值评估标准空白**。科创并购标的价值很大程度在"技术值多少钱"，但目前全行业没有统一的、受银行风控认可的技术价值评估准则。银行审批人员看不懂技术，只能回到看现金流的旧路。
2. **担保机制缺位**。科创企业核心资产（知识产权、研发团队）在银行传统风控中难以作为合格担保品，缺乏"技术到信用"的转换机制。
3. **缺乏风险分担与流动性工具**。既无针对科创并购贷款的政府增信机制，也无将并购贷款资产证券化（ABS）以回收流动性的成熟通道。
4. **集中度约束下的协同困境**。《办法》将集中度收紧至2.5%，单一银行对大型科创并购（百亿级）参与空间被压缩，而科创并购银团因技术研判能力不对等难以组建。

四个短板指向同一诊断：**有了并购贷款的"制度文本"，但还没有科创并购融资的"生态系统"。**

## 二、分析框架：五层生态位

并购贷款不只是银行的一款信贷产品——它是科创金融生态的"枢纽型基础设施"，一端连接风险投资退出通道，一端连接科技企业规模化跃升。以下五条建议，每条对应一个生态位：

| 生态位 | 功能 | 解决工具 | 状态 |
|--------|------|---------|------|
| 制度框架（骨骼） | 能不能做？ | 《并购贷款管理办法》+科技企业试点 | 已解决（2025） |
| 通用语言（神经系统） | 各方能否就"技术值多少钱"达成共识？ | 技术价值评估国家标准 | 建议一 |
| 风险消化（免疫系统） | 银行敢不敢承担技术风险？ | 增信基金+科技保险双层分担 | 建议二 |
| 协同网络（循环系统） | 多家银行能否协作做大项目？ | 银团协调机制+二级转让 | 建议三 |
| 代谢循环（代谢系统） | 资金能否流转持续放贷？ | 科创并购贷款ABS | 建议四 |
| 进化机制（适应系统） | 制度能否随市场演进而更新？ | 试点→评估→立法+认定透明化 | 建议五 |

## 三、政策建议

### 建议一：建立科创企业技术价值评估国家标准

这是科创金融生态的"通用语言"。VC、银行、监管、评估机构需要同一套话语体系讨论"技术值多少钱"。建议由国家知识产权局牵头，联合中国资产评估协会、科技部，制定《科创企业技术价值评估准则》：

1. **分层评估框架**：按技术成熟度（TRL 1-9级）分层设定评估方法——实验室阶段侧重专家评议，小批量产阶段侧重市场可比交易法，规模商业化阶段侧重收益法。
2. **双签制度**：科创并购贷款的技术评估报告须由"技术专家+注册资产评估师"双签，分别对技术可行性和价值合理性负责。
3. **评估机构白名单**：由国家知识产权局认可的技术价值评估机构名录，银行审批科创并购贷款时优先采信白名单机构的评估结论。
4. **与银行风控对接**：明确技术价值评估结论在银行内部评级、风险加权资产计量中的采信路径。

### 建议二：设立"科创并购融资增信基金"

这是科创金融生态的"风险消化器"。银行不敢给技术资产放贷，不是因为不看好，而是因为缺乏风险分担——增信基金把"无限恐惧"变成可计算的有限风险。建议由财政部、科技部联合出资，在上海、北京、深圳率先试点设立科创并购融资增信基金：

1. **聚焦无形资产担保缺口**：专项为以知识产权、技术收益权为主要担保方式的科创并购贷款提供增信，分担比例建议为贷款金额的40%-60%。
2. **风险定价挂钩技术评级**：增信费率与并购标的技术评级挂钩——评级越高、费率越低，形成正向激励。
3. **多元资金来源**：中央财政20%、地方财政30%、商业银行（按受益比例认购）20%、保险资金30%。
4. **双层风险分担**：探索"担保基金+科技保险"双层机制——担保基金覆盖第一损失层（0-20%），科技保险覆盖第二损失层（20-50%），银行自留剩余风险。
5. **退出与造血机制**：被并购企业若成功上市或被战略收购，增信基金按约定比例分享超额收益，形成基金自我造血能力。

### 建议三：建立科创并购贷款银团协调机制

这是科创金融生态的"协同网络基础设施"。不同银行的技术研判能力不对等，需要标准化合约+信息共享+牵头行激励，才能让多家银行在信息不对称下愿意共同参与。针对集中度2.5%约束下大型科创并购的融资缺口，建议由中国人民银行上海总部、金融监管总局上海监管局联合推动：

1. **牵头行激励**：对牵头组建科创并购银团的银行，给予差异化MPA考核加分或存款准备金率优惠。
2. **标准化合约**：由银行业协会制定科创并购银团贷款标准化合同模板，降低谈判成本。
3. **信息共享平台**：在上海票据交易所或全国银行间同业拆借中心设立科创并购融资信息专栏，降低信息搜寻和撮合成本。
4. **二级市场转让**：允许银团参与行将所持份额在银行间市场转让，提升退出灵活性。

### 建议四：推动科创并购贷款资产证券化（ABS）试点

这是科创金融生态的"代谢循环系统"。10年期贷款若趴在资产负债表上不动就是"血栓"，通过ABS形成"发放→证券化→回收→再发放"的循环，生态才有持续的血液流动。科创并购贷款期限长达10年，银行资产负债期限错配压力显著。建议：

1. **专项额度**：由央行给予科创并购贷款ABS试点额度，不计入银行间市场信用债存量管控。
2. **优先级结构化设计**：优先级（AAA，银行间市场投资者）、中间级（AA/AA-，保险资金/券商资管）、劣后级（发起银行自留5%+增信基金认购5-10%）。
3. **底层资产合格标准**：进入ABS池的贷款须满足：标的属战略性新兴产业、存续超2年且本息正常、并购整合效果经第三方评估"良好"以上。
4. **循环购买结构**：采用循环购买型ABS，银行可不断注入新发放的合格贷款，形成"发放-ABS-回收资金-再发放"的持续循环。

### 建议五：明确并扩大科技企业并购贷款试点政策的适用范围

这是科创金融生态的"进化机制"。生态不是设计出来的，是演化出来的——先在"生态缸"里跑通，再把成熟经验固化进制度。建议：

1. **认定标准透明化**：建议以"高新技术企业"资质+研发投入强度（研发费用占营收≥5%）+持有有效发明专利≥3项，三要素联合作为试点科技企业认定标准。
2. **试点城市适当扩围**：在总结18个城市经验基础上，将苏州工业园区、武汉东湖高新区等"科创走廊"核心节点以"区内适用"方式纳入试点。
3. **明确存续期与立法路径**：明确试点有效期5年，配合"试点（2025-2027）→评估（2027-2028）→纳入《办法》正式修订（2028）"的路径，消除政策不确定性。

## 四、预期成效

五条建议构成有机整体：技术评估国标建立生态"通用语言"，增信基金构建"风险消化"机制，银团协调打通横向协作，ABS实现资金纵向循环，试点到立法提供制度进化通道。预期效果：

1. 推动科创并购融资从"政策推动"转向"市场驱动"，银行从资产持有者转型为资产组织者。
2. 打通大型科创并购境内融资通道，百亿级项目从"需要特批"变为"标准化操作"。
3. 形成"上海先行先试→长三角推广→全国复制"的可复制孵化路径。
4. 完善并购融资生态直接服务于"十五五"科技自立自强——活跃的并购市场让VC更敢投早期，让大企业更敢收购技术公司，让小企业更专注研发。
