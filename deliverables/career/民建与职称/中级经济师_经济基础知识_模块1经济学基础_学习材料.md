## 第1章 社会主义基本经济制度

### 1.1 内涵

| 层面 | 内容 |
|:----|:-----|
| **所有制结构** | 公有制为主体、多种所有制经济共同发展 |
| **分配制度** | 按劳分配为主体、多种分配方式并存 |
| **资源配置方式** | **社会主义市场经济体制**（市场在资源配置中起**决定性**作用，更好发挥政府作用） |

### 1.2 所有制结构

**公有制经济**：
- **国有经济**：控制国民经济命脉，起主导作用
- **集体经济**：部分劳动者共同占有生产资料
- **混合所有制经济中的国有和集体成分**

**非公有制经济**：
- 个体经济、私营经济、外资经济等
- 是社会主义市场经济的重要组成部分

**混合所有制经济**：不同所有制资本交叉持股、相互融合（如国企混改）

### 1.3 收入分配制度

**按劳分配**（主体地位）：
- 公有制经济中的主要分配方式
- "多劳多得，少劳少得"

**按要素分配**：
- 劳动、资本、土地、技术、管理、数据等生产要素按贡献参与分配

**初次分配**：在创造国民收入的物质生产领域内进行的分配
**再分配**：政府通过税收、社会保障、转移支付等手段进行的调节
**第三次分配**：慈善、公益等自愿性分配

### 1.4 社会主义市场经济体制

**市场在资源配置中起决定性作用**：
- 市场决定价格
- 企业自主经营、自负盈亏
- 公平竞争

**更好发挥政府作用**：
- 宏观调控（经济手段+法律手段+行政手段）
- 市场监管
- 公共服务
- 环境保护

## 第2章 市场需求、供给与均衡价格

### 2.1 需求函数与需求曲线

**需求定律**：价格上升→需求量下降（替代效应+收入效应）

| 概念 | 要点 |
|:----|:-----|
| 需求函数 | Qd = α - βP（线性），α为自发需求，β为价格敏感系数 |
| 需求曲线 | 右下方倾斜（价格↑→需求量↓） |
| 需求量的变动 | **价格变化**引起，沿同一条曲线移动 |
| 需求的变动 | **非价格因素**（收入/偏好/相关品价格/预期）引起，曲线整体位移 |

**影响需求变动的非价格因素**：
1. 消费者收入（正常品↑→需求↑；低档品↑→需求↓）
2. 消费者偏好（偏好↑→需求↑）
3. 相关品价格（替代品↑→需求↑；互补品↑→需求↓）
4. 预期（预期涨价→现期需求↑）

### 2.2 供给函数与供给曲线

**供给定律**：价格上升→供给量上升

| 概念 | 要点 |
|:----|:-----|
| 供给函数 | Qs = -δ + γP |
| 供给曲线 | 右上方倾斜（价格↑→供给量↑） |
| 供给量的变动 | 价格变化引起，沿曲线移动 |
| 供给的变动 | 非价格因素（成本/技术/预期/相关品价格/政府政策）引起，曲线位移 |

**影响供给的非价格因素**：
1. 生产成本（成本↑→供给↓）
2. 技术水平（技术↑→供给↑）
3. 预期（预期涨价→现期供给可能↓）
4. 相关品价格
5. 政府政策（税收/补贴）

### 2.3 均衡价格

**均衡条件**：Qd = Qs

**最高限价**（政府规定低于均衡价）：
- 目的：保护消费者
- 后果：**供给短缺** → 排队/黑市/配给
- ⚠ 必须低于均衡价格才有效

**保护价格/最低限价**（政府规定高于均衡价）：
- 目的：保护生产者（农业）
- 后果：**供给过剩** → 政府收购/库存积压
- ⚠ 必须高于均衡价格才有效

### 2.4 弹性理论 ⭐每年必考

#### 需求价格弹性

| 弹性值 | 类型 | 含义 | 例子 |
|:-----:|:----|:----|:----|
| Ed = 0 | 完全无弹性 | 价格变→需求量不变 | 胰岛素 |
| 0 < Ed < 1 | 缺乏弹性 | 需求量变动<价格变动 | 粮食、盐 |
| Ed = 1 | 单位弹性 | 需求量变动=价格变动 | — |
| 1 < Ed < ∞ | 富有弹性 | 需求量变动>价格变动 | 奢侈品 |
| Ed → ∞ | 完全弹性 | 价格微变→需求剧变 | 完全竞争市场 |

**弧弹性公式**（考试用这个）：
```
Ed = (ΔQ/Q̄) / (ΔP/P̄)
```
其中 Q̄ = (Q₁+Q₂)/2, P̄ = (P₁+P₂)/2

**影响弹性的因素**：
- 替代品越多→弹性越大
- 用途越广泛→弹性越大
- 时间越长→弹性越大
- 必需品→弹性小

**总收入与弹性的关系**（高频考点）：
- 缺乏弹性(Ed<1)：价格↑→总收入↑
- 富有弹性(Ed>1)：价格↑→总收入↓
- 单位弹性(Ed=1)：价格变→总收入不变

#### 需求交叉弹性 ⭐

```
Exy = ΔQx/Qx ÷ ΔPy/Py
```

| Exy | 关系类型 | 例子 |
|:--:|:--------|:----|
| Exy > 0 | **替代品** | 茶和咖啡 |
| Exy < 0 | **互补品** | 汽车和汽油 |
| Exy = 0 | 无关系 | — |

#### 需求收入弹性

```
Ey = ΔQ/Q ÷ ΔY/Y
```

- Ey > 1：高档品（收入↑→需求更快↑）
- 0 < Ey < 1：必需品
- Ey < 0：低档品（收入↑→需求↓）

#### 供给价格弹性

```
Es = ΔQs/Qs ÷ ΔP/P
```

- 时间越长→供给弹性越大
- 生产周期越短→供给弹性越大
- 调整产量越容易→供给弹性越大

## 第3章 生产和成本理论

### 3.1 生产函数

**短期生产函数**（至少一种要素固定）：
```
Q = f(L, K̄)  — 仅劳动可变
```

**总产量(TP)、平均产量(AP)、边际产量(MP)**：
```
AP = TP/L
MP = ΔTP/ΔL
```

**边际报酬递减**（短期生产基本规律）：
- 其他要素不变→连续增加某要素→达到某点后，MP递减

**三个生产阶段**：
- 阶段Ⅰ（0→AP最高点）：MP > 0且AP↑，固定要素利用不足
- 阶段Ⅱ（AP最高点→MP=0）：**合理生产区域**
- 阶段Ⅲ（MP<0）：不经济

### 3.2 成本函数

| 成本类型 | 公式 | 说明 |
|:--------|:----|:----|
| 总成本(TC) | TC = TFC + TVC | 固定+可变 |
| 平均总成本(ATC) | ATC = TC/Q | — |
| 平均固定成本(AFC) | AFC = TFC/Q | Q↑→AFC↓ |
| 平均可变成本(AVC) | AVC = TVC/Q | — |
| 边际成本(MC) | MC = ΔTC/ΔQ | **考试核心** |

**U型成本曲线**（必考）：
- ATC、AVC、MC 三条曲线都是**先降后升**（U形）
- MC 先于 AVC 到达最低点
- MC = AVC 时 → AVC最低
- MC = ATC 时 → ATC最低

**短期成本函数**：至少一种要素固定
**长期成本函数**：所有要素可变
- 规模经济（长期平均成本↓）
- 规模不经济（长期平均成本↑）

### 3.3 利润最大化

```
利润 = 总收入(TR) - 总成本(TC)
利润最大化条件：MR = MC
```
完全竞争市场中：P = MR = MC

## 第4章 市场结构理论

### 4.1 四种市场结构对比 ⭐

| 特征 | 完全竞争 | 垄断竞争 | 寡头垄断 | 完全垄断 |
|:----|:--------|:--------|:--------|:--------|
| 企业数量 | **很多** | 较多 | 少数 | **一家** |
| 产品 | **同质** | **有差别** | 同质或有差别 | 无替代品 |
| 价格控制 | **无** | 有一些 | 有（相互依赖） | **很大** |
| 进入障碍 | **无** | 低 | 高 | **极高** |
| 长期利润 | **零** | 零 | 正 | 正 |
| 例子 | 农产品 | 餐饮/服装 | 电信/石油 | 自来水/电网 |

### 4.2 完全竞争市场

**短期决策**：
- P > ATC → 盈利
- P = ATC → 盈亏平衡
- P < AVC → **停产**（停业点）
- **短期供给曲线**：MC曲线在AVC以上的部分

**长期**：经济利润=0（P=ATC最低点）
- 超额利润吸引新企业进入 → 供给↑ → 价格↓ → 利润消失

### 4.3 完全垄断市场

**需求曲线**：整个市场的需求曲线，右下方倾斜

**价格歧视**：
| 级别 | 定义 | 例子 |
|:---:|:----|:----|
| 一级 | 向每个消费者收不同的价 | 拍卖 |
| 二级 | 按购买量定价 | 批发价vs零售价 |
| 三级 | 按群体定价 | 学生票/老人票 |

**自然垄断**：单个企业生产比多个企业更便宜（规模经济）

**政府对垄断的干预**：
1. 法律手段（反垄断法）
2. 价格管制 → 平均成本定价(P=ATC) 或 边际成本定价(P=MC)

## 第5章 生产要素市场理论

### 5.1 生产要素需求

**引致需求**：对生产要素的需求**派生自**对产品的需求

**联合需求**：生产要素必须**组合使用**

### 5.2 劳动供给曲线 ⭐

**向后弯曲**（特征）：
- 工资低时：工资↑→劳动供给↑（替代效应>收入效应）
- 工资高时：工资↑→劳动供给↓（收入效应>替代效应 → 选择更多闲暇）

### 5.3 工资、地租、利息

**工资**：劳动的价格。工会通过减少供给或增加需求来提高工资

**地租**：土地的价格。土地供给**固定**（垂直供给曲线），地租由需求决定

**利息**：资本的价格。由资本的供给（储蓄）和需求（投资）决定

**正常利润**：企业生存必须的最低利润（本质是企业家才能的报酬）
**超额利润**：超过正常利润的部分（来自垄断/创新/风险）

## 第6章 市场失灵和政府的干预

### 6.1 市场失灵的四类原因 ⭐

| 原因 | 定义 | 例子 | 干预措施 |
|:----|:----|:----|:--------|
| **外部性** | 私人成本≠社会成本 | 污染(负)/教育(正) | 税收/补贴/许可证 |
| **公共物品** | 非竞争性+非排他性 | 国防/灯塔 | 政府提供 |
| **信息不对称** | 一方比另一方知道得多 | 二手车市场 | 信息披露/监管 |
| **垄断** | 市场势力导致低效 | 自然垄断 | 反垄断法/价格管制 |

### 6.2 外部性与科斯定理

**负外部性**→ 过度生产（私人成本<社会成本→如污染）
**正外部性**→ 生产不足（私人收益<社会收益→如教育）

**科斯定理**：只要**产权明确**且**交易成本为零**，市场交易可以解决外部性问题。

### 6.3 公共物品

| 特征 | 非竞争性 | 非排他性 |
|:----|:--------|:--------|
| **纯公共物品** | ✅ | ✅ | 国防、灯塔 |
| **准公共物品** | ❌或部分 | ❌或部分 | 教育、医疗 |

**搭便车问题**：公共物品的非排他性导致私人部门不愿意提供。

### 6.4 信息不对称

**逆向选择**：交易前隐藏信息（二手车劣车驱逐好车）
**道德风险**：交易后改变行为（买了保险后更冒险）

## 第7章 国民收入核算和简单的宏观经济模型

### 7.1 国内生产总值GDP ⭐每年必考

**GDP定义**：一国内一定时期内生产的**最终产品和服务的市场价值**

**GDP vs GNP**：
- GDP：**地域**概念（在中国境内）
- GNP：**国民**概念（中国公民）

**三种核算方法**：
```
1. 生产法 = 各行业总产出 - 中间投入
2. 收入法 GDP = 劳动者报酬 + 生产税净额 + 固定资产折旧 + 营业盈余
3. 支出法 GDP = 消费(C) + 投资(I) + 政府购买(G) + 净出口(X-M)
```

**支出法 GDP**（最常用）：
| 部分 | 构成 |
|:----|:----|
| 消费(C) | 居民购买消费品和服务 |
| 投资(I) | 固定资产投资+存货投资（⚠不含股票债券） |
| 政府购买(G) | **政府消费性支出**（⚠不包括转移支付） |
| 净出口(X-M) | 出口-进口 |

**GDP平减指数**：
```
GDP平减指数 = 名义GDP / 实际GDP × 100
```

### 7.2 宏观经济模型

**两部门经济**：Y = C + I；均衡条件：I = S
**三部门经济**：Y = C + I + G；均衡条件：I + G = S + T
**四部门经济**：Y = C + I + G + (X-M)；均衡条件：I + G + (X-M) = S + T

### 7.3 消费函数与储蓄函数 ⭐

```
消费函数：C = α + βY
  其中：α = 自发消费（收入为0时的最低消费）
        β = 边际消费倾向 MPC
        βY = 引致消费

储蓄函数：S = Y - C = -α + (1-β)Y

核心关系：APC + APS = 1 且 MPC + MPS = 1
```

### 7.4 乘数效应 ⭐

```
投资乘数 k = 1/(1-MPC) = 1/MPS
```

- MPC越大→乘数越大→投资拉动效应越强
- 乘数>1：投资通过消费链产生多轮拉动

### 7.5 总需求与总供给

**总需求曲线**：右下方倾斜（价格↑→总需求↓）
**总供给曲线**：
- 短期：右上方倾斜
- 长期：**垂直**（AS-AD模型）

## 第8章 经济增长和经济发展理论

### 8.1 经济增长因素分解

**经济增长 = 劳动增长 + 资本增长 + 技术进步**

**索洛余值**（全要素生产率）：
```
经济增长率 = 技术进步率 + α×劳动增长率 + β×资本增长率
索洛余值 = 经济增长率 - α×劳动增长率 - β×资本增长率
```

### 8.2 经济周期 ⭐

**四阶段**：**复苏 → 繁荣 → 衰退 → 萧条**

**按长度分类**：
| 周期 | 长度 | 别称 |
|:---:|:----|:----|
| 短周期 | ~3-5年 | 基钦周期（库存周期） |
| 中周期 | ~8-10年 | **朱格拉周期**（设备投资周期） |
| 长周期 | ~50-60年 | **康德拉季耶夫周期**（技术革命周期） |

**先行指标**（预测）：股票指数、制造业订单、货币供给量
**一致指标**（同步）：GDP、工业增加值
**滞后指标**（确认）：CPI、失业率

### 8.3 经济发展

**经济发展 ≠ 经济增长**：
- 经济增长：GDP增长（**数量**）
- 经济发展：经济+生活质量+技术+结构（**质量+数量**）

**新发展理念**：创新、协调、绿色、开放、共享

### 8.4 ⭐2026新增：新质生产力

**定义**：创新起主导作用，摆脱传统经济增长方式，具有**高科技、高效能、高质量**特征的先进生产力

**核心要素**：
| 要素 | 内容 |
|:----|:-----|
| **技术革命性突破** | 人工智能、量子信息、生物技术、新能源等前沿技术 |
| **生产要素创新性配置** | 数据作为新生产要素，优化劳动力/资本/土地配置 |
| **产业深度转型升级** | 战略性新兴产业+未来产业 |

**与新质生产力相关的产业方向**：
- 数字经济（AI、大数据、云计算）
- 绿色经济（新能源、碳达峰碳中和）
- 生物经济（基因技术、生物制造）
- 高端装备（航空航天、机器人）

**对新质生产力的政策支持**：
- 科技自立自强（解决"卡脖子"问题）
- 完善科技创新体制
- 推动传统产业智能化、绿色化转型

## 第9章 价格总水平和就业、失业

### 9.1 价格总水平

**衡量指标**：CPI（居民消费价格指数）

**通货膨胀类型**：
| 类型 | 年通胀率 | 影响 |
|:----|:--------|:----|
| 爬行/温和 | <10% | 正常 |
| 奔腾/加速 | 10%-100% | 经济不稳定 |
| 恶性 | >100% | 货币崩溃 |

### 9.2 失业

| 类型 | 原因 | 例子 |
|:----|:----|:----|
| 摩擦性失业 | 信息不完美 | 换工作期间 |
| 结构性失业 | 技能不匹配 | 技术替代人工 |
| 周期性失业 | **总需求不足** | 经济衰退 |
| 自愿失业 | 不愿接受当前工资 | — |

**充分就业≠零失业**：仍有摩擦性+结构性失业（自然失业）

### 9.3 ⭐2026新增：就业优先政策

**就业优先政策的核心**：
- 将促进就业放在经济社会发展**优先位置**
- 强化财政、货币、投资等政策**支持就业**
- 健全**就业公共服务体系**
- 完善**重点群体就业支持体系**（高校毕业生、农民工、退役军人）

**数字经济与灵活就业**：
- 平台经济（外卖/网约车/直播）创造大量灵活就业岗位
- 灵活就业者的**社会保障覆盖**成为政策重点

### 9.4 奥肯定律 ⭐

> GDP每下降**高于潜在水平**的部分→失业率上升
> 简化：GDP下降约**2%**，失业率上升约**1%**

### 9.5 菲利普斯曲线 ⭐

**短期菲利普斯曲线**：失业率与通胀率**反向相关**（通胀↑→失业↓）
**长期菲利普斯曲线**：**垂直**（失业率回归自然失业率）
- 长期货币**中性**：货币供给只影响价格，不影响产出和就业

## 第10章 国际贸易理论和政策

### 10.1 国际贸易理论

| 理论 | 提出者 | 核心 |
|:----|:------|:----|
| **绝对优势理论** | 亚当·斯密 | 各国生产各自成本最低的产品，然后交换 |
| **比较优势理论** | 大卫·李嘉图 ⭐ | "两优取重，两劣取轻"——只要相对成本不同，贸易就有利 |
| **要素禀赋理论** | 赫克歇尔-俄林 | 各国出口密集使用其丰富要素的产品 |
| **规模经济贸易理论** | 克鲁格曼 | 规模经济也是贸易的原因 |

### 10.2 国际贸易政策

**保护主义**：关税、非关税壁垒（配额/许可证/技术标准）
**自由贸易**：WTO、RCEP、CPTPP

**倾销**：低于国内价格或成本向进口国销售

### 10.3 汇率与国际贸易

**本币贬值**→出口↑进口↓→改善贸易收支
**马歇尔-勒纳条件**：进出口需求弹性之和>1，贬值才有效
