## 第1章 市场需求、供给与均衡价格

### 1.1 需求函数与需求曲线

**需求定律**：价格上升→需求量下降（替代效应+收入效应）

| 概念 | 要点 |
|:----|:-----|
| 需求函数 | Qd = α - βP（线性），α为自发需求，β为价格敏感系数 |
| 需求曲线 | 右下方倾斜（价格↑→需求量↓） |
| 需求量的变动 | **价格变化**引起，沿同一条曲线移动 |
| 需求的变动 | **非价格因素**（收入/偏好/相关品价格/预期）引起，曲线整体位移 |

**影响需求变动的非价格因素**（按重要性）：
1. 消费者收入（正常品↑→需求↑；低档品↑→需求↓）
2. 消费者偏好（偏好↑→需求↑）
3. 相关品价格（替代品↑→需求↑；互补品↑→需求↓）
4. 预期（预期涨价→现期需求↑）

### 1.2 供给函数与供给曲线

**供给定律**：价格上升→供给量上升

| 概念 | 要点 |
|:----|:-----|
| 供给函数 | Qs = -δ + γP |
| 供给曲线 | 右上方倾斜（价格↑→供给量↑） |
| 供给量的变动 | 价格变化引起，沿曲线移动 |
| 供给的变动 | 非价格因素（成本/技术/预期/相关品价格/政府政策）引起，曲线位移 |

**影响供给的非价格因素**：
1. 生产成本（成本↑→供给↓）
2. 技术水平（技术↑→供给↑）
3. 预期（预期涨价→现期供给可能↓）
4. 相关品价格
5. 政府政策（税收/补贴）

### 1.3 均衡价格

**均衡条件**：Qd = Qs

**最高限价**（政府规定低于均衡价）：
- 目的：保护消费者
- 后果：**供给短缺** → 排队/黑市/配给
- ⚠ 必须低于均衡价格才有效

**保护价格/最低限价**（政府规定高于均衡价）：
- 目的：保护生产者（农业）
- 后果：**供给过剩** → 政府收购/库存积压
- ⚠ 必须高于均衡价格才有效

### 1.4 弹性理论 ⭐每年必考

#### 需求价格弹性

| 弹性值 | 类型 | 含义 | 例子 |
|:-----:|:----|:----|:----|
| Ed = 0 | 完全无弹性 | 价格变→需求量不变 | 胰岛素 |
| 0 < Ed < 1 | 缺乏弹性 | 需求量变动<价格变动 | 粮食、盐 |
| Ed = 1 | 单位弹性 | 需求量变动=价格变动 | — |
| 1 < Ed < ∞ | 富有弹性 | 需求量变动>价格变动 | 奢侈品 |
| Ed → ∞ | 完全弹性 | 价格微变→需求剧变 | 完全竞争市场 |

**弧弹性公式**（考试用这个）：
```
Ed = (ΔQ/Q̄) / (ΔP/P̄) 
```
其中 Q̄ = (Q₁+Q₂)/2, P̄ = (P₁+P₂)/2

**影响弹性的因素**：
- 替代品越多→弹性越大
- 用途越广泛→弹性越大
- 时间越长→弹性越大
- 必需品→弹性小

**总收入与弹性的关系**（高频考点）：
- 缺乏弹性(Ed<1)：价格↑→总收入↑
- 富有弹性(Ed>1)：价格↑→总收入↓
- 单位弹性(Ed=1)：价格变→总收入不变

#### 需求交叉弹性 ⭐

```
Exy = ΔQx/Qx ÷ ΔPy/Py
```

| Exy | 关系类型 | 例子 |
|:--:|:--------|:----|
| Exy > 0 | **替代品** | 茶和咖啡 |
| Exy < 0 | **互补品** | 汽车和汽油 |
| Exy = 0 | 无关系 | — |

#### 需求收入弹性

```
Ey = ΔQ/Q ÷ ΔY/Y
```

- Ey > 1：高档品（收入↑→需求更快↑）
- 0 < Ey < 1：必需品
- Ey < 0：低档品（收入↑→需求↓）

#### 供给价格弹性

```
Es = ΔQs/Qs ÷ ΔP/P
```

- 时间越长→供给弹性越大
- 生产周期越短→供给弹性越大
- 调整产量越容易→供给弹性越大

## 第2章 消费者行为分析

### 2.1 效用理论

| 概念 | 定义 |
|:----|:-----|
| 总效用(TU) | 消费者在一定时期内消费一定数量商品所获得的**满足总量** |
| 边际效用(MU) | 每增加一个单位商品所增加的满足量（MU = ΔTU/ΔQ） |
| **边际效用递减** | 随着消费量的增加，消费者从每增加一单位商品中获得的满足增量是**递减的** |

**消费者均衡条件**（考试重点）：
```
MU₁/P₁ = MU₂/P₂ = ... = MUₙ/Pₙ = λ（货币的边际效用）
```
即：**每一元钱花在每种商品上得到的边际效用相等**

### 2.2 预算约束线

```
P₁X₁ + P₂X₂ = M（收入）
```

- 斜率 = -P₁/P₂
- 收入变化→曲线平移
- 某商品价格变化→曲线旋转

### 2.3 无差异曲线 ⭐

| 特征 | 说明 |
|:----|:-----|
| 1. 离原点越远 | 效用水平越高 |
| 2. 不相交 | 偏好一致性假设 |
| 3. 凸向原点 | **边际替代率递减**（MRS递减） |

**边际替代率 MRS**：
```
MRS = -ΔX₂/ΔX₁ = MU₁/MU₂
```
MRS递减的原因：边际效用递减。

**消费者效用最大化点**：无差异曲线与预算约束线的**切点**
此时：MRS = P₁/P₂ （即 MU₁/MU₂ = P₁/P₂）

## 第3章 生产和成本理论

### 3.1 生产函数

**短期生产函数**（至少一种要素固定）：
```
Q = f(L, K̄)  — 仅劳动可变
```

**总产量(TP)、平均产量(AP)、边际产量(MP)**：
```
AP = TP/L
MP = ΔTP/ΔL
```

**边际报酬递减**（短期生产基本规律）：
- 其他要素不变→连续增加某要素→达到某点后，MP递减

**三个生产阶段**（高频判断）：
- 阶段Ⅰ（0→AP最高点）：MP > 0且AP↑，固定要素利用不足
- 阶段Ⅱ（AP最高点→MP=0）：**合理生产区域**
- 阶段Ⅲ（MP<0）：不经济

### 3.2 成本函数

| 成本类型 | 公式 | 说明 |
|:--------|:----|:----|
| 总成本(TC) | TC = TFC + TVC | 固定+可变 |
| 平均总成本(ATC) | ATC = TC/Q | |
| 平均固定成本(AFC) | AFC = TFC/Q | Q↑→AFC↓ |
| 平均可变成本(AVC) | AVC = TVC/Q | |
| 边际成本(MC) | MC = ΔTC/ΔQ | **考试核心** |

**U型成本曲线**（必考）：
- ATC、AVC、MC 三条曲线都是**先降后升**（U形）
- MC 先于 AVC 到达最低点
- MC = AVC 时 → AVC最低
- MC = ATC 时 → ATC最低

**短期成本函数**（至少一种要素固定）：
- 固定成本：不随产量变动
- 变动成本：随产量变动

**长期成本函数**（所有要素可变）：
- 无固定成本，只有总成本
- **规模经济**（长期平均成本↓）：产量越大越便宜
- **规模不经济**（长期平均成本↑）：产量太大管理失效

### 3.3 利润最大化

```
利润 = 总收入(TR) - 总成本(TC)

利润最大化条件：MR = MC
```
完全竞争市场中：P = MR = MC

## 第4章 市场结构理论

### 4.1 四种市场结构对比 ⭐

| 特征 | 完全竞争 | 垄断竞争 | 寡头垄断 | 完全垄断 |
|:----|:--------|:--------|:--------|:--------|
| 企业数量 | **很多** | 较多 | 少数 | **一家** |
| 产品 | **同质** | **有差别** | 同质或有差别 | 无替代品 |
| 价格控制 | **无** | 有一些 | 有（相互依赖） | **很大** |
| 进入障碍 | **无** | 低 | 高 | **极高** |
| 长期利润 | **零** | 零 | 正 | 正 |
| 例子 | 农产品市场 | 餐饮/服装 | 电信/石油 | 自来水/电网 |

### 4.2 完全竞争市场

**短期**：P > ATC → 盈利；P = ATC → 盈亏平衡；P < AVC → 停产

**短期供给曲线**：MC曲线在AVC以上的部分

**长期**：企业的长期均衡 → 经济利润=0（P=ATC最低点=MC）
- 因为自由进出，超额利润吸引新企业进入，供给↑→价格↓→利润∅

### 4.3 完全垄断市场

**需求曲线**：就是整个市场的需求曲线（右下方倾斜）

**价格歧视**：
| 级别 | 定义 | 例子 |
|:---:|:----|:----|
| 一级 | 向每个消费者收不同的价 | 拍卖 |
| 二级 | 按购买量定价 | 批发价vs零售价 |
| 三级 | 按群体定价 | 学生票/老人票 |

**自然垄断**：单个企业生产比多个企业更便宜（具有规模经济），如电网、自来水。

**政府对垄断的干预**：
1. 法律手段（反垄断法/反不正当竞争法）
2. 价格管制 → 平均成本定价法(P=ATC) 或 边际成本定价法(P=MC)
   - 平均成本定价：企业盈亏平衡
   - 边际成本定价：企业亏损，需政府补贴

## 第5章 生产要素市场

### 5.1 生产要素需求

**引致需求**：对生产要素的需求**派生自**对产品的需求（消费者需求→产品需求→要素需求）

**联合需求**：生产要素必须**组合使用**（劳动+资本+土地等）

### 5.2 劳动供给曲线

**劳动供给曲线的特征**：**向后弯曲** ⭐
- 工资较低时：工资↑→劳动供给↑（替代效应>收入效应）
- 工资较高时：工资↑→劳动供给↓（收入效应>替代效应）— 选择更多闲暇

### 5.3 工资、地租、利息、利润

**工资**：劳动的价格。工会有助于提高工资（减少供给或增加需求）

**地租**：土地的价格。土地的供给**固定**（垂直供给曲线），地租由需求决定

**利息**：资本的价格。由资本的供给（储蓄）和需求（投资）决定

**正常利润**：企业经营所必须的最低利润（本质上是对企业家才能的报酬）
**超额利润**：超过正常利润的部分（来自垄断/创新/风险承担）

## 第6章 市场失灵和政府的干预

### 6.1 市场失灵的原因（四类）⭐

| 原因 | 定义 | 例子 | 干预措施 |
|:----|:----|:----|:--------|
| **外部性** | 私人成本≠社会成本 | 污染（负外部性） | 税收/补贴/许可证 |
| **公共物品** | 非竞争性+非排他性 | 国防/灯塔 | 政府提供 |
| **信息不对称** | 一方比另一方知道得多 | 二手车市场 | 信息披露/监管 |
| **垄断** | 市场势力导致低效 | 自然垄断 | 反垄断法/价格管制 |

### 6.2 外部性与科斯定理

**负外部性**：私人成本<社会成本 → 过度生产（如污染）
**正外部性**：私人收益<社会收益 → 生产不足（如教育）

**科斯定理**：只要**产权明确**且**交易成本为零**，市场交易可以解决外部性问题。

### 6.3 公共物品

| 特征 | 非竞争性 | 非排他性 |
|:----|:--------|:--------|
| **纯公共物品** | ✅ | ✅ | 国防、灯塔 |
| **准公共物品** | ❌或部分 | ❌或部分 | 教育、医疗、收费公路 |

**搭便车问题**：公共物品的非排他性导致人们不付钱也受益，私人部门不愿意提供。

### 6.4 信息不对称

**逆向选择**：交易前一方隐藏信息（二手车市场劣车驱逐好车）
**道德风险**：交易后一方改变行为（买了保险后更冒险）

## 第7章 国民收入核算和简单的宏观经济模型

### 7.1 国内生产总值GDP ⭐每年必考

**GDP定义**：一个国家或地区在一定时期内生产的**最终产品和服务的市场价值**

**GDP vs GNP**：
- GDP：**地域**概念（在中国境内生产的）
- GNP：**国民**概念（中国公民生产的）

**GDP的三种核算方法**：

```
1. 生产法 = 各行业总产出 - 中间投入
2. 收入法 GDP = 劳动者报酬 + 生产税净额 + 固定资产折旧 + 营业盈余
3. 支出法 GDP = 消费(C) + 投资(I) + 政府购买(G) + 净出口(X-M)
```

**支出法 GDP是最常用的核算方法**，四大部分：
| 部分 | 构成 |
|:----|:----|
| 消费(C) | 居民购买消费品和服务 |
| 投资(I) | 固定资产投资+存货投资（⚠不含股票债券等金融投资） |
| 政府购买(G) | **政府消费性支出**（⚠不包括转移支付） |
| 净出口(X-M) | 出口-进口 |

**GDP平减指数**（价格指数）：
```
GDP平减指数 = 名义GDP / 实际GDP × 100
```

### 7.2 宏观经济模型

**两部门经济**：
```
总需求 Y = C + I
总供给 Y = C + S
均衡条件：I = S
```

**三部门经济**（加入政府）：
```
总需求 Y = C + I + G
总供给 Y = C + S + T
均衡条件：I + G = S + T  即 I = S + (T - G)
```

**四部门经济**（加入国外）：
```
总需求 Y = C + I + G + (X-M)
总供给 Y = C + S + T
均衡条件：I + G + (X-M) = S + T
```

### 7.3 消费函数与储蓄函数 ⭐

```
消费函数：C = α + βY
  其中：α = 自发消费（收入为0时的最低消费）
        β = 边际消费倾向 MPC
        βY = 引致消费（随收入变化的部分）

平均消费倾向 APC = C/Y  （可能>1）
边际消费倾向 MPC = ΔC/ΔY  （0 < MPC < 1）

储蓄函数：S = Y - C = -α + (1-β)Y
平均储蓄倾向 APS = S/Y
边际储蓄倾向 MPS = 1 - MPC = ΔS/ΔY

核心关系：APC + APS = 1 且 MPC + MPS = 1
```

### 7.4 乘数效应 ⭐

```
投资乘数 k = 1/(1-MPC) = 1/MPS
```

- MPC越大→乘数越大→投资对经济的拉动效应越大
- 乘数>1：说明投资增加不仅直接拉动经济，还通过消费链产生多轮拉动

## 第8章 经济增长和经济发展理论

### 8.1 经济增长因素分解

**经济增长 = 劳动增长 + 资本增长 + 技术进步**

**索洛余值**（全要素生产率）：
```
经济增长率 = 技术进步率 + α×劳动增长率 + β×资本增长率
其中α+β=1
索洛余值 = 经济增长率 - α×劳动增长率 - β×资本增长率
```

### 8.2 经济周期⭐

**经济周期的四个阶段**：**复苏 → 繁荣 → 衰退 → 萧条**

**按周期长度分类**：
| 周期 | 长度 | 别称 |
|:---:|:----|:----|
| 短周期 | ~3-5年 | 基钦周期（库存周期） |
| 中周期 | ~8-10年 | **朱格拉周期**（设备投资周期） |
| 长周期 | ~50-60年 | **康德拉季耶夫周期**（技术革命周期） |

**先行指标**（预测）：股票指数、制造业订单、货币供应量
**一致指标**（同步）：GDP、工业增加值、就业
**滞后指标**（确认）：CPI、失业率

### 8.3 经济发展

**经济发展 ≠ 经济增长**
- 经济增长：GDP、人均GDP的增长（**数量**维度）
- 经济发展：经济增长 + 生活质量提升 + 技术进步 + 结构优化（**质量+数量**）

**新发展理念**：创新、协调、绿色、开放、共享

## 第9章 价格总水平和就业、失业

### 9.1 价格总水平

**衡量指标**：CPI（居民消费价格指数）最常用

**通货膨胀类型**：
| 类型 | 年通胀率 | 影响 |
|:----|:--------|:----|
| 爬行/温和 | <10% | 正常范围 |
| 奔腾/加速 | 10%-100% | 经济不稳定 |
| 恶性 | >100% | 货币崩溃 |

**通货膨胀产生的原因**：需求拉动/成本推动/结构性

### 9.2 失业

**失业类型**：
| 类型 | 原因 | 例子 |
|:----|:----|:----|
| 摩擦性失业 | 信息不完美+流动成本 | 换工作期间的失业 |
| 结构性失业 | 技能不匹配 | 技术替代人工 |
| 周期性失业 | **总需求不足** | 经济衰退期失业 |
| 自愿失业 | 不愿接受当前工资 | — |

**充分就业≠零失业**：充分就业时仍然存在**摩擦性失业+结构性失业**（合称自然失业）

**奥肯定律** ⭐：
> GDP每下降**高于潜在水平**的一部分→失业率上升
> 简单版：GDP每下降**2%**→失业率上升约**1%**

### 9.3 菲利普斯曲线 ⭐

**短期菲利普斯曲线**：失业率与通胀率**反向相关**（通胀↑→失业↓）
**长期菲利普斯曲线**：**垂直**（失业率最终回归自然失业率，通胀率不影响长期失业率）
- 长期中，货币**中性**：货币供给量只影响价格总水平，不影响产出和就业

## 第10章 国际贸易理论和政策

### 10.1 国际贸易理论

| 理论 | 提出者 | 核心观点 |
|:----|:------|:--------|
| **绝对优势理论** | 亚当·斯密 | 各国生产各自最擅长的产品，然后交换 |
| **比较优势理论** | 大卫·李嘉图 ⭐ | **只要两国生产某产品的相对成本不同**，就可以贸易互利。**"两优取重，两劣取轻"** |
| **要素禀赋理论** | 赫克歇尔-俄林 | 各国出口密集使用其丰富要素的产品 |
| **规模经济贸易理论** | 克鲁格曼 | 规模经济也是贸易的原因 |

### 10.2 国际贸易政策

| 政策 | 工具 |
|:----|:----|
| **保护主义** | 关税、非关税壁垒（进口配额/许可证/技术标准） |
| **自由贸易** | WTO框架、区域贸易协定（RCEP/CPTPP） |

**倾销**：出口国以低于国内价格甚至低于成本的价格向进口国销售产品

### 10.3 汇率与国际贸易

**本币贬值**（汇率↑）→ 出口↑ 进口↓ → 改善贸易收支
**本币升值**（汇率↓）→ 出口↓ 进口↑ → 恶化贸易收支

**马歇尔-勒纳条件**：本币贬值改善贸易收支的前提是**进出口需求弹性之和 > 1**

### 选择题

**1.** 在其他条件不变的情况下，如果某商品的价格下降10%，需求量增加8%，则该商品的需求价格弹性为：
A. 0.8（缺乏弹性） ✅
B. 1.25（富有弹性）
C. 0.8（富有弹性）
D. 1.25（缺乏弹性）

> 解析：Ed = 8%/10% = 0.8 < 1 → 缺乏弹性

**2.** 以下哪项会导致需求曲线向右移动？
A. 该商品价格下降
B. 替代品价格下降
C. 消费者收入增加 ✅
D. 该商品预期价格下降

**3.** 政府对某种商品实行最高限价，最可能导致：
A. 供给过剩
B. **供给短缺** ✅
C. 市场均衡
D. 生产者收入增加

**4.** 以下属于互补品的是：
A. 茶和咖啡
B. **汽车和汽油** ✅
C. 牛肉和猪肉
D. 苹果和香蕉

**5.** 边际消费倾向MPC为0.8时，投资乘数为：
A. 1.25
B. **5** ✅
C. 4
D. 8
> 解析：k = 1/(1-0.8) = 5

**6.** 短期菲利普斯曲线表明：

B. **失业率与通胀率负相关** ✅

D. 失业率与经济增长率正相关

B. **产品同质** ✅
C. 存在进入障碍

**8.** 收入法GDP中不包含以下哪项？

B. 生产税净额
C. **净出口** ✅（净出口是支出法的组成部分）
D. 营业盈余

A. 绝对优势理论
B. **比较优势理论** ✅
