## 二、客户金融需求分析

### 2.1 企业核心痛点

脑虎科技正处于从"科研突破"到"产业化量产"的关键跨越期——这是硬科技企业最"烧钱"也最危险的阶段。陶虎辞去厅局级公职ALL IN后，公司进入**创始人无退路、全力冲刺产业化**的新阶段。

| 痛点 | 具体表现 | 紧迫度 |
|------|---------|--------|
| **超级工厂建设+设备投入大** | 江西赣江新区工厂2026H2投产，万套级产能需大量设备采购（微纳加工设备、封装线、测试平台等） | 🔴 极高 |
| **医疗器械注册长周期高成本** | 三级医疗器械注册需2-3年，临床试验（扩大样本量）、GMP车间、体系认证等需持续投入 | 🔴 高 |
| **57例临床→扩大至200-300例** | 注册临床试验需要更大样本量，每例植入成本20-50万 | 🔴 高 |
| **硅基芯片自研（BCI专用芯片）** | 脑虎自研BCI信号采集/处理专用芯片，需MPW/Full Mask流片 | 🟡 中高 |
| **科研市场收入有限，尚未盈利** | 科研市场天花板低（数千万级别），医疗市场需拿证后才能大规模商业化 | 🟡 中 |

## 三、产品方案

### 3.1 核心策略定位

脑虎科技是国内侵入式BCI的绝对龙头，拥有顶级技术壁垒（蚕丝蛋白微创植入+2,640通道+57例临床+全球独家汉语解码）和顶级资本背书（陈天桥+红杉+中平资本）。陶虎辞去厅局级公职ALL IN后，创始人已无退路，公司全力冲刺产业化。邮储的角色是"产业化加速器"：提供从超级工厂建设→临床试验扩大→医疗器械注册→量产的**全周期金融伴跑**。

### 3.2 "脑虎产业化加速"综合方案

| 产品 | 额度 | 期限 | 用途 | 风控抓手 |
|------|------|------|------|---------|
| **超级工厂设备融资租赁** | 5,000万元 | 5年 | 江西工厂微纳加工设备/封装线/测试平台 | ①设备所有权（租赁期内）②陈天桥/盛大集团信用背书 |
| **临床试验专项研发贷** | 3,000万元 | 3年 | 三级医疗器械注册临床试验（扩大样本量至200-300例） | ①临床伦理批件+华山医院合作合同 ②临床进度里程碑分期放款 |
| **BCI芯片流片贷** | 1,000万元 | 2年 | 自研神经信号采集/处理芯片MPW+Full Mask | ①流片合同 ②已有55例临床数据作为技术验证 |
| **GMP+注册体系项目贷** | 1,000万元 | 3年 | GMP车间建设+ISO 13485+医疗器械注册申报 | ①以政府补贴/陈天桥后续投资为辅助还款来源 |
| **合计敞口** | **1亿元** | — | — | — |

### 3.3 超级工厂设备融资租赁（核心产品）

| 维度 | 方案 |
|------|------|
| 背景 | 脑虎江西"超级工厂"2026年1月动工，2026H2投产，目标万套级稳定供货 |
| 设备清单 | 微纳加工设备（光刻/刻蚀/沉积）、柔性电极封装线、BCI芯片测试平台、手术机器人装配线、可靠性测试设备 |
| 机制 | 邮储提供设备融资租赁，设备所有权在租赁期内归属邮储/租赁公司，脑虎按期支付租金 |
| 额度 | 覆盖设备采购金额的60-80%，5,000万元 |
| 亮点 | ①脑虎技术壁垒极高，设备专用性强（非通用设备），残值可控 ②江西政府大概率有建厂补贴/税收优惠 |

### 3.4 临床试验专项研发贷

脑机接口医疗器械注册是全球前沿监管领域，NMPA尚无成熟审批路径，但这恰恰意味着——谁先拿证谁定义标准。

| 维度 | 方案 |
|------|------|
| 背景 | 55例短期临床已完成，注册临床需扩大至200-300例，单例植入成本20-50万 |
| 机制 | 以临床进度为里程碑分期放款：伦理批件→首例植入→50例→100例→200例 |
| 额度 | 3,000万元，覆盖植入电极/手术费/住院/随访/数据管理费用 |
| 合作医院 | 华山医院（神经外科全国第一）、瑞金医院 |
| 风控特色 | **技术里程碑=放款节点**：每完成一个临床里程碑，释放下一笔，实现"临床进度与信贷风险同步" |

### 3.5 BCI芯片流片贷

| 维度 | 方案 |
|------|------|
| 背景 | 脑虎自研神经信号采集/处理专用ASIC芯片（替代进口芯片），需MPW→Full Mask |
| 机制 | MPW阶段：覆盖流片费用80%（金额小，50-200万）；Full Mask阶段：覆盖50%（500-1,000万） |
| 技术验证 | 已有55例临床数据作为芯片设计验证依据 |

### 3.6 联合创新

| 项目 | 内容 | 价值 |
|------|------|------|
| **"脑机接口金融标准"** | 邮储+脑虎联合制定：将脑机接口医疗器械注册进度（伦理→临床→注册检验→拿证）映射为信贷评级因子 | 全球首创"脑机接口金融评级模型"，定义新兴赛道风控标准 |
| **"医疗器械注册加速贷"SOP** | 基于脑虎案例提炼"创新医疗器械注册全周期金融伴跑"标准流程 | 可向其他创新医疗器械企业（如脑虎生态链、华山医院衍生企业）复制 |
| **中科大校友脑科学创业金融** | 陶虎为中科大校友（本科），联合打造"中科大校友硬科技创业金融"品牌 | 中科大在脑科学/量子/AI/芯片领域校友资源极丰富，陶虎辞去厅局级ALL IN的标杆效应极强 |
| **超声波BCI新赛道** | 关注原联合创始人彭雷的新公司格式塔科技（超声波BCI，2026.1成立），可作为脑机接口赛道延伸拓客 | 同一创始人圈子+同一投资方（陈天桥），自然连带机会 |
