# 上海光机所 & 杭州光机所 — 科技金融合作方案

> 陈颖芳 | 邮储银行上海分行科技金融事业部 | 2026-05-27
> 用途：行内汇报 + 合作方初次接洽

---

## 一、 为什么是光机所？（一句话价值主张）

上海光机所是中国激光光学技术的"国家队"，杭州光机所是其成果产业化主阵地——5年孵化70+企业、总估值超500亿元。这批企业正处于"技术突破→量产放量"的临界点，是科技金融的优质标的，也恰好是银行尚未系统覆盖的增量市场。

**一句话定位**：以杭州光机所孵化生态为入口，以邮储"研发贷+知识产权质押"为产品载体，建立"科研院所+金融机构+科创企业"三方联动的科技金融新模式。

### 2.4 两所关系：上海研发 → 杭州转化

```
上海光机所（基础研究+技术发明）
        ↓ 技术导入
杭州光机所（概念验证+公司孵化+产业加速）
        ↓ 企业毕业
市场（科创板上市/并购/独立发展）
```

两所已形成"上海基础研究→杭州/台州概念验证与产业化"的长三角一体化协同模式，2025年底被科技部列为"长三角一体化科技成果转移转化合作典范"。

## 四、 合作模式设计

### 4.1 三层合作架构

```
┌─────────────────────────────────────────────────────┐
│  第一层：战略框架合作（行级）                          │
│  邮储银行上海分行 ↔ 上海光机所 / 杭州光机所             │
│  签署战略合作协议，建立常态化对接机制                    │
├─────────────────────────────────────────────────────┤
│  第二层：产品服务嵌入（业务级）                         │
│  针对孵育企业集群，定制"研发贷+知识产权质押+投贷联动"     │
│  产品包，嵌入杭州光机所八大孵化平台中的"投融资平台"       │
├─────────────────────────────────────────────────────┤
│  第三层：联合创新实验（项目级）                         │
│  借鉴邮储天津分行"生物制造科技金融联合实验室"模式，       │
│  探索共建"光电科技金融联合实验室"                       │
└─────────────────────────────────────────────────────┘
```

### 4.2 已可复用的邮储体系内先例

| 先例 | 合作方 | 可复用要素 |
|------|--------|-----------|
| 天津"生物制造联合实验室" | 天大+南开+合成生物中心 | 六方共建架构："政产学研用金" |
| 北京"科金荟"量子专场 | 北京科创中心 | "U益创"产品体系+3000万预授信模式 |
| 潍坊农科院战略合作 | 潍坊市农科院 | "科技指导+金融赋能"双轮驱动 |
| "火炬引航 邮企共进" | 工信部火炬中心 | 在高新区设科技金融特色网点 |

### 4.3 推荐合作切入路径

**第一步（1-2个月）**：拜访建立关系
- 通过中科大校友网络（中科大与中科院关系紧密）或嘉定区政府（上海光机所所在地）获取引荐
- 拜访上海光机所成果转化处 / 杭州光机所产业合作部
- 参加一次杭州光机所的孵育企业路演/开放日

**第二步（3-4个月）**：试点业务落地
- 从孵育企业中选取3-5家有明确融资需求的企业，匹配"研发贷"或"知识产权质押贷"
- 首单建议：富加镓业（氧化镓扩产）或芯空光联（空芯光纤产业化），两个项目均有清晰的固定资产/设备采购需求，符合银行授信偏好

**第三步（6-12个月）**：战略合作签约
- 签署邮储上海分行与杭州光机所的战略合作协议
- 将邮储金融产品嵌入杭州光机所投融资平台，成为推荐合作银行
- 探索在嘉定/富阳设立科技金融特色服务窗口

## 六、 风险提示

| 风险类型 | 说明 | 应对 |
|---------|------|------|
| **技术风险** | 氧化镓、空芯光纤处于产业化早期，技术路线可能变化 | 以设备贷款/知识产权质押为主，减少纯信用敞口 |
| **估值风险** | 孵育企业VC估值可能偏高，与传统银行授信逻辑冲突 | 引入投贷联动，与杭州光机所合作VC共享尽调信息 |
| **集中度风险** | 单一技术方向/单一孵化器集中度过高 | 首期试点3-5家，覆盖2-3个不同技术方向 |
| **人员投入** | 光电技术专业性强，信贷人员需要学习曲线 | 借力杭州光机所的技术团队做联合尽调 |
