## 三、 产品方案

### 3.1 对公融资产品包

#### （1）智算项目贷款（主打产品）

| 要素 | 方案 |
|------|------|
| 产品类型 | 中长期项目贷款 |
| 额度 | 5亿元（首次授信，后续可增至10亿） |
| 期限 | 3-5年，匹配智算项目建设+回款周期 |
| 用途 | 智算中心设备采购（GPU服务器/存储/网络）+ 软件集成 |
| 担保方式 | 项目应收账款质押 + 智算设备抵押 + 母公司担保 |
| 利率 | 对标同业科技企业贷款利率，LPR下浮5-10% |
| 还款来源 | ①政企项目回款（政府/运营商智算采购合同）②金融软服收入 |

#### （2）研发贷（创新产品）

| 要素 | 方案 |
|------|------|
| 产品类型 | 中长期信用贷款（匹配邮储2025年发布的"研发贷"） |
| 额度 | 1-2亿元 |
| 期限 | 最长10年 |
| 用途 | AI for Process研发、乾坤数智底座升级、50个智能体开发 |
| 担保方式 | 知识产权质押（神州持有300+金融科技相关专利/软著） |
| 特色 | 可联合人保财险提供"信贷+保险"组合 |

#### （3）并购贷款

| 要素 | 方案 |
|------|------|
| 额度 | 3亿元（不超过并购交易价款的60%） |
| 期限 | 5年 |
| 用途 | AI/金融科技领域战略并购 |
| 条件 | 并购标的需经我行技术尽调通过 |

#### （4）供应链金融

| 要素 | 方案 |
|------|------|
| 模式 | 反向保理 + 订单融资 |
| 额度 | 3亿元 |

## 五、 推进路径

### 5.1 三阶段推进

```
Phase 1（1-2月）：存量升级 + 快赢
├─ 代发工资迁移（已有合作关系，顺理成章）
├─ 现金管理升级（基于现有财资云）
├─ 员工消费贷批量授信
└─ 目标：3个月内落地，制造合作势头

Phase 2（3-6月）：主力产品落地
├─ 智算项目贷款授信审批
├─ 研发贷（知识产权质押）首笔放款
├─ 供应链金融嵌入财资云平台
└─ 目标：6个月内有1-2个大额产品落地

Phase 3（6-12月）：创新深化
├─ 联合技术价值评估模型V1.0
├─ 科创债券承销（视市场窗口）
├─ 并购贷款项目储备
└─ 目标：12个月内建成"银企联合创新"标杆模式
```

## 七、 与已有合作的衔接

本次综合金融服务方案**不是另起炉灶**，而是在5个已有合作基础上自然延伸：

```
已有合作                        本次新增
────────────────────────────────────────────
财资云/云司库          →    现金管理升级 + 供应链金融嵌入
数字人民币预付费       →    数字人民币对公场景拓展
数智化转型标准         →    技术价值评估标准联合制定
信创框架服务           →    智算项目贷款（信创设备采购）
长三角论坛同台         →    联合创新实验室 + 联合发布
```
