## 第1章：市场需求、供给与均衡价格

### 1.1 需求与供给的基本概念

| 概念 | 定义 | 关键点 |
|------|------|--------|
| 需求 | 购买欲望+支付能力 | 两个条件缺一不可 |
| 需求规律 | 价格↑ → 需求↓（反方向） | 价格与需求的负相关关系 |
| 供给规律 | 价格↑ → 供给↑（正方向） | 价格与供给的正相关关系 |

**考点：区分需求变动（曲线位移）与需求量变动（曲线上移动）**
- 需求量变动：价格变化引起的 → 沿同一条曲线移动
- 需求变动：价格以外的因素（收入/偏好/相关品价格）→ 曲线整体位移

### 1.2 市场均衡

**均衡价格形成**：需求曲线与供给曲线交点，该点价格下需求量=供给量

| 场景 | 后果 | 常见考题 |
|------|------|---------|
| 最高限价 < 均衡价 | 供不应求→短缺（排队/黑市） | 住房限价、药品限价 |
| 最低限价 > 均衡价 | 供过于求→过剩（政府收购/失业） | 最低工资、农产品保护价 |

**记忆口诀**：最高限价→短缺（限到下面去），最低限价→过剩（撑在上面）

### 1.3 弹性分析（高频计算考点）

**需求价格弹性 Ed = |%ΔQ / %ΔP|**

| Ed范围 | 类型 | 含义 | 例子 |
|--------|------|------|------|
| Ed=0 | 完全无弹性 | 价格变需求不变 | 胰岛素 |
| 0<Ed<1 | 缺乏弹性 | 需求对价格不敏感 | 粮食、电 |
| Ed=1 | 单位弹性 | 同比例变动 | 理论情况 |
| Ed>1 | 富有弹性 | 需求对价格敏感 | 奢侈品、非必需品 |
| Ed→∞ | 完全弹性 | 价格小幅涨需求归零 | 完全竞争市场 |

**考点：弹性和总收益的关系**
- 缺乏弹性（Ed<1）：涨价→总收益增加 **（谷贱伤农的原理）**
- 富有弹性（Ed>1）：降价→总收益增加
- 单位弹性（Ed=1）：价格变动→总收益不变

**需求交叉弹性 Exy = %ΔQx / %ΔPy**
- Exy > 0 → 替代品（牛肉↑→猪肉需求↑）
- Exy < 0 → 互补品（汽油↑→汽车需求↓）
- Exy = 0 → 无关品

**需求收入弹性 Ey = %ΔQ / %ΔY**
- Ey > 1 → 高档品（收入增幅度小于需求增）
- 0 < Ey < 1 → 必需品
- Ey < 0 → 低档品（收入增需求反降）

**供给价格弹性 Es = %ΔQs / %ΔP**
- 短期供给弹性小，长期供给弹性大（产能调整需要时间）

## 第2章：消费者行为分析

### 2.1 效用理论

| 概念 | 说明 | 考点 |
|------|------|------|
| 总效用TU | 消费一定量商品获得的总满足感 | 随消费量增加而增加 |
| 边际效用MU | 每多消费一单位增加的满足感 | 递减规律 |
| 边际效用递减 | 连续消费中MU逐渐下降 | **必考**：水与钻石悖论的解释 |

**消费者均衡条件**：MUx/Px = MUy/Py = λ（单位货币的边际效用相等）

**记忆**：最后一元钱花在哪里得的满足感都一样，就不调整了。

### 2.2 无差异曲线（了解基本特征）

- 离原点越远→效用越大
- 同一平面任意两条不相交
- 凸向原点（边际替代率递减）
- 斜率的绝对值 = 边际替代率 MRSxy = MUx/MUy

### 2.3 预算约束线

- 预算线方程：Px·X + Py·Y = I
- 收入变化 → 平移；价格变化 → 旋转

**考点：收入效应 vs 替代效应**
- 正常品：价格↓ → 替代效应（多买此）+ 收入效应（多买此）= 两者同方向
- 低档品：价格↓ → 替代效应（多买此）+ 收入效应（少买此）= **总效应不确定**
- 吉芬商品：价格↓ → 替代效应（多买此）+ 收入效应大幅（少买此）= 总效应为负（最贵时消费最多）

## 第3章：生产和成本理论

### 3.1 生产函数

| 概念 | 定义 | 关键点 |
|------|------|--------|
| 短期 | 至少一种要素固定 | 劳动可变，资本固定 |
| 长期 | 所有要素可变 | 企业可调整规模 |
| 边际产量MP | 加一单位劳动增加的产出 | 先增后减→**边际报酬递减** |
| 总产量TP | 所有要素投入的总产出 | 与MP的关系决定形状 |

**边际报酬递减规律**（必考）：连续增加可变要素，达到一定点后MP下降。**不意味着递减才发生，而是达到某一点后开始。**

### 3.2 成本函数

| 成本类型 | 公式 | 说明 |
|----------|------|------|
| 总成本TC = TFC + TVC | 固定+可变 | 短期成本 |
| 平均总成本ATC = TC/Q | 单位产品成本 | U型曲线 |
| 平均固定成本AFC = TFC/Q | 单位固定成本 | 随Q增加递减 |
| 平均可变成本AVC = TVC/Q | 单位可变成本 | U型曲线 |
| 边际成本MC = ΔTC/ΔQ | 多生产一单位增加的成本 | U型曲线，在ATC最低点穿过ATC |

**MC、ATC、AVC的关系图**（必考图形题）：
- MC曲线先降后升
- MC穿过ATC的最低点
- MC穿过AVC的最低点
- AVC在ATC下方

### 3.3 规模报酬

| 类型 | 含义 | 原因 |
|------|------|------|
| 规模报酬递增 | 要素↑1倍→产出↑>1倍 | 专业化分工、设备不可分割 |
| 规模报酬不变 | 要素↑1倍→产出↑1倍 | 最优规模 |
| 规模报酬递减 | 要素↑1倍→产出↑<1倍 | 管理协调成本上升 |

## 第4章：市场结构理论

### 4.1 四种市场结构

| 特征 | 完全竞争 | 垄断竞争 | 寡头垄断 | 完全垄断 |
|------|---------|---------|---------|---------|
| 企业数量 | 很多 | 较多 | 少数 | 一家 |
| 产品差异 | 同质 | 有差异 | 可同质可差异 | 唯一（无替代品） |
| 价格控制力 | 无（价格接受者） | 有限 | 较大 | 很大 |
| 进入壁垒 | 无 | 低 | 高 | 极高 |
| 例子 | 农产品 | 餐饮、服装 | 电信、石油 | 公用事业、专利药 |

### 4.2 完全竞争企业的决策

**短期利润最大化条件**（必考）：**MR = MC**

- 完全竞争企业：MR = P（价格不变，每多卖一单位的收入等于价格）
- 所以条件简化为：**P = MC**
- 短期亏损仍生产：P > AVCmin 就继续生产（覆盖可变成本）
- 停止营业点：P = AVCmin

**记忆**：补住可变成本就干，补不住就停。

### 4.3 完全垄断企业

- 垄断企业是价格制定者
- 利润最大化：仍然 MR = MC
- 垄断企业的需求曲线 = 行业需求曲线（向右下倾斜）
- 价格歧视（三级）：
  - 一级：每个消费者收不同价（最理想，现实中难做到）
  - 二级：不同批量不同价（实务中常见）
  - 三级：不同市场不同价（学生票/成人票）

## 第5章：生产要素市场

### 5.1 引致需求

- 对生产要素的需求是"引致需求"——由对产品的需求派生而来
- 联合需求：各种生产要素须组合使用，相互依赖

### 5.2 劳动供给曲线（特殊）

- 劳动供给曲线不是单调递增——它是**向后弯曲**的：
  - 工资低时：工资↑ → 劳动供给↑（替代效应占主导）
  - 工资高到一定程度：工资↑ → 劳动供给↓（收入效应占主导，宁愿用收入换闲暇）

**替代效应 vs 收入效应（劳动的）**：
- 替代效应：工资↑→闲暇更贵→多工作少休息
- 收入效应：工资↑→收入↑→少工作多休息

### 5.3 资本和土地

- **资本**：机器、设备、厂房等——用于生产其他商品的人造资源
- **利息**：资本的价格 = 机会成本 + 风险补偿
- **地租**：土地的价格。土地供给固定→地租由需求决定

## 第6-10章：宏观经济（第2周 6/8-6/14）

第6-10章是高分解模块，包括GDP核算、宏观经济模型、经济周期、失业通货膨胀等。**将在Week 2学习时细化。**

### 第6章逻辑速览：国民收入核算

| 概念 | 公式/说明 |
|------|----------|
| GDP = C + I + G + (X-M) | 支出法（必背！） |
| GNP = GDP + 国外净要素收入 | 国民 vs 国土 |
| GDP缩减指数 = 名义GDP/实际GDP × 100 | 综合价格指数 |

### 第7章逻辑速览：宏观经济模型

- **乘数效应**：K = 1/(1-MPC)，MPC越大乘数越大
- 储蓄-投资恒等式：I = S（两部门经济）
- **IS-LM模型**：理解即可（中级要求识别）

### 第8-9章逻辑速览：经济周期+失业+通胀

| 类型 | 特征 | 对策 |
|------|------|------|
| 周期性失业 | 总需求不足 | 扩张性财政/货币政策 |
| 结构性失业 | 技能不匹配 | 职业培训 |
| 摩擦性失业 | 正常流动 | 不需要干预 |
| 需求拉动通胀 | 总需求过大 | 紧缩政策 |
| 成本推动通胀 | 原材料工资涨 | 收入政策 |

### 第10章逻辑速览：国际贸易

- **绝对优势**（亚当·斯密）：生产成本绝对低
- **比较优势**（李嘉图）：机会成本相对低（必考区别）
- 赫克歇尔-俄林：要素禀赋决定贸易结构
- 汇率变动：本币贬值→出口↑进口↓（J曲线效应）

### 必考计算公式

| 公式 | 用途 | 记忆法 |
|------|------|--------|
| Ed = %ΔQ / %ΔP | 需求价格弹性 | 取绝对值 |
| Exy = %ΔQx / %ΔPy | 交叉弹性 | 正→替代，负→互补 |
| MR = MC | 利润最大化条件 | 多赚≥多花就继续 |

### 易混淆概念

| 易混对 | 关键区别 |
|--------|---------|
| 需求量变动 vs 需求变动 | 前者价格引起（曲线上动），后者其他因素（曲线平移） |
| 供给量变动 vs 供给变动 | 同上逻辑 |
| 替代品 vs 互补品 | 交叉弹性正负号 |
| 正常品 vs 低档品 | 收入弹性正负号 |
| 名义GDP vs 实际GDP | 是否调整通胀 |
| GDP vs GNP | 国土 vs 国民原则 |
| 绝对优势 vs 比较优势 | 绝对 vs 相对成本 |
| 自愿失业 vs 非自愿失业 | 主观不愿 vs 客观不能 |
| 需求拉动 vs 成本推动通胀 | 需求侧 vs 供给侧 |
