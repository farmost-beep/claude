## 一、支行行长能力模型

### 1.1 业务能力

#### 1.1.1 信贷业务能力

| 子模块 | 能力要求 | 达标标准 | 自评 |
|---|---|---|---|
| 对公信贷 | 独立完成贷前调查、授信方案设计、贷后管理全流程 | 可主持贷审会，签字权限预计在500万元以内（科技型企业授信），大额需经分行贷审会集体审批。具体授权额度以行内授权文件为准，[待确认：当前实际签字权限上限] | 熟悉对公信贷全流程，具备主持或参与贷审会的经验。在科技型企业信贷领域有深度积累，传统行业（制造业、房地产、贸易融资等）信贷经验相对较少 |
| 零售信贷 | 熟悉个人住房贷款、消费贷、经营贷的产品规则与风控要点 | 能指导客户经理处理复杂零售贷款审批 | 接触较少，主要集中在对公业务。零售信贷产品知识和审批实务经验是需要系统补强的领域 |
| 不良清收 | 掌握重组、诉讼、核销等手段，能制定一户一策化解方案 | 不良率控制在分行考核线以下 | 有一定了解，在科技型企业不良贷款处置方面有实务经验。传统行业大额不良清收经验可能不足，[待确认：是否主导过不良清收项目] |

#### 1.1.2 风险管理能力

- **信用风险**：行业分析、财务分析、交叉验证技巧 -- 在科技型企业信用评估方面有较强能力，传统行业信用风险识别需补强
- **操作风险**：案件防控、员工行为管理、柜面操作规范 -- 相对短板，作为事业部干部对柜面操作和案件防控实务接触较少
- **合规反洗钱**：熟悉反洗钱法规、大额可疑交易识别、监管检查应对 -- 具备基本合规意识，但完整应对监管检查的经验可能不足
- **声誉风险**：客户投诉处理、舆情应对、媒体沟通 -- 相对短板，未独立承担过声誉风险处置的第一责任人角色
- **市场风险**：利率风险、流动性管理的实务操作 -- 了解基本概念，缺乏资产负债管理的实操经验

> 信用风险（科技型企业方向）是强项，操作风险和声誉风险是明显的短板。建议：主动参与分行组织的运营检查/内控检查以积累操作风险实务经验；关注分行声誉风险管理培训和应急演练。是否参与过分行级别的风控专项工作？[待本人补充]

#### 1.1.3 零售与财富管理能力

- 存款营销：财政存款、社保资金、企业年金等大额存款的获取与维护 -- 以对公存款营销为主，机构类存款（财政、社保等）经验可能不足
- 理财产品销售：合规销售、投资者适当性管理、双录管理 -- 非主要工作领域，需系统学习零售理财合规要求
- 信用卡与电子银行：场景化获客策略，数字人民币推广 -- 了解但不深入，可通过参与分行零售条线专题会快速补强
- 私人银行 / 高端客户：高净值客户资产配置需求的理解 -- 有对公高端客户（企业家客群）的服务经验，纯个人高净值客户的财富管理经验较少

> 当前以对公和科技金融业务为主，零售业务是明显的短板。若竞聘支行行长，需要系统补强零售条线的管理知识。当前直接管理的零售AUM规模、管户的高净值客户数量，[待本人补充具体数据]

#### 1.1.4 对公与机构业务能力

- 政府机构客户：财政、社保、公积金、学校、医院等公务卡/代发工资业务 -- 通过科技金融业务与科委、经信委有工作联系，但财政、社保等传统政府机构客户的拓展经验不足
- 国企/央企客户：供应链金融、现金管理、银企直连 -- 有一定接触（科技型国企），但大型央企集团的综合金融服务经验有待积累
- 普惠金融：小微贷款（税贷、流水贷、抵押快贷等）的规模与质量平衡 -- 科技型小微企业是日常工作对象，普惠金融理念相通，有一定经验积累
- 绿色金融：碳减排贷款、ESG评级的实务 -- 新兴领域，与科技金融有交叉（如新能源、节能环保赛道），有拓展潜力

#### 1.1.5 科技金融（TechFin）专项能力 -- 核心竞争力

- **投贷联动**：与PE/VC机构的合作机制、认股权贷款的设计与风控
- **知识产权质押**：专利、软件著作权等无形资产的估值与处置
- **专精特新企业服务**：对硬科技企业的技术判断能力和赛道分析
- **科技保险与担保**：与科委/经信委的政策性担保工具协同

> 这是当前岗位的优势领域，也是未来差异化竞争的核心。详见第三章。

## 三、科技金融投资框架构建计划

### 3.1 框架核心要素

目标：形成一套 **"邮储体系内可复制、可讲授、可落地"** 的科技金融投资方法论，成为个人核心竞争力标签。

#### 3.1.1 理论层：定义科技金融的投资哲学

- 科技金融与传统投贷的本质区别：从"看报表+看抵押"转向"看团队+看技术+看赛道"
- 硬科技企业的估值逻辑：技术成熟度（TRL）、市场天花板（TAM/SAM/SOM）、创始人能力
- 四种退出/回款路径的逻辑：IPO、并购、经营现金流、政府补贴
- 你个人在科技金融实践中提炼的核心投资理念。建议方向：围绕"投科技就是投团队"、"看技术壁垒不如看客户验证"、"硬科技的尽头是现金流"等原则，提炼2-3条个人信条。请本人总结，[待补充]

#### 3.1.2 实战层：形成可操作的工具箱

- **行业/赛道筛选矩阵**：半导体、新能源、生物医药、AI、新材料等赛道的关键判断指标
- **企业技术评估清单**：专利质量（引用数、国际布局）、核心团队背景、研发投入占比、客户验证
- **风控模型**：科技型企业的信用评级补充模型（轻资产、高前期亏损、现金流后置等特征如何对标传统风控）
- **投贷联动操作手册**：合作PE/VC筛选标准、认股权定价、投后管理节点
- 是否已有操作手册或作战地图的雏形？（如贷款准入标准清单、尽调问卷模板、行业分析模板等）。估算：在实践中应有部分工具积累但尚未系统化整理。请本人列出已有的工作文档和工具清单，[待补充]

#### 3.1.3 案例层：建立项目库

- **成功案例**（目标：整理3个以上）：完整复盘，含投资逻辑、尽调过程、投后管理、退出/回款、经验总结

## 四、晋升路径图

### 4.1 邮储银行典型晋升阶梯

```
部门主管/副经理（当前：科技金融事业部高级副经理）
    ↓ (2-3年)
部门经理 / 支行副行长（主持工作）
    ↓ (3-5年，含竞聘/交流/外派)
一级支行行长 / 分行部门总经理
    ↓ (5-8年)
分行副行长 / 总行部门副总经理
    ↓
分行行长 / 总行部门总经理
```

### 4.2 路径A：事业部内部晋升（主推路径）

```mermaid
graph LR
    A["当前: 科技金融事业部<br/>高级副经理"] --> B["下一站: 科技金融事业部<br/>副总经理/总经理助理"]
    B --> C["关键跳板: 支行副行长<br/>(分管对公/科技金融)"]
    C --> D["目标: 一级支行行长"]
```

**优势**：延续科技金融标签，积累在纵向专业领域的深度和权威性。
**风险**：可能缺乏综合管理经验和零售/运营的磨炼。
**弥补策略**：主动申请到一线支行挂职副行长（6-12个月），补上支行运营和零售的管理经验。

### 4.3 路径B：支行一线通道

```mermaid
graph LR
    A["当前岗位"] --> B["支行行长助理<br/>或分管对公副行长"]
    B --> C["一级支行行长"]
```

**优势**：全方位锻炼，传统晋升路径，干部履历"完整"。
**风险**：科技金融标签可能被稀释。
**策略**：在支行继续深耕科技金融业务线，把支行做成"科技金融特色支行"。

### 4.4 关键时间节点（预估）

| 阶段 | 时间 | 目标 | 关键动作 |
|---|---|---|---|
| 短期 | 0-6个月 | 明确晋升方向，完成科技金融框架V1.0 | 框架搭建+行内分享1次 |
| 短期 | 6-12个月 | 争取挂职/交流机会，或T+1岗位任职 | 主动表达晋升意愿，提交履历 |
| 中期 | 1-2年 | 晋升至部门副总经理级或支行副行长级 | 至少1-2个标志性科技金融项目落地，有高质量内部报告/文章 |
| 中期 | 2-4年 | 达到支行行长任职条件 | 通过竞聘/选聘，获得行长任职资格 |
| 长期 | 4年+ | 支行行长 | 形成个人品牌，带领支行做出特色业绩 |
| 长期 | 8年+ | 分行副行长级 | 在更大平台上推广科技金融方法论 |

> 行内干部竞聘周期通常每1-2年组织一次。国有银行支行行长任职一般要求：中级及以上职称（经济师/会计师）、银行从业资格证；高级职称（高级经济师）、AFP/CFP、FRM等为加分项。需确认：①行内最近一次竞聘时间和结果；②行内对支行行长的具体资格要求；③是否需要参加上级行组织的后备干部培训班；④49岁的年龄在邮储银行支行行长竞聘中是否存在年龄限制或不利因素。以上各项，[待本人确认]
