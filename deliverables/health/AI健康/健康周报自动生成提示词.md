# 健康周报自动生成提示词

> 用途：每周日复制下面这段提示词发给Claude，自动生成本周健康周报
> 前置条件：`健康数据/health_data.json` 中已录入本周每日数据
> 适用对象：陈颖芳（49岁，男性，178cm）

---

## 使用方法

每周日晚，将以下提示词完整复制发给Claude：

---

## 复制此线以下内容 ---

```
[健康周报] 请生成本周健康周报。

## 步骤1：读取数据
1. 读取文件 `/Users/cyingfang/claude/deliverables/health/健康数据/health_data.json`
2. 确定本周的日期范围：
   - 计算本周一（ISO周）到本周日的日期
   - 或：如果今天是周日，本周 = 本周一到今天
3. 筛出 `daily_records` 中 `date` 在本周范围内的记录

## 步骤2：计算本周汇总指标
基于本周有数据的记录，计算以下内容（标注"数据缺失X天"如果某指标有缺失）：

| 类别 | 指标 | 计算方式 | 目标值 |
|------|------|---------|--------|
| 血压 | 本周均值 | 本周所有记录的收缩压/舒张压均值 | <130/80 mmHg |
| 血压 | 本周最高 | 本周单日最高收缩压/舒张压 | — |
| 血压 | 测量天数 | 本周测了几天血压 | >=5天 |
| 体重 | 本周值 | 最近一次体重记录 | <76kg (BMI<24) |
| 体重 | 周变化 | 与上周最后一次记录对比 | 周增幅<1kg |
| 步数 | 日均步数 | 本周步数记录取均值 | >=8000步 |
| 步数 | 达标天数 | 步数>=8000的天数 | — |
| 睡眠 | 日均时长 | 本周睡眠时长取均值 | 6.5-8.5h |
| 睡眠 | 日均入睡 | 本周入睡时间取均值 | <=23:30 |
| 睡眠 | 质量均值 | 本周睡眠质量取均值 | >=3分 |
| 饮水 | 日均饮水 | 本周饮水记录取均值 | >=2.5L |
| 运动 | 运动天数 | 本周exercise_done=true的天数 | >=5天 |
| 运动 | 快走总分钟 | 本周快走分钟数总和 | — |
| 运动 | 力量训练总组数 | 本周力量训练组数总和 | — |
| 饮食 | 达标天数 | 本周diet_score>=7的天数 | >=6天 |
| 饮食 | 日均达标项 | 本周diet_score均值 | >=7项 |
| 吸烟 | 本周情况 | 本周是否吸烟 | 否 |
| 饮酒 | 本周情况 | 本周饮酒次数 | 0次 |

## 步骤3：趋势分析
对比上周数据（如有），标记每个指标的变动方向：
- ↑ 改善（如血压下降、步数上升、体重下降）
- → 持平（变化在±5%以内或无明显变化）
- ↓ 恶化（如血压上升、步数下降、睡眠变差）

## 步骤4：三级预警检查
基于 `alert_thresholds` 定义和本周数据，逐项检查：

### 🔴 红灯（需立即就医或紧急处理）
- [ ] 血压 >140/90 连续3天 → 建议本周内挂号心内科
- [ ] 血压 >145/95 连续5天 → 必须就医，不可延迟
- [ ] 运动中胸痛/胸闷/心慌/出冷汗 → 立即停止运动，必要时急诊
- [ ] 连续3天未运动（6月W2起适用，W1基线周豁免）

### 🟡 黄灯（需调整生活方式）
- [ ] 血压连续3天 >135/85 → 回顾本周盐摄入和睡眠
- [ ] 周体重增幅 >1kg → 复查饮食记录，重点关注晚餐时间和宵夜
- [ ] 连续3天睡眠 <6小时 → 当天运动改为散步/休息，优先补睡眠
- [ ] 连续3天饮水 <2L → 设置手机闹钟每小时提醒喝水
- [ ] 运动后肌肉酸痛持续 >24小时 → 下周减量，强度过大

### 🟢 绿灯（保持）
- [ ] 本周运动>=5天 且 饮食>=6天达标 → "本周一切平稳，继续保持"
- [ ] 所有指标在目标范围内 → 周末可以给自己一个小奖励

对于每个触发的预警，输出：
- 预警等级 + 预警内容
- 什么意思（通俗解释）
- 行动建议（具体怎么做）
- 如果不做会怎样（后果提示）

## 步骤5：生成周报文件
将以上分析输出为markdown文件，保存到：
`/Users/cyingfang/claude/deliverables/health/健康数据/2026-W{周号}-周报.md`

周报文件需包含以下结构：

```
# 健康周报 — 2026年第XX周（M月D日 ~ M月D日）

## 一、本周数据速览
（一张表格汇总所有指标：本周值 / 目标值 / 达标? / 趋势 / 数据完整度）

## 二、趋势分析
（逐项分析变化方向和可能原因）

## 三、预警信号
（仅列出触发的预警，未触发的标注"本周无预警"）

## 四、运动执行
（运动天数、快走/力量详情，与计划对比）

## 五、饮食分析
（饮食达标天数、最薄弱环节、应酬记录）

## 六、睡眠与恢复
（睡眠时长/质量趋势，入睡时间分析）

## 七、身体感受
（如果用户提供了"本周感受"备注，纳入此处；没有则标注"本周无主观反馈"）

## 八、下周计划与提醒
- 下周运动建议
- 下周饮食重点
- 下周特别事项（应酬/出差/加班/就医预约）
- 减压小贴士（每周换一条，不重复）

## 九、一周一句话
（20字以内总结本周健康状况）
```

## 步骤6：更新JSON数据

## 重要提醒
- 第一周（W22, 6/1-6/7）是基线周，即使数据不完整也生成周报，标注"基线建立中"
- 如果本周完全没有任何数据，生成一份空模板周报，标题追加"(无数据)"
- 所有数值保留合理精度（血压整数、体重1位小数、步数整数、睡眠时长1位小数）
- 减压小贴士每周换一条，参考清单见下方

## 减压小贴士库（每周轮换使用一条）
1. 每天找10分钟完全不看手机，就坐着喝杯茶发呆
2. 睡前1小时把手机放在卧室外面充电
3. 工作间隙站起来走2分钟，顺便喝口水——两个目标一次完成
4. 这周选一天晚上不安排任何事，想干嘛干嘛
5. 遇到让你生气的事，先数10下再回复
6. 每天写下3件"今天做得不错的事"
7. 午饭后散步10分钟，不和任何人说话
8. 这周主动拒绝一件事——不是所有饭局都必须去
```
