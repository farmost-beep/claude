# 健康日打卡模板

> 每天复制下面这段，填入数据，发给Claude
> Claude会自动归档到 健康数据/ 目录下的周报文件

## 每日打卡模板（复制使用）

```
每日打卡 2026-06-XX
血压：___ / ___ mmHg
静息心率：___ bpm
体重：___ kg
饮水量：___ L
步数：___ 
睡眠时长：___ 小时
入睡时间：___ 
精力评分(1-5)：___
运动完成：是/否
  - 快走：___ 分钟
  - 力量训练：___ 组
饮食检查：
  [ ] 早餐 [ ] 蔬菜 [ ] 蛋白质 [ ] 坚果 [ ] 全谷物
  [ ] 无含糖饮料 [ ] 19:00前晚餐 [ ] 无宵夜 [ ] ≥2.5L水 [ ] 低盐
备注：___
```

## 每周日附加模板

```
每周汇总 2026-WXX
吸烟：是/否
饮酒：是/否
本周感受（1-5）：
下周目标：
```

## 预警规则（Claude自动检查）

- 🔴 血压连续3天 >135/85 → 推送黄灯预警
- 🔴 血压连续5天 >145/95 → 推送红灯预警，建议就医
- 🟡 连续3天未运动 → 推送提醒
- 🟡 周体重增幅 >1kg → 饮食复查提醒
- 🟢 全部指标达标 → 周末表扬

## 使用方法

1. 每天早上测量后，复制模板填入数据
2. 通过微信发给Claude（或直接在Claude对话中粘贴）
3. Claude自动解析并归档
4. 每周日Claude自动生成周报
