# 投资决策日志

> 投资行为规范§7.2要求：每月回顾本月所有交易决策。这份日志是回顾的基础——没有记录就没有回顾。

---

## 决策记录模板

每次买入/卖出/加减仓后，复制下方模板填写。200字以内。

```markdown
## [买入/卖出/加仓/减仓] — [公司名] — [日期]

**操作**：[买/卖/加/减] [数量]股，价格 [X]元，金额 [X]元
**操作后仓位**：[X]%（占总投资账户）

**买入逻辑**（买入/加仓时填写，≤200字）：
我为什么买这家公司？我看懂了什么市场没看懂的东西？

**卖出理由**（卖出/减仓时填写，≤200字）：
为什么卖？是买入逻辑被破坏了，还是仓位管理需要，还是发现了更好的机会？

**买入前检查清单**（买入/加仓时必须全部打勾）：
- [ ] 读完最近3份季报+2份年报
- [ ] 综合评分≥70（Agent A评分）
- [ ] 读完≥3份券商研报并交叉验证
- [ ] 写下买入逻辑（即上方"买入逻辑"栏）
- [ ] 写下最差情况和最大亏损
- [ ] 确定买入价格上限
- [ ] 确定仓位上限（≤15%核心/10%一般/2%观察）
- [ ] 等了至少24小时冷却期

**事后复盘**（至少1个月后回头看，≤200字）：
当时的决策对吗？逻辑有没有被后面的数据验证/推翻？
```
