## 2. 指标定义

### 指标 1: 市盈率倒数 (E/P)
- **用途**：衡量估值水平，E/P越高越便宜
- **数学公式**：E/P = 1 / PE_TTM
- **合理范围**：PE_TTM ∈ (0, 100]
- **计算前提**：PE_TTM > 0（排除亏损股）

### 指标 2: 市净率倒数 (B/P)
- **用途**：衡量资产估值，B/P越高越便宜
- **数学公式**：B/P = 1 / PB_LF
- **合理范围**：PB_LF ∈ (0, 10]
- **计算前提**：PB_LF > 0（排除资不抵债）

### 指标 3: 净资产收益率 (ROE)
- **用途**：衡量盈利能力，排除价值陷阱
- **数学公式**：ROE = 归母净利润_TTM / 归母权益
- **合理范围**：(0, 0.50]
- **计算前提**：需要最新一期财报数据

### 指标 4: 营收增长率
- **用途**：衡量成长性
- **数学公式**：RevGrowth = (当期营收 - 去年同期) / abs(去年同期)
- **合理范围**：(-0.3, 1.0]
- **计算前提**：上市满1年

### 指标 5: 6个月动量
- **用途**：控制价值陷阱，排除持续下跌的股票
- **数学公式**：Momentum = (今日收盘 / 126交易日前收盘) - 1
- **合理范围**：(-0.5, 1.0]
- **计算前提**：上市满126个交易日

### 指标 6: 综合评分
- **用途**：多因子复合排名
- **数学公式**：先对每项做截面 Z-score，再加权求和
- **Z-score 公式**：Z_i = (X_i - mean) / std
- **加权公式**：Score = 0.30 × Z(1/PE) + 0.20 × Z(1/PB) + 0.25 × Z(ROE) + 0.15 × Z(RevGrowth) + 0.10 × Z(Momentum)

## 3. 信号逻辑

### 3.1 准入过滤（不符合的不参与评分）

    条件1: PE_TTM > 0（排除亏损股）
    条件2: PB_LF > 0（排除资不抵债）
    条件3: 日均成交额 ≥ 10,000,000（流动性，如数据可用）

### 3.2 买入信号（BUY）

**触发条件**（所有条件必须同时满足）：

    条件1: 季度调仓日（3/6/9/12月最后交易日）
    条件2: 综合评分排名在标的池前 top_n 名
    条件3: 当前持仓中该股票未达到目标仓位
    条件4: 总持仓数量 ≤ top_n
    条件5: 该标的未被暂停交易

    附加约束:
    - 新建仓买入数量 = 目标仓位 / 当前价格，按100股取整（A股）
    - 总目标仓位 = 总资产 × total_position_pct / top_n（等权）
    - 单票不超过 single_stock_pct

### 3.3 卖出信号（SELL）

**触发条件**（满足任一即触发）：

    条件1: 季度末综合评分跌出前 (top_n + buffer_n) → 调仓退出
    条件2: 单票亏损超过 stop_loss_pct% → 止损退出
    条件3: 总资产回撤超过 drawdown_meltdown_pct% → 熔断（清仓50%）

### 3.4 信号优先级

    1. 回撤熔断 > 2. 止损退出 > 3. 调仓卖出 > 4. 调仓买入
    同一交易日先执行所有卖出，再执行买入。

## 4. 仓位管理

    方法: 等权配置
    目标单票仓位 = min(总资产 × total_position_pct / top_n, 总资产 × single_stock_pct)
    买入数量:
      - A股: floor(买入金额 / 当前价格 / 100) × 100
      - 港股: floor(买入金额 / 当前价格 / 每手股数) × 每手股数

    约束:
    - 单只标的仓位上限: 总资金的 15%
    - 总仓位上限: 总资金的 80%
    - 现金永远 ≥ 20%

## 5. 风控规则

| 规则 | 默认阈值 | 触发动作 |
|------|---------|---------|
| 单票止损 | 买入成本的 15% | 无条件清仓该股票 |
| 总回撤熔断 | 高点回撤 20% | 清仓50%持仓，保留评分最高的5只 |
| 连续止损 | 季度内3只触发止损 | 下季度仓位减半 |
| 极端行情 | 单日跌幅 > 8% | 暂停该标的交易至下个调仓日 |
| 单市场上限 | 单一市场 > 60% | 超出部分移至另一市场 |

## 6. 回测参数

| 参数 | 默认值 | 说明 |
|------|--------|------|
| 初始资金 | 1,000,000 人民币 | 回测起始资金 |

| 滑点 | 0.1% | 买入价+滑点，卖出价-滑点 |
| 回测区间 | 2021-01-01 ~ 2025-12-31 | 5年，覆盖牛熊 |
| 基准 | 沪深300 | A股基准对比 |

## 7. 禁止事项

1. 不可使用未来数据
2. 不可忽略手续费和滑点
3. 不可在信号中引入未在本文档定义的指标
4. 不可修改信号优先级规则
5. 不可将风控阈值设为比本文档更宽松的值
6. 不可修改因子权重（v1.0固定权重）
7. 不可修改调仓频率（固定季度）

## 8. 已知局限

- 价值因子在流动性牛市（如2024年9-10月题材炒作期）可能显著跑输
- 财报数据滞后1-3个月，但不影响价值投资框架
- 不含行业中性化，可能在特定赛道过于集中
- 季度调仓可能在快速下跌行情中反应偏慢（由止损规则弥补）
