# 用户协议与隐私声明

## 一、服务说明
本应用为个人健康数据管理工具，用于读取华为运动健康服务中的个人健康数据（步数、心率、睡眠、血氧等），仅限本人使用。

## 二、数据收集与使用
1. 本应用仅读取用户明确授权的健康数据
2. 数据仅存储于用户本地设备，不会上传至任何第三方服务器
3. 不会将数据用于任何商业目的

## 三、数据安全
1. 所有数据传输使用OAuth 2.0加密通道
2. Access Token仅用于本次会话，定期刷新
3. 用户可随时撤销授权

## 四、用户权利
1. 可随时在华为健康App中撤销对本应用的授权
2. 可要求删除所有已存储的健康数据

## 五、免责声明
本应用为个人项目，按"现状"提供，不承担因使用或无法使用本服务所产生的任何直接或间接损失。

## 六、联系方式
如有疑问，可通过 GitHub Issues 联系。
